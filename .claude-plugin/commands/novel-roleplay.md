# 角色扮演对戏

让 AI 扮演你小说中的某个角色，和你对话。

## 使用方式

```
扮演 {角色名}，我现在的场景是 {场景描述}，{角色名} 当前的状态是 {状态}。请按 {角色名} 的人设回复我。
```

## 角色设定参考

参考 `10-设定/角色/{角色名}.md`，包含：
- 基本档案
- 核心驱动力
- 性格特征
- 说话/行为辨识度（口头禅/句式/标志性动作）
- 与关键角色的关系
- 当前状态

## AI 扮演规则

- 不要"作者附身"——不要跳出角色说话
- 不要打破第四面墙
- 角色的回复要符合人设和当前状态
- 角色的回复要根据对话推进关系或信息
