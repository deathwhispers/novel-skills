# 项目结构校验

对用户的中文小说项目做全面校验。

## 校验流程

运行：

```bash
python3 scripts/validate_novel_project.py /path/to/novel --check-consistency --verbose
```

## 检查项

- 必选目录和文件是否存在
- 各章节字数统计
- 设定与正文的连续性隐患
- 回收总账是否更新
- 连载驾驶舱是否更新
- 人物称呼一致性（高频人名是否在角色卡中）
- 关键物件失联（30+ 章未出现）
- 路程记录是否与地点距离表一致
- 信息债紧迫性矩阵（哪些债到了"立即触碰"格）
- 百万级文件缺失建议（30/50/100 章分别需要什么文件）
- 角色 voice 漂移检测

## 输出

按 P0/P1/P2 排序的问题清单 + 自动修复建议。
