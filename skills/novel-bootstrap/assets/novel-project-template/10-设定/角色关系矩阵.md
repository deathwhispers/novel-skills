# 角色关系矩阵

> **使用说明**：本表是角色之间的横向关系层，对应 `10-设定/角色/*.md` 的纵向单角色档案。
> 当书内有 10 个以上重要角色时，必须建立本表——只跟踪"主角-配角"是绝对不够的，配角之间的恩怨、师门情、合作也会推动主线。
> 百万级长篇建议本表覆盖 30-50 个角色，超过 50 个时分卷建立子表。
> 行=角色 A，列=角色 B，格子=关系。

## 关系类型枚举

| 缩写 | 全称 | 说明 |
|------|------|------|
| 血亲 | 血亲/家人 | 父母、子女、兄弟姐妹 |
| 师徒 | 师徒/传承 | 师傅、弟子、传人 |
| 恋人 | 恋人/CP | 感情线上的核心 |
| 暧昧 | 暧昧/未明 | 单箭头或双箭头未挑明 |
| 挚友 | 挚友/知己 | 互相信任的核心朋友 |
| 盟友 | 盟友/合作 | 利益合作但不一定亲密 |
| 同门 | 同门/同派 | 同一势力但不一定熟 |
| 敌对 | 敌对/对抗 | 公开冲突 |
| 仇人 | 仇人/血仇 | 有历史恩怨 |
| 陌生 | 陌生/未接触 | 当前没有显著互动 |
| 师出 | 师出同门 | 历史上有共同的师父或派系 |

## 关系亲疏强度（1-5）

- **5**：生死之交/血亲/挚爱，不可分割
- **4**：高信任，重大决定会同步
- **3**：常规信任，平时有往来
- **2**：浅层关系，有一定信任但有限
- **1**：点头之交，几乎无实际往来
- **-1**：轻度不信任/小摩擦
- **-2**：紧张/冷战
- **-3**：敌对/对抗
- **-4**：仇人/有旧恨
- **-5**：血仇/必须你死我活

## 角色关系矩阵

> 单元格写法：关系类型 / 亲疏强度（1-5/-1-5）/ 关键事件或角色间张力来源（一句话）
> 自反格子（角色A ↔ 角色A）填"—"。
> 不是所有格子都需要填——只有"剧情相关的关系"才需要记录。

### 填写示例

> 假设有 6 个角色：主角林尘（青云宗内门弟子）、苏晚（林尘挚友）、李昭（青云宗内门大师兄，反派）、江鹤（秘境同盟者）、赵长老（青云宗长老，幕后反派）、苏晚师父（已死）

|  | 林尘 | 苏晚 | 李昭 | 江鹤 | 赵长老 | 苏晚师父 |
|--|------|------|------|------|------|------|
| **林尘** | — | 挚友 / 4 / 入门起的朋友，互相信任 | 仇人 / -4 / 入门起被针对 | 盟友 / 3 / 秘境中被迫合作 | 紧张 / -2 / 察觉到他在试探 | 师徒- / 2 / 敬佩其为人但未直接接触 |
| **苏晚** | 挚友 / 4 / 入门起的朋友 | — | 敌对 / -3 / 目睹他对林尘的陷害 | 暧昧 / 1 / 江鹤对苏晚有意 | 紧张 / -2 / 撞见其密谈险些被害 | 血亲- / 5 / 师父是她唯一亲人 |
| **李昭** | 仇人 / -4 | 敌对 / -3 | — | 敌对 / -3 / 江鹤被李昭势力追杀 | 师徒 / 4 / 表面是李昭的引路人 | 师徒- / 1 / 入门时见过几面 |
| **江鹤** | 盟友 / 3 | 暧昧 / 1 | 敌对 / -3 | — | 紧张 / -3 / 江鹤失踪前最后见的人 | 陌生 / 0 |
| **赵长老** | 紧张 / -2 | 紧张 / -2 | 师徒 / 4 | 紧张 / -3 | — | 师徒- / 2 / 同为青云宗长老 |
| **苏晚师父** | 师徒- / 2 | 血亲- / 5 | 师徒- / 1 | 陌生 / 0 | 师徒- / 2 | — |

## 关系大事记

> 关系不是静态的。每一次重大变化（结盟/破裂/翻脸/告白/反目）都要落一行。

## 关系推进节奏

> 把"该在什么时候推哪段关系"明确化。百万级长篇里，最常见的死法是某段关系突然冒头——读者会一脸问号"这人谁"。

| 卷范围 | 应推进的关系 | 推进方式 | 预期读者情绪 |

| 第 1 卷 | | | |
| 第 2 卷 | | | |
| 第 3 卷 | | | |
| 第 4 卷 | | | |
| 第 X 卷 | | | |

### 关系推进常见的坑

- ❌ 关系推进只靠对话（缺乏共同事件支撑）
- ❌ 配角之间的关系全部通过主角传递
- ❌ 长期没有新关系注入或旧关系变异
- ❌ 关系状态长时间不变，失去张力
- ❌ 一卷里把多对关系同时推到高潮（焦点分散）

## 关系自检

- [ ] 主角所在的关系网络是否包含至少 3 种不同类型的关系（不要全是师徒或全是朋友）
- [ ] 是否有 2 段以上的关系处于"长期未推进"状态（> 20 章没变化）
- [ ] 是否有 1 段以上的"三角/四角"关系在主线中
- [ ] 关系矩阵中是否有关系强度为 5 但已经很长时间没出现（提示这段关系正在被浪费）
- [ ] 最近一卷里推进的关系数量（建议 1 卷 1-2 段关系深度推进，不要批量）
- [ ] 是否有"应该成为敌人但没成"或"应该成为盟友但没成"的关系漏洞
