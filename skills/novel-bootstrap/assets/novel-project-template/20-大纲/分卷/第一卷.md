# 第一卷卷纲

> **使用说明**：第一卷是全书最关键的一卷——它决定了读者会不会继续追。
> 本模板匹配卷通用模板的全部字段，并额外加入了写作指导字段。
> 不需要一次性填完。先填「主要卖什么」「开卷局面」「卷末收束」三个核心字段就可以开始铺节拍卡，其余字段在写作过程中逐步补全。
> **核心检查**：第一卷写完时，读者应该已经知道「这书在讲什么」「为什么要继续追」。

写本卷在全书主线中的功能：是起势段、升级段、转折段还是收束段
> 第一卷通常是「起势段」——立人、立世界、立追读承诺。
