## 卷末快照的使用流程

> 这是写"下一卷卷纲"前的标准动作：

1. 复制本表模板，命名为 `卷末快照-第X卷.md`
2. 填写所有主要角色
3. 找出当前最危险的 3 个角色
4. 找出当前最不稳定的 3 段关系
5. 找出当前最该在下一卷出场的 3 个角色
6. 这 9 个对象就是下一卷的"核心驱动"

## 自检

- [ ] 是否所有主要角色都有快照（不要漏人）
- [ ] 快照是否足够具体（"变强了"不算，要写"修为从筑基初期升到筑基中期"）
- [ ] 是否标出了"下一卷非出场不可"的角色（避免卷内遗忘）
- [ ] 关键关系状态是否被记录（"和某人和好"不够，要写"从敌人变暧昧"）
- [ ] 上一次卷末快照和这次的对比，是否能看出"非读不可的进化"（即整本书在推进）
