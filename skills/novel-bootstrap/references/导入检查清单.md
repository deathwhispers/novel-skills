# 旧稿导入清单

## 导入顺序

1. 先建立统一模板目录
2. 盘点旧文件来源和版本
3. 设定、人物卡、说明文档归入 `10-设定/`
4. 大纲、分卷纲、章节计划归入 `20-大纲/`
5. 正文归入 `30-正文/导入/`
6. 把导入过程中的取舍和缺口记下来

## 推荐映射

- 角色卡 -> `10-设定/角色/`
- 世界观文档 -> `10-设定/世界规则.md`
- 设定补充 -> `10-设定/硬设定.md`
- 分卷纲 -> `20-大纲/分卷/`
- 分章纲 -> `20-大纲/节拍卡/`
- 已写正文 -> `30-正文/导入/`

## 导入后别漏做

- 在 `30-正文/导入/源文件索引.md` 记录来源索引
- 在 `90-运行/当前进度.md` 写清当前进度和明显缺口
- 判断下一步该转给哪个 skill，而不是继续把所有问题混在导入阶段
