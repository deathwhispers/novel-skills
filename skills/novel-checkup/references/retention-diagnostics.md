# 追读弱点排查表

用于读者开始觉得“最近几章没那么想追了”时，排查问题根源。

## 检查范围

- 检查对象：
- 最近几章：
- 当前阶段：起章 / 中段 / 高潮前 / 卷末 / 转卷

## 一、章尾发动机是不是变弱了

- 最近几章的章尾主要靠什么挂住读者：
- 是否连续出现平静收束：
- 是否连续几章都没有不可逆决定、新问题或关系破口：

## 二、推进是不是变成了说明

- 最近几章是否事件很多，但局面变化很少：
- 最近几章是否在大量交代信息，却没有真正兑现戏：
- 是否出现“看着写了很多，其实可以跳读”的段落：

## 三、读点是不是被拖没了

- 当前最值钱的爽点 / 痛点 / 悬念点是什么：
- 最近几章有没有持续在碰它：
- 它是不是被延迟过久、稀释过度或被别的无效段落冲散：

## 四、关系和人物是不是失去拉力

- 关键关系最近一次真正翻动是在什么时候：
- 主角最近几章是否一直被动、没有新决定：
- 配角是否只在提供信息，没有制造拉扯：

## 五、最像“弃追点”的位置

- 最容易让读者觉得“先放一放再看”的章节：
- 该章最伤追读的原因：
- 最应该补强的是：章尾 / 冲突 / 回收 / 关系 / 升级

## 六、结论

- 当前追读最弱的核心原因：
- 优先修哪几章最划算：
- 下一章必须补回来的东西：
