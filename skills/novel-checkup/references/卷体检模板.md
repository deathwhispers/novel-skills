# 卷体检模板

用于一卷完成后，或长篇中段怀疑“这一卷节奏出问题了”时做统查。

## 基本信息

- 检查对象：
- 卷范围：
- 本卷理论承诺：
- 本卷实际写到了哪：

## 一、开卷承诺有没有立住

- 这卷一开始答应读者的东西是什么：
- 前几章有没有把这个承诺立稳：
- 主角、主冲突、主读点是否尽快清楚：

## 二、中段有没有塌

- 中段是否出现连续重复推进：
- 中段是否一直加事件，但没有真正抬高代价：
- 中段是否存在明显平章堆积：
- 关系线、升级线、悬念线是否至少有一条持续在推进：

## 三、卷末有没有给到阶段性满足

- 卷末收住了什么：
- 卷末放大了什么：
- 读者在卷末应该获得的满足是否真的给到：
- 卷末有没有只留坑、不兑现的风险：

| 悬念线 | | | | |

## 五、本卷最伤追读的地方

- 哪几章明显拖慢了：
- 哪几个章尾发动机偏弱：
- 哪些爽点 / 痛点 / 关系推进延迟过久：
