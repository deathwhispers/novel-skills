# 商业化检查清单

逐项检查，命中 3 项以上红色需要优先改稿。

## 书名自查
- [ ] 书名能在 3 秒内让目标读者知道「这是什么题材」
- [ ] 书名包含至少 1 个情绪词或冲突信号
- [ ] 不存在同平台相似书名（避免搜不到）
- [ ] 去掉书名后，仅凭简介也能靠关键词锁定读者

## 简介自查
- [ ] 前 100 字包含异常、目标、阻力三要素
- [ ] 没有超过 3 个不熟悉的名词（人名/地名/力量名）
- [ ] 不剧透前 30 章以后的内容
- [ ] 简介结束时留下一个「想知道怎么解决」的悬念

## 开篇自查
- [ ] 第 1 段是画面或动作，不是背景说明
- [ ] 前 300 字内出现了冲突或异常
- [ ] 前 1000 字内有第一次逆境或压力
- [ ] 第一章结尾有明确追读钩子
- [ ] 第三章结束时读者能说出「这本书最特别的地方」

## 节奏自查
- [ ] 前 10 章内有第一个小高潮/兑现
- [ ] 前 30 章内有第一个中高潮/阶段性兑现
- [ ] 每章结尾都有「新压力/新问题/新信息」之一
- [ ] 没有连续 3 章只推剧情没有情绪变化

## 付费节奏自查
- [ ] 免费期字数不低于 15 万
- [ ] 第一个付费节点前有足够追读钩拉高期待
- [ ] 非付费章节没有注水超过 30%
- [ ] 付费章节的内容密度不低于免费期平均线
