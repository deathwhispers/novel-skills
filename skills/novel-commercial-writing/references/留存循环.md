# 留存与追读循环

## 章尾发动机类型

每章结尾至少使用以下一种，最好两种叠加：

| 类型 | 定义 | 示例 |
|------|------|------|
| 新问题 | 章末出现一个本章解决不了的新情况 | 「门外响起了脚步声——不可能这个时候有人来」 |
| 不可逆决定 | 角色做了一个无法反悔的选择 | 「他签了字，从此再无退路」 |
| 倒计时 | 时间压力逼近 | 「距离秘境关闭还有三炷香」 |
| 权力反转 | 强弱关系颠倒 | 「刚才跪着的人，现在站了起来」 |
| 情绪余震 | 章末留一个情绪炸点 | 「她终于知道那封信是谁写的了」 |
| 关系破口 | 关系状态突变 | 「我们离婚吧。」 |

## 追读循环模型

```
章尾钩 → 读者点下一章 → 小满足 → 新钩 → 循环
```

关键：
- 小满足必须在下一章前 30% 内兑现
- 新钩必须在下一章后 30% 内埋下
- 连续 3 章不给小满足 → 追读掉

## 长线追读点

- 升级线：读者想知道下一级是什么
- 关系线：读者想知道关系下一次翻面
- 悬念线：读者想知道真相或反转

## 追读自检

- [ ] 最近 3 章的章尾是否都有明确推进器
- [ ] 最近 3 章是否没有连续平淡收尾
- [ ] 最近 5 章内是否有至少一次情绪变化
- [ ] 最近 10 章内是否有至少一次关系变化
- [ ] 最近 10 章内是否有至少一次局面翻面
