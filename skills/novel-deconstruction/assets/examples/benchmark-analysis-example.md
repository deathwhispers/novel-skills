# 对标拆解：灰灯夜雨

## 基本信息

- 平台：番茄
- 形态：长篇
- 题材：都市悬疑
- 为什么选它：开篇拉人强，章节尾钩密集

## 这本书真正卖什么

- 这部作品真正卖的不是“悬疑”两个字，而是“主角每次接近真相都会失去一点现实安全感”。

## 黄金钩子

- 标题钩子：一个有氛围感的危险意象
- 简介钩子：异常现场 + 主角职业 + 个人代价
- 开篇钩子：第一场就给出不合常理的视觉证据

## 结构观察

- 主角亮相方式：先让她做专业判断，再让判断失效
- 冲突递进方式：每次局部答案都带来更大的问题
- 情绪循环：发现 -> 被压制 -> 逆向验证 -> 新证据
- 章尾推进方式：总在主角刚形成一个结论时打断

## 值得借的商业写法

1. 用职业判断制造可信感
2. 用“证据反咬主角”制造追读
3. 每章只推进一格真相，但一定推进

## 不能照抄的地方

- 世界观外壳：不要复制案件系统
- 名场面文案：不要搬原句
- 角色包装表层：不要照抄冷感女主壳子

## 迁移到我项目里的注意点

- 我能借用的：证据反咬主角
- 我需要变形的：职业场景要换成我的题材语境
- 不适合我的：过重的社会派口吻
