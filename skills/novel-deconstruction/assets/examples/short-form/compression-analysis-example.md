# 短篇压缩拆解示例

## 压缩点落在哪里

- 题眼在标题和第一段同时抛出，读者一上来就知道这篇到底要打什么
- 反转在中后段触发，当读者以为矛盾来自外部时，突然反指主角自身
- 结尾回到开头意象，但含义已经彻底翻面

## 压缩逻辑怎么成立

- 提前压进题眼的信息：关系失衡、关键代价、最后要补上的情绪账
- 故意延后的信息：真正背叛者、主角最想藏住的事实
- 兼做双重功能的段落：一段对话既推动关系，又悄悄埋下反转信息

## 结尾回响为什么不硬

- 回响前文的是开头一句看似平常的话，到结尾才暴露它其实是最后的证词
- 前文同时给了误读线索和真实线索，所以读者被“带偏”是顺理成章，不是作者强翻
