# Short-Form Compression Analysis Example

## Short Form Compression

- 题眼出现位置：标题和第一段同时给出
- 反转触发位置：中后段，当读者以为矛盾来自外部时反指主角
- 结尾回响方式：回到开头意象，但含义完全反转

## Compression Logic

- 哪些信息被提前压进题眼：关系失衡、关键代价、最后会付出的情绪账
- 哪些信息被故意延后：真正背叛者、主角最想隐瞒的事实
- 哪些段落承担双重功能：一段对话既推进关系，又埋反转信息

## Ending Echo

- 结尾回响前文的什么：开头看似普通的一句话，在结尾暴露为最后的证词
- 为什么这不是硬反转：前文已给足误读线索和真实线索，只是读者先站错边
