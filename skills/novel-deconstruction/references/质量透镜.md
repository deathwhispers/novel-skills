# 拆书质量观察角度

拆文时，不要只看“读者爽不爽”，还要看“作品为什么没塌”。

## 1. 因果与推动

重点问：

- 每个关键转折前有没有足够的原因
- 角色的决定是否符合已建立的人设和处境
- 情节推进是不是靠角色行动，而不是作者硬推

## 2. 场景是否站得住

重点问：

- 每个场景是否有明确目标
- 场景里有没有有效阻力
- 场景结束后局面是否真的发生变化

## 3. 细节是否有用

重点问：

- 细节是不是具体而有功能
- 细节是在装饰，还是在承担信息与情绪
- 有没有重复抽象词替代真实场景

## 4. 信息怎么放

重点问：

- 读者知道什么，角色知道什么
- 什么时候故意不说
- 什么信息是一开始不说，后面再回收

## 5. 埋线和回收是否守规矩

重点问：

- 伏笔是否只是丢出，还是被真正回收
- 回收时有没有带来关系、认知或局面变化
