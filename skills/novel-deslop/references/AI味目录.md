# 常见 AI 腔对照表

## 句式类

| AI 腔 | 自然替代 |
|-------|---------|
| 由于……所以…… | 直接写原因或结果，不用连词 |
| 他/她感到/觉得/认为 | 删掉，换成动作或对话 |
| 随着…… | 拆成两句，不要用「随着」开头 |
| 不禁/不由地/忍不住 | 换成具体动作：「他笑了一声」vs「他忍不住笑了」 |
| 说明/意味着/代表 | 留给读者判断，不要说穿 |
| 在……的带领下/帮助下 | 换成具体互动 |
| 只见/就看到 | 删掉，直接写画面 |

## 结构类

| AI 腔 | 自然替代 |
|-------|---------|
| 每个段落 3-5 句均衡 | 长短段交替，关键段落可只有一句 |
| 先景后人 | 先人后景，或人和景同时出现 |
| 对话后紧跟解释 | 删掉解释，信任读者能看懂 |
| 叙事后紧跟心理 | 把心理化成行动或对话潜台词 |
| 每章开头重述前文 | 直接推进，不要回顾 |

## 词汇类

| AI 腔 | 自然替代 |
|-------|---------|
| 点了点头 | 点了下头 / 没说话 |
| 与此同时 | 删掉，或者用「这时」 |
| 然而/但是 | 一半的情况可以删掉，用短句转折 |
| 众所周知 | 不要用，直接写世界普遍认知 |
| 毫无疑问 | 删掉，用事实说话 |
| 某种/某种程度 | 换成具体描述或直接删 |
