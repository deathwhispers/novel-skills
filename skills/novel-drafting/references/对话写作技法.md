# 对话写作技法

> **使用说明**：网文 60-70% 篇幅是对话。本表是写对话的技法库——长篇网文里"怎么写对话"是核心技术活。
> 写对话前先扫一眼本表；写完后用本表自检。

## 一、对话的 5 种基础类型

> 一段对话只承担一种功能，不要混用。

| 类型 | 目的 | 标志 | 错误用法 |
|------|------|------|---------|
| **信息传递** | 推进剧情/揭示设定 | 双方一问一答推进 | ❌ 把它写成作者的"信息说明栏" |
| **关系推进** | 改变两个角色的关系 | 一方主动说/做/暗示 | ❌ 没有任何行动/暗示，纯聊天 |
| **冲突对抗** | 制造张力/立场对立 | 双方都坚持不退让 | ❌ 一方太快服软 |
| **情感表达** | 表达/隐藏感情 | 潜台词/动作/沉默 | ❌ 直接说"我爱你" |
| **氛围营造** | 渲染场景/铺垫情绪 | 环境+对话交织 | ❌ 完全脱离场景 |

## 二、对话的"3 段式"结构

> 一段有效对话至少有 3 段：开场铺垫 → 主体交锋 → 收尾落点。

### 段 1：开场铺垫（1-2 轮）

- **目的**：建立对话的语境
- **写法**：先说"无关紧要的话"（问候/议论他事），让读者进入场景
- **错误**：一上来就直奔主题，读者会感觉"假"

```
❌ 林尘一进门就问："龙魂的事，你想说什么？"
✅ 林尘坐下，倒了杯茶，沉默片刻才开口："你那枚玉佩，师父给你的？"
```

### 段 2：主体交锋（3-N 轮）

- **目的**：推进对话的核心目标
- **写法**：用"信息差"驱动——每轮对话双方都知道得更多
- **核心**：每轮都要"推进"，不能原地打转

> **每轮推进的 5 种方式**：
> 1. 揭示（说出一个新信息）
> 2. 反驳（否定对方的观点）
> 3. 试探（问一个模糊的问题）
> 4. 转移（说一个看似无关的细节）
> 5. 沉默（用动作代替语言）

### 段 3：收尾落点（1 轮）

- **目的**：让对话"落地"——留下钩子或完成动作
- **写法**：最后一句要"刺"——让读者想翻下一章
- **错误**：对话草草收尾，没留任何余味

```
❌ "那就这样吧。"（散场式收尾）
✅ "你以为今晚走得掉？"（钩子式收尾）
```

## 三、潜台词设计

> 长篇网文最忌"直说"。每一段重要对话都应该有"言外之意"。

### 潜台词的 3 层

| 层次 | 说什么 | 真正想说什么 | 写法 |
|------|------|------------|------|
| **表层** | 字面意思 | — | 直接说出来 |
| **中层** | 字面意思 | 暗示感情/态度 | 用语气/动作/停顿 |
| **深层** | 字面意思 | 揭示关键真相 | 看似无意实则有意 |

### 潜台词的 5 种实现技法

1. **反话正说**：嘴上说不在意，行为暴露在意
   ```
   "我才不担心他。"（边说边往窗外看）
   ```

2. **答非所问**：用无关回答暗示态度
   ```
   问："你愿意帮我吗？"
   答："今晚的月亮挺圆的。"（潜台词：不愿意）
   ```

3. **动作代替语言**：用动作表达说不出口的话
   ```
   她没说话，只是把手里的玉佩递了过去。
   ```

4. **沉默与停顿**：用空白承载情绪
   ```
   "我……" 他张了张嘴，最终什么都没说。
   ```

5. **重复对方的话**：用回声暗示内心
   ```
   "你觉得我不该信他？"
   "该不该信……"（重复 = 在思考 = 内心动摇）
   ```

## 四、对话节奏

> 一章里对话不能"匀速"——要有快有慢。

### 节奏的 4 种组合

| 节奏 | 适用 | 写法 |
|------|------|------|
| **快对快** | 紧张场景/冲突 | 短句、单字、反问、不用动作 |
| **慢对慢** | 情感场景/铺垫 | 长句、停顿、动作交织 |
| **快对慢** | 实力悬殊的对话 | 一方急促，另一方从容 |
| **沉默接力** | 情感戏 | 大量动作/环境，没有台词 |

### 节奏控制的 4 个信号

- ✅ 紧张场景中插入一段安静对话
- ✅ 长段对话后用一句短句"切断"
- ✅ 情感戏中突然出现一句"刺"的话
- ❌ 整章都是快对快（读者累）
- ❌ 整章都是慢对慢（读者闷）

## 五、对话中的角色辨识度

> 写到中后期，所有角色说话都一个味——这是 voice 漂移。
> 每个核心角色必须有"语言指纹"。

### 语言指纹的 5 维

| 维度 | 例子 |
|------|------|
| **口头禅** | 林尘："我不认命。" |
| **句式偏好** | 苏晚长句、问句多；李昭短句、反问多 |
| **用词倾向** | 赵长老用书面语；苏晚用俚语 |
| **语气** | 林尘沉稳；李昭尖刻 |
| **绝不说的话** | 主角不说"谢谢"，用行动表达 |

### 让角色"不像别人"的 3 个方法

1. **同一情境，不同反应**：面对侮辱，主角沉默、李昭反唇、苏晚岔开
2. **同一信息，不同句式**：A 说"我明白了"，B 说"哦"，C 说"原来如此"
3. **同一情感，不同表达**：爱意——A 写在诗里，B 藏在礼物里，C 化为行动

## 六、信息释放控制

> 长篇网文的"何时说"比"说什么"更难。

### 信息释放的 5 个"金时机"

| 时机 | 适合释放的信息 | 示例 |
|------|--------------|------|
| **章首** | 上一章悬念的兑现 | 第 5 章开：赵长老的信终于送到了 |
| **章中冲突中** | 让信息改变局势 | 战斗中突然揭示敌人弱点 |
| **章尾前** | 章尾钩子的关键信息 | "其实，你师父没死。"（章尾） |
| **场景转换时** | 视角切换中的信息 | A 视角结束 → B 视角开始有新的信息 |
| **倒叙/插叙** | 历史/背景信息 | 主角昏迷时回忆一段往事 |

### 信息释放的 5 个错误

- ❌ **说得太早**：还没铺垫就揭示，读者反应冷淡
- ❌ **说得太晚**：读者已经忘了前面铺垫
- ❌ **说得太直**：直接说"他的身份是 X"，没有任何铺垫
- ❌ **说得太碎**：一个信息分 10 次说，读者不耐烦
- ❌ **说得太密**：一章内释放 5+ 个关键信息，读者消化不了

## 七、对话的"刺点"设计

> 一段对话的最后一两句是"刺"——让读者读完想翻下一章。

### 刺点的 4 种类型

1. **悬念刺**：
   ```
   "你到底要说什么？"
   "——你师父没死，但活不过今晚了。"
   ```

2. **反转刺**：
   ```
   "这件事只有师父知道。"
   "——师父已经告诉我了。"
   ```

3. **情感刺**：
   ```
   "我不会原谅你。"
   "我知道。" 她转身，"我从没要你原谅。"
   ```

4. **行动刺**（不说，用动作）：
   ```
   主角没回答，把玉佩往桌上一放，转身离去。
   ```

## 八、对话自检清单

写完一段对话后，逐项检查：

- [ ] 这段对话承担了哪种功能（信息/关系/冲突/情感/氛围）
- [ ] 是否至少有 3 段（铺垫-主体-收尾）
- [ ] 每轮对话都在推进（没有原地打转）
- [ ] 是否有潜台词（不是直说）
- [ ] 节奏是否有快有慢
- [ ] 角色说话是否有辨识度（不是"统一口吻"）
- [ ] 信息释放的时机是否合适
- [ ] 收尾是否有"刺点"
- [ ] 对话是否符合角色的"语言指纹"
- [ ] 对话是否推动了关系/剧情/悬念至少一项

## 九、不同场景的对话模式

### 战斗前的对话

- 短句、对峙
- 用动作/眼神代替长篇大论
- 最后一句要"刺"——引发战斗

### 战斗中的对话

- 极短：单字、反问、命令
- 边打边说（"嘿，接招！")
- 揭示弱点或关键信息

### 战斗后的对话

- 缓和节奏——喘息、疗伤
- 重新建立人物关系
- 揭示战斗带来的新信息

### 床戏/亲密戏的对话

- 极简——动作/环境/呼吸
- 内心活动多于对话
- 一句关键的话点题

### 诀别/生离死别的对话

- 极慢——长句、断句
- 用未完成的话、未说出的话
- "我……" + 沉默

### 反派对峙的对话

- 不正面交锋——绕弯
- 一方在试探，一方在演戏
- 最后一句要"刺"——形成对峙

### 师徒对话

- 师父话里有话
- 徒弟听一半懂一半
- 最后会有"传功/传物"的动作

### 群像对话（多人）

- 每个角色说话必须不同
- 用动作标注谁在说话
- 不能"五个人一起说同一件事"
