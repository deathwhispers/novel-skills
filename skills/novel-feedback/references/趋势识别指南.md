## 一、5 种典型趋势

### 趋势 1：稳步上升

```
追读率
  80% ┤                              ╭──
  70% ┤                          ╭───╯
  60% ┤                      ╭───╯
  50% ┤                  ╭───╯
  40% ┤              ╭───╯
      └───────────────────────────────
        Ch1   Ch10   Ch20   Ch30   Ch40
```

**含义**：作品渐入佳境，读者粘性增强。
**原因**：节奏稳定、爽点兑现、角色讨喜。
**行动**：维持当前策略，记录成功因素。

---

### 趋势 2：平稳运行

```
追读率
  70% ┤  ╭─╮  ╭─╮  ╭─╮  ╭─╮  ╭─╮
  60% ┤──╯  ╰──╯  ╰──╯  ╰──╯  ╰─
  50% ┤
      └───────────────────────────────
        Ch1   Ch10   Ch20   Ch30   Ch40
```

**含义**：作品稳定，但没有突破。
**原因**：套路化、缺乏惊喜、读者不增不减。
**行动**：考虑加入新元素/新读点，打破平稳。

---

### 趋势 3：缓慢下降

```
追读率
  80% ┤╭──
  70% ┤╯  ╰──
  60% ┤      ╰──
  50% ┤          ╰──
  40% ┤              ╰──
      └───────────────────────────────
        Ch1   Ch10   Ch20   Ch30   Ch40
```

**含义**：读者逐渐流失，存在慢性问题。
**原因**：节奏疲劳、套路重复、期待值消耗。
**行动**：
- 检查近 10 章的推进密度
- 检查是否有"无效重复"（同样的事情做了多次）
- 考虑引入新角色/新冲突/新设定

---

### 趋势 4：波动震荡

```
追读率
  80% ┤      ╭─╮              ╭─╮
  60% ┤  ╭──╯  ╰──╮      ╭──╯  ╰──
  40% ┤──╯          ╰──╮──╯
  20% ┤                ╰──
      └───────────────────────────────
        Ch1   Ch10   Ch20   Ch30   Ch40
```

**含义**：作品质量不稳定，时好时坏。
**原因**：
- 爽点章和过渡章质量差距大
- 更新不稳定（断更/少更影响追读）
- 剧情波动（好章节和差章节交替）

**行动**：
- 分析波峰章和波谷章的差异
- 提高波谷章的质量下限
- 稳定更新频率

---

### 趋势 5：断崖下跌

```
追读率
  80% ┤╭──
  60% ┤╯  ╰──
  40% ┤      ╰──────────────
  20% ┤                      ╰──
      └───────────────────────────────
        Ch1   Ch10   Ch20   Ch30   Ch40
```

**含义**：某个节点出现严重问题，读者大量流失。
**可能原因**：
- 重大剧情失误（角色降智/逻辑断裂）
- 期待值崩塌（承诺未兑现/反转失败）
- 更新问题（断更/少更/质量骤降）
- 外部因素（平台推荐变化/竞争作品出现）

**行动**：
1. 定位断崖点（哪一章开始跌）
2. 分析该章内容：剧情/角色/节奏/设定
3. 对比前后章差异
4. 制定修复方案
5. 修复后监控是否回升

## 二、趋势诊断决策树

```
你的追读率趋势是什么？
  ├─ 稳步上升 → 维持，记录成功因素
  ├─ 平稳运行 → 考虑加新元素打破平稳
  ├─ 缓慢下降 → 检查推进密度/套路重复
  ├─ 波动震荡 → 分析波峰波谷差异，提高下限
  └─ 断崖下跌 → 定位断崖点，分析原因，制定修复

下降的具体形态？
  ├─ 卷末跌 → 卷末钩子弱
  ├─ 中段跌 → 推进密度不够/空转
  ├─ 开卷跌 → 书名/简介/开篇问题
  ├─ 单章跌 → 该章写法问题
  └─ 整体跌 → 结构性问题（节奏型/卷方向）
```

## 三、与评论的交叉验证

数据趋势 + 评论内容 = 真实问题

| 数据趋势 | 评论关键词 | 真实问题 | 修复方向 |
|---------|-----------|---------|---------|
| 缓慢下降 | "水""拖" | 推进段效率低 | 压缩铺垫/加冲突 |
| 断崖下跌 | "毒""弃了" | 重大剧情失误 | 定位问题章修复 |
| 波动震荡 | "时好时坏" | 质量不稳定 | 提高下限 |
| 平稳运行 | "还行""没惊喜" | 缺乏突破 | 加新元素 |

## 快速检查（3 分钟）

1. 最近 10 章的追读率画出来，是什么形状？（升/平/降/波/断）
2. 如果是下降/波动/断崖，对应评论的关键词是什么？
3. 趋势和评论是否匹配？（不匹配→重新分析）

**第 3 题"不匹配"→数据或评论至少一个有问题，需要重新收集**
