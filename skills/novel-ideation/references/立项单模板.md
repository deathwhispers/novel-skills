# 创作简报模板

```md
# 创作简报

## 一句话书核

一句话说清主角、核心冲突和最吸引人的变化。

## 这本书写给谁

目标读者是谁，他们点开这本书是想得到什么。

## 这本书反复兑现什么

这本书会持续兑现给读者的快感、痛感、悬念或关系满足是什么。

## 新鲜感从哪里来

列出 3 到 5 个让它不落成常规模板文的差异点。

## 情绪发动机

角色最稳定、最能不断推剧情的情绪驱动力是什么。

## 不要滑进去的老套路

明确不要落入哪些桥段习惯、语言腔调和题材陈词。

## 篇幅规划

### 总体预算

- **总字数**：XX万字
- **总章节数**：XX章
- **分卷数**：XX卷
- **每卷章节数**：XX章
- **每章字数**：XXXX字
- **预计写作周期**：XX个月

### 分卷预算

| 卷号 | 预计章节数 | 预计字数 | 卷名 | 功能定位 |
|------|-----------|---------|------|---------|
| 第1卷 | | | | |
| 第2卷 | | | | |
| 第3卷 | | | | |
| 第X卷 | | | | |

### 更新计划

- **更新频率**：日更/隔日更/周更/不固定
- **发布时间**：每天XX:XX
- **存稿要求**：至少存XX章

### 验证清单

- [ ] 总章节数 ≥ 400章（百万字标准）
- [ ] 分卷数 ≥ 10卷
- [ ] 每卷章节数 ≥ 30章
- [ ] 每卷字数 ≥ 6万字
- [ ] 更新频率可持续
- [ ] 写作周期合理
```
