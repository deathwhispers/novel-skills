# 市场研究与商业化

## 功能定位

把市场信号、爆款技法和商业化包装整合成一个连贯的流程。这个 skill 负责：

1. **市场扫描**：分析榜单、平台差异、题材机会
2. **拆解爆款**：提炼钩子、人设、节奏、爽点等可迁移技法
3. **商业化包装**：书名、简介、开篇、追读、平台适配

## 工作流程

### 第一步：市场扫描（可选）

如果用户想了解市场趋势，执行以下流程：

1. **平台分析**：起点、番茄、晋江等平台的特点
2. **趋势分析**：当前热门题材、读者追读偏好
3. **产出**：`05-市场/趋势笔记.md`、`05-市场/对标书单.md`

### 第二步：拆解爆款（可选）

如果用户想分析爆款作品，执行以下流程：

1. **选择拆解对象**：用户提供的样章或对标书
2. **拆解维度**：钩子设计、人设塑造、节奏控制、爽点/虐点设计
3. **产出**：`05-市场/拆解/` 目录下的拆解报告

### 第三步：商业化包装（可选）

如果用户想做商业化改造，执行以下流程：

1. **书名优化**：提供多个书名选项
2. **简介优化**：优化简介结构，突出卖点和钩子
3. **开篇优化**：黄金三章设计，追读点布局
4. **平台适配**：根据目标平台调整风格
5. **产出**：`05-市场/商业化/` 目录下的包装方案

## 互动流程

**在与用户互动时，一次性确认多项需求**：

> 请问您需要哪些市场服务？
> - [ ] 市场趋势分析（了解当前热门题材和平台特点）
> - [ ] 爆款作品拆解（分析成功作品的技法）
> - [ ] 商业化包装（优化书名、简介、开篇）
> - [ ] 全部都需要
