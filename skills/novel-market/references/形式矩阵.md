## 形式三维度

每本书都可以从三个维度定位：

```
题材（世界观） × 类型（叙事模式） × 风格（表达方式）
```

**示例**：
```
《诡秘之主》= 蒸汽朋克（题材）× 悬疑升级（类型）× 克系暗调（风格）
《大奉打更人》= 架空历史（题材）× 探案升级（类型）× 轻松搞笑（风格）
```

## 维度 1：题材（世界观）

### 主流题材

| 题材 | 核心设定 | 读者期待 | 常见子题材 |
|------|----------|----------|------------|
| 玄幻 | 灵气/斗气/魔法 | 升级、战斗、奇遇 | 凡人流、废材流、系统流 |
| 仙侠 | 修仙、门派、长生 | 长生、逍遥、问道 | 凡人流、洪荒流、封神流 |
| 都市 | 现代背景、异能 | 逆袭、装逼、感情 | 重生、系统、神医、兵王 |
| 科幻 | 未来、外星、科技 | 探索、危机、进化 | 星际、末世、机甲、无限 |
| 历史 | 古代/近代背景 | 权谋、争霸、改变历史 | 穿越、架空、种田 |
| 悬疑 | 推理、惊悚、探险 | 解谜、紧张、反转 | 刑侦、灵异、盗墓 |
| 言情 | 感情为主线 | 甜蜜、虐心、成长 | 现言、古言、仙侠言、耽美 |
| 游戏 | 游戏世界/虚拟 | 攻略、竞技、冒险 | 网游、电竞、无限流 |
| 二次元 | 轻小说风格 | 日常、搞笑、冒险 | 同人、穿越动漫 |
| 灵异/恐怖 | 超自然、恐怖 | 恐怖、生存、揭秘 | 鬼怪、规则怪谈、克系 |

### 题材组合

单一题材不够独特时，可以组合：

| 组合 | 示例 | 独特点 |
|------|------|--------|
| 玄幻 + 悬疑 | 《诡秘之主》 | 升级 + 解谜 |
| 都市 + 仙侠 | 《我有一座冒险屋》 | 现代 + 灵异 |
| 历史 + 科幻 | 《临高启明》 | 古代 + 工业 |
| 言情 + 权谋 | 《甄嬛传》 | 感情 + 政治 |
| 玄幻 + 种田 | 《修真聊天群》 | 修仙 + 日常 |

> **核心规则**：组合不是"混搭"——是"用 A 题材的世界观讲 B 类型的故事"。

## 维度 2：类型（叙事模式）

### 主流类型

| 类型 | 核心机制 | 读者情绪 | 常见搭配 |
|------|----------|----------|----------|
| 升级流 | 主角逐步变强 | 爽、期待 | 玄幻/仙侠/都市 |
| 无敌流 | 主角开局最强 | 爽、轻松 | 都市/二次元 |
| 重生流 | 主角回到过去 | 先知优势、逆袭 | 都市/历史 |
| 系统流 | 主角有系统辅助 | 爽、期待 | 任何题材 |
| 种田流 | 主角逐步建设 | 满足、积累 | 历史/玄幻/科幻 |
| 权谋流 | 主角运筹帷幄 | 紧张、智爽 | 历史/玄幻 |
| 探案流 | 主角解谜破案 | 紧张、恍然大悟 | 悬疑/历史/玄幻 |
| 感情流 | 感情发展为主线 | 甜、虐、揪心 | 言情/任何题材 |
| 冒险流 | 主角探索未知 | 刺激、好奇 | 玄幻/科幻/悬疑 |
| 日常流 | 主角日常生活 | 轻松、温馨 | 都市/二次元 |
| 黑暗流 | 主角黑化/残酷 | 压抑、释放 | 玄幻/都市 |
| 搞笑流 | 以幽默为主 | 轻松、欢乐 | 任何题材 |

### 类型组合

| 组合 | 示例 | 效果 |
|------|------|------|
| 升级 + 探案 | 《大奉打更人》 | 变强 + 解谜，双重爽感 |
| 重生 + 种田 | 《临高启明》 | 先知 + 建设，满足延迟 |
| 系统 + 搞笑 | 《修真聊天群》 | 金手指 + 轻松，降低阅读压力 |
| 无敌 + 日常 | 《一拳超人》 | 最强 + 反差，制造喜剧 |
| 感情 + 权谋 | 《甄嬛传》 | 感情 + 政治，双重张力 |

## 维度 3：风格（表达方式）

### 主流风格

| 风格 | 特点 | 适合题材 | 读者感受 |
|------|------|----------|----------|
| 轻松搞笑 | 幽默、调侃、段子 | 都市/二次元/种田 | 放松、欢乐 |
| 热血燃向 | 激昂、战斗、牺牲 | 玄幻/仙侠/科幻 | 激动、共鸣 |
| 黑暗压抑 | 残酷、绝望、挣扎 | 玄幻/都市/悬疑 | 压抑、释放 |
| 文艺细腻 | 描写细腻、情感丰富 | 言情/仙侠/历史 | 沉浸、感动 |
| 快节奏爽 | 爽点密集、节奏快 | 都市/玄幻/番茄文 | 爽、停不下来 |
| 智斗谋略 | 算计、博弈、反转 | 历史/权谋/悬疑 | 智爽、恍然大悟 |
| 日常温馨 | 平淡、温暖、小确幸 | 都市/种田/二次元 | 温馨、治愈 |
| 恐怖惊悚 | 恐怖、紧张、未知 | 悬疑/灵异/科幻 | 紧张、刺激 |
| 史诗宏达 | 世界观大、格局大 | 玄幻/科幻/历史 | 震撼、沉浸 |
| 讽刺批判 | 讽刺、隐喻、批判 | 都市/历史/科幻 | 思考、共鸣 |

### 风格组合

| 组合 | 示例 | 效果 |
|------|------|------|
| 热血 + 黑暗 | 《进击的巨人》 | 燃 + 绝望，情绪冲击强 |
| 轻松 + 智斗 | 《大奉打更人》 | 搞笑 + 算计，降低阅读压力 |
| 文艺 + 热血 | 《天官赐福》 | 细腻 + 激昂，情感丰富 |
| 快节奏 + 史诗 | 《诡秘之主》 | 快 + 大，兼顾爽感和沉浸 |

## 形式创新检查

### 创新度评估

| 维度 | 常见组合 | 你的组合 | 创新度 |
|------|----------|----------|--------|
| 题材 | {{常见 A}} | {{你的}} | 高/中/低 |
| 类型 | {{常见 B}} | {{你的}} | 高/中/低 |
| 风格 | {{常见 C}} | {{你的}} | 高/中/低 |

**创新度判断**：
- 3 维都高 = 高风险高回报（读者可能不接受）
- 2 维高 1 维低 = 最佳平衡点（有新鲜感但不陌生）
- 1 维高 2 维低 = 安全但不够独特
- 3 维都低 = 同质化严重（不建议）

### 形式陷阱

| 陷阱 | 表现 | 修复 |
|------|------|------|
| 过度创新 | 题材+类型+风格都罕见 | 保留 1-2 个常见维度降低门槛 |
| 形式冲突 | 类型和风格矛盾 | 调整风格匹配类型 |
| 跟风同质化 | 3 维都和市场爆款一样 | 至少改变 1 个维度 |
| 形式大于内容 | 形式很新但故事空洞 | 先确保故事核扎实 |

## 形式矩阵自检

- [ ] 题材明确（读者一眼知道这是什么世界）
- [ ] 类型明确（读者知道这是什么叙事模式）
- [ ] 风格明确（读者知道这是什么阅读感受）
- [ ] 三维度不冲突（热血 + 黑暗可以，热血 + 日常难）
- [ ] 和直接对标至少有 1 个维度不同
- [ ] 形式组合有先例（不是完全没人做过的）
- [ ] 形式服务于故事核（不是为形式而形式）

---

## 常见形式错误

| 错误 | 表现 | 修复 |
|------|------|------|
| **过度创新** | 题材+类型+风格都罕见，读者不接受 | 保留 1-2 个常见维度 |
| **跟风同质化** | 3 维都和爆款一样，没有差异 | 至少改变 1 个维度 |
| **形式冲突** | 热血类型 + 日常风格，不兼容 | 调整风格匹配类型 |
| **形式大于内容** | 形式很新但故事空洞 | 先确保故事核扎实 |
| **题材不清** | "玄幻+都市+科幻"混搭 | 确定主题材，其他作为元素 |
| **类型不明** | "有点像升级流，又像无敌流" | 明确核心叙事机制 |

## 快速检查（2 分钟）

用一句话说清你的书：
1. 题材是什么？（玄幻/都市/言情...）
2. 类型是什么？（升级/重生/种田...）
3. 风格是什么？（热血/轻松/黑暗...）

**任一题说不清→形式定位不清，需要重新定义**
