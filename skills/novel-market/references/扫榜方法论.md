# 扫榜方法论

> **用途**：手把手教你怎么看网文平台榜单，不是"看榜单"——是"从榜单中提取可操作的信号"。
> 适用于 novel-market 第一步：市场扫描。

---

## 一、扫榜前准备

### 明确目标

扫榜前问自己：
1. **我要找什么？**（题材机会/对标作品/书名模式/开篇模式）
2. **在哪个平台扫？**（起点/番茄/晋江/其他）
3. **扫哪个榜？**（热销/新书/完结/推荐）
4. **扫多少本？**（前 30 名足够看出趋势）

### 工具准备

| 工具 | 用途 | 必须？ |
|------|------|--------|
| 表格/笔记 | 记录数据 | 是 |
| 浏览器 | 打开榜单 | 是 |
| 计时器 | 控制时间 | 建议 |
| 爬虫脚本 | 自动抓取（可选） | 否 |

> **时间控制**：单次扫榜 30-60 分钟，不要扫太久会疲劳。

---

## 二、扫榜五步法

### 第一步：看整体（5 分钟）

**做什么**：快速浏览前 30 名的书名，不看内容。

**记录什么**：
- 哪些题材反复出现？（玄幻/仙侠/都市...）
- 书名有什么共同模式？（"我"字开头/反差型/悬念型...）
- 哪些作者名字熟悉？（判断是否是"大神效应"）

**输出**：
```
前 30 名题材分布：玄幻 X 本，都市 X 本，言情 X 本...
书名模式：XX 型 X 本，XX 型 X 本...
熟悉作者：X 个（注意：这些可能是大神自带流量）
```

### 第二步：看头部（10 分钟）

**做什么**：精读前 3-5 名的书名、简介、前 3 章。

**记录什么**：

| 排名 | 书名 | 题材 | 核心卖点（1 句话） | 书名模式 | 简介钩子 | 开篇反常事件 |
|------|------|------|-------------------|----------|----------|--------------|
| 1 | | | | | | |
| 2 | | | | | | |
| 3 | | | | | | |

**关键问题**：
- 头部作品的成功是因为"大神效应"还是"技法过硬"？
- 如果换成新人作者，这本书还能火吗？

### 第三步：看新书（10 分钟）

**做什么**：看新书榜前 20 名。

**为什么看新书**：新书反映"当前读者最新偏好"，不是"老书长尾"。

**记录什么**：
- 新书榜和热销榜的题材分布差异
- 新书榜独有的"新方向"（老榜没有的题材/模式）
- 新书作者是否是新人（判断"新人能不能出头"）

### 第四步：看跨平台（10 分钟）

**做什么**：对比 2-3 个平台的榜单。

**为什么跨平台**：
- 起点火的，番茄可能不火（读者群不同）
- 两个平台都火的 = 真实大趋势
- 只有个别平台火的 = 平台特色，不一定适合跨平台

**记录什么**：

| 题材/模式 | 起点 | 番茄 | 晋江 | 判断 |
|-----------|------|------|------|------|
| XX 题材 | 前 10 有 X 本 | 前 10 有 X 本 | 前 10 有 X 本 | 跨平台趋势/平台特色 |

### 第五步：做判断（5 分钟）

**做什么**：回答 3 个问题。

1. **我的题材在榜单中处于什么位置？**
   - 前 30 名出现 0-2 本 → 小众/niche，有机会但需谨慎
   - 前 30 名出现 3-8 本 → 主流题材，有市场空间
   - 前 30 名出现 10+ 本 → 可能饱和，需要强差异化

2. **当前最火的书名/开篇/人设模式是什么？**
   - 记录 1-2 个可借鉴的模式
   - 不要直接复制，要"变形"

3. **我能做出什么差异化？**
   - 和头部作品比，我至少有 1 个维度不同

## 三、扫榜记录表

```markdown
# 扫榜记录

## 基本信息

- 日期：YYYY-MM-DD
- 平台：起点/番茄/晋江/其他
- 榜单：热销/新书/完结/推荐
- 范围：前 N 名
- 耗时：X 分钟

## 整体观察

| 维度 | 观察 |
|------|------|
| 题材分布 | 玄幻 X 本、都市 X 本、言情 X 本... |
| 书名模式 | "我"字开头 X 本、反差型 X 本... |
| 熟悉作者 | X 个（注意大神效应） |

## 头部分析

| 排名 | 书名 | 题材 | 核心卖点 | 可学点 | 可规避点 |
|------|------|------|----------|--------|----------|
| 1 | | | | | |
| 2 | | | | | |
| 3 | | | | | |

## 趋势判断

| 趋势 | 证据 | 可信度 | 对本书的启示 |
|------|------|--------|--------------|
| XX 题材上升 | 新书榜有 X 本 | 高/中/低 | 可以考虑 |

## 行动项

- [ ] 具体行动（截止日期）
```

## 四、扫榜频率

| 阶段 | 频率 | 深度 | 目的 |
|------|------|------|------|
| 立项前 | 连续 3 周，每周 1 次 | 前 30 名 | 确认题材机会 |
| 开卷前 | 每周 1 次 | 前 20 名 | 跟踪趋势变化 |
| 卡文时 | 即时 | 浅扫前 10 名 | 找灵感 |
| 连载中 | 每月 1 次 | 跨平台对比 | 调整方向 |

---

## 五、扫榜纪律

1. **不要只扫一次**——连续扫 3 周才能确认趋势
2. **不要只扫一个平台**——跨平台对比发现真实趋势
3. **不要只看题材**——书名、开篇、人设模式同样重要
4. **不要数据迷信**——数据好不等于值得学，要看"为什么好"
5. **不要凭记忆**——用表格记录，避免记忆失真
6. **不要扫太久**——30-60 分钟足够，长了会疲劳

---

## 常见扫榜错误

| 错误 | 表现 | 修复 |
|------|------|------|
| **无目标扫榜** | "看看最近什么火"——没有明确要找什么 | 扫榜前明确：找题材/对标/书名模式/开篇模式 |
| **只看数据** | "这本收藏 10 万+，所以好" | 剥离大神效应，看技法是否过硬 |
| **不记录** | 看完觉得"学到了"但什么都说不出来 | 强制填写扫榜记录表 |
| **疲劳扫榜** | 一次扫 3 小时，后面全是走马观花 | 控制时间 30-60 分钟 |
| **不行动** | 扫完不分析，分析完不行动 | 每次扫榜必须有 1 个行动项 |
| **盲目跟风** | "这个题材火，我也写"——不考虑自己是否擅长 | 问自己：这个题材我能写出独特之处吗？ |

---

## 快速检查（3 分钟）

扫完榜单后：
1. 我能说出前 30 名中前 3 名的核心卖点吗？
2. 我能说出当前最火的书名模式吗？
3. 我能说出我的题材在榜单中处于什么位置吗？
4. 我找到了 1 个可借鉴的技法吗？

**任一题"否"→重新扫榜，这次带着问题看**
