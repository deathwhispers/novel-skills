# 大纲规划

## 功能定位

把全书总纲、分卷卷纲、章节节拍卡整合成一个连贯的流程。这个 skill 负责：

1. **全书总纲**：主线核心矛盾、冲突演化路径、叙事预算
2. **分卷卷纲**：每卷在全书中扮演的角色、开卷/中段/卷末规划
3. **章节节拍卡**：每章戏剧任务、阻力、局面变化、章尾钩子

## 工作流程

### 第一步：全书总纲（与用户互动）

**一次性确认以下信息**：

1. **核心冲突**：用一句话说清整本书不断升级的那一个最核心的胜负、追求或冲突
2. **冲突演化路径**：核心矛盾在不同阶段会变形
3. **主角成长弧线**：起点状态和终点状态
4. **叙事预算**：总字数、总章节数、分卷数

### 第二步：分卷卷纲（与用户互动）

**一次性确认以下信息**：

1. **确定卷数**：根据叙事预算确定全书分几卷
2. **逐卷建立骨架**：定位、卖点、开卷/卷末、核心转折
3. **逐卷填充细节**：中段升级路径、主线推进量、卷末钩子

### 第三步：章节节拍卡（与用户互动）

**一次性确认以下信息**：

1. **确定章节数预算**：根据卷字数确定章节数
2. **分配高潮点**：按「起→承→转→合」四段分配
3. **为每一章锁定核心戏剧任务**：每章至少要明确9项信息
