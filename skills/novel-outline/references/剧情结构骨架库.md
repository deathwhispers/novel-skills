# 剧情结构骨架库

> **使用说明**：选择适合你故事的剧情结构型。
> 不是"套用模板"——是"用结构型帮你检查故事是否完整"。
> 适用于重构流程 Step 1（重审总纲）。

## 骨架选择表

| 你的故事特征 | 推荐骨架 | 次选 |
|-------------|---------|------|
| 主角从弱变强，一路升级 | 英雄之旅 | 三幕式 |
| 主角身份揭秘，反转不断 | 身份揭秘型 | 英雄之旅 |
| 多主角/群像，多条线并进 | 多线并进型 | 网状结构 |
| 悬疑推理，层层揭秘 | 悬疑揭秘型 | 三幕式 |
| 感情为主，关系递进 | 关系递进型 | 英雄之旅 |
| 穿越/重生，先知优势 | 先知改命型 | 英雄之旅 |
| 种田/基建，逐步发展 | 螺旋上升型 | 英雄之旅 |
| 复仇主线，黑化/洗白 | 复仇螺旋型 | 三幕式 |

## 骨架 1：三幕式（经典通用）

### 结构

```
第一幕（25%）：设定 + 冲突引入
  └── 触发事件 → 主角被迫行动

第二幕（50%）：对抗 + 升级
  └── 中点反转 → 主角发现更大的威胁

第三幕（25%）：高潮 + 解决
  └── 最终对决 → 不可逆结局
```

### 检查清单

- [ ] 第一幕结束时，主角已"无法回头"
- [ ] 第二幕中点有"真相揭露"或"重大损失"
- [ ] 第三幕高潮有"终极代价"
- [ ] 结局有"不可逆变化"

### 适用

大多数网文，尤其是玄幻/仙侠/都市升级文。

---

## 骨架 2：英雄之旅（Campbell 模型简化）

### 结构

```
1. 平凡世界 ──→ 2. 冒险召唤
   ↓
3. 拒绝召唤 ──→ 4. 遇见导师
   ↓
5. 跨越门槛 ──→ 6. 试炼/盟友/敌人
   ↓
7. 深入洞穴 ──→ 8. 磨难
   ↓
9. 奖赏 ──→ 10. 归途
   ↓
11. 复活 ──→ 12. 携宝归乡
```

### 网文适配

| 原阶段 | 网文对应 | 常见位置 |
|--------|----------|----------|
| 平凡世界 | 主角的初始状态 | 开篇 1-3 章 |
| 冒险召唤 | 触发事件 | 第 3-5 章 |
| 拒绝召唤 | 主角犹豫/准备 | 第 5-8 章 |
| 遇见导师 | 获得指引/功法 | 第 8-12 章 |
| 跨越门槛 | 离开安全区 | 第 12-15 章 |
| 试炼 | 连续战斗/考验 | 第 15-30 章 |
| 深入洞穴 | 面对核心敌人 | 卷中高潮 |
| 磨难 | 重大失败/失去 | 卷末/转折卷 |
| 奖赏 | 获得关键能力/信息 | 高潮后 |
| 归途 | 返回/新挑战 | 下一卷起 |
| 复活 | 重生/突破 | 大高潮 |
| 携宝归乡 | 新世界/新地位 | 结局 |

### 适用

玄幻/仙侠/奇幻/冒险类网文。

## 骨架 3：身份揭秘型

### 结构

```
第一层：主角表面的身份（废物/普通人/被追杀者）
  ↓
铺垫期：不断暗示主角"不普通"（10-30 章）
  ↓
第一层揭秘：主角的真实能力/身份被少数人发现（30-50 章）
  ↓
第二层铺垫：暗示主角身份背后还有更深的秘密（50-100 章）
  ↓
第二层揭秘：主角的终极身份/使命被揭示（100-150 章）
  ↓
第三层铺垫：终极身份背后还有更大的局（150-250 章）
  ↓
第三层揭秘：全书最大反转（250-400 章）
```

### 关键设计

| 层次 | 铺垫密度 | 揭秘方式 | 读者反应 |
|------|----------|----------|----------|
| 第一层 | 轻（1-2 处/章） | 被事件逼出 | "原来如此！" |
| 第二层 | 中（2-3 处/章） | 被敌人揭露 | "还有更深？！" |
| 第三层 | 重（3-5 处/章） | 主角主动揭示/终极对决 | "全明白了！" |

### 适用

凡人流、废材流、马甲流、赘婿流。

## 骨架 4：多线并进型

### 结构

```
主线 A：主角的成长线
  └── 触发 → 试炼 → 突破 → 代价 → 终极

主线 B：感情线
  └── 相遇 → 冲突 → 和解 → 深化 → 考验

主线 C：势力/世界线
  └── 发现 → 卷入 → 对抗 → 博弈 → 重塑

主线 D：悬疑/揭秘线
  └── 疑问 → 线索 → 推理 → 反转 → 真相

交汇点：多线在某章交汇，产生复合高潮
```

### 线间关系

| 关系类型 | 说明 | 示例 |
|----------|------|------|
| 互补 | A 线的突破推动 B 线 | 主角变强 → 感情线推进 |
| 冲突 | A 线和 B 线互相阻碍 | 主角追求力量 → 牺牲感情 |
| 并行 | A 线和 B 线独立推进 | 主角修炼 + 感情发展 |
| 隐藏 | C 线在暗处推进，最后揭露 | 势力博弈 → 最后揭晓 |

### 适用

群像文、权谋文、复杂世界观文。

---

## 骨架 5：悬疑揭秘型

### 结构

```
谜团引入：一个核心谜团（谁杀的/谁背叛的/秘密是什么）
  ↓
线索铺设：每章释放 1-2 条线索（真线索 + 假线索）
  ↓
推理推进：主角/读者根据线索推理
  ↓
中点反转：之前的推理被推翻（假线索被揭露）
  ↓
新线索层：更深的谜团出现
  ↓
终极揭秘：真相大白 + 代价支付
```

### 线索管理

| 线索类型 | 数量 | 作用 |
|----------|------|------|
| 真线索 | 60% | 指向真相 |
| 假线索 | 30% | 误导读者/主角 |
| 红鲱鱼 | 10% | 完全无关，增加复杂度 |

### 适用

悬疑、推理、惊悚、谍战。

## 骨架 6：关系递进型

### 结构

```
阶段 1：陌生人（1-10 章）
  └── 第一印象 + 冲突/误会

阶段 2：初识（11-20 章）
  └── 互相了解 + 小互动

阶段 3：朋友（21-40 章）
  └── 共同经历 + 互相帮助

阶段 4：暧昧（41-80 章）
  └── 心动 + 试探 + 信息差

阶段 5：表白（81-100 章）
  └── 明确关系 + 新压力

阶段 6：恋人/伴侣（101-150 章）
  └── 甜蜜 + 外部阻力

阶段 7：考验（151-200 章）
  └── 危机 + 信任测试

阶段 8：归宿（200+ 章）
  └── 圆满/开放式
```

### 每个阶段的"标志事件"

| 阶段 | 必须发生 | 不能发生 |
|------|----------|----------|
| 陌生人 | 第一次相遇 | 立刻有好感 |
| 初识 | 知道对方名字+身份 | 立刻信任 |
| 朋友 | 第一次互相帮助 | 单方面付出 |
| 暧昧 | 双方意识到感情 | 立刻表白 |
| 表白 | 明确关系 | 没有阻力就在一起 |
| 恋人 | 外部阻力出现 | 一帆风顺 |
| 考验 | 重大危机测试感情 | 轻易通过 |

### 适用

言情、耽美、感情流。

## 骨架 7：先知改命型

### 结构

```
1. 主角获得先知信息（重生/穿越/系统/预知）
   ↓
2. 利用先知优势改变小事（积累优势）
   ↓
3. 蝴蝶效应：改变小事引发连锁反应（意外）
   ↓
4. 先知失效：某个关键节点和前世不同（危机）
   ↓
5. 重新适应：主角在没有先知的情况下应对（成长）
   ↓
6. 超越先知：主角做得比前世更好（终局）
```

### 关键设计

| 阶段 | 核心张力 | 读者情绪 |
|------|----------|----------|
| 先知优势 | "我知道未来" | 爽（信息不对称） |
| 蝴蝶效应 | "但事情开始变了" | 紧张（失控感） |
| 先知失效 | "未来和前世不一样了" | 危机（优势丧失） |
| 超越先知 | "但我比前世更强" | 满足（成长） |

### 适用

重生文、穿越文、系统文。

---

## 骨架 8：螺旋上升型

### 结构

```
循环 1：发现问题 → 尝试解决 → 失败/代价 → 总结经验
  ↓
循环 2：新问题 → 用经验解决 → 新失败/代价 → 新经验
  ↓
循环 3：更大问题 → 综合经验 → 部分成功 → 更大代价
  ↓
...
终极循环：终极问题 → 全部经验 → 成功 → 终极代价
```

### 每个循环的升级

| 维度 | 循环 1 | 循环 2 | 循环 3 | ... |
|------|--------|--------|--------|-----|
| 问题规模 | 个人 | 团队 | 势力 | 世界 |
| 解决方式 | 个人能力 | 团队协作 | 势力博弈 | 规则改变 |
| 代价 | 个人损失 | 团队损失 | 势力损失 | 不可逆代价 |
| 经验 | 个人成长 | 团队默契 | 势力运营 | 世界认知 |

### 适用

种田文、基建文、经营文、末日求生。

---

## 骨架 9：复仇螺旋型

### 结构

```
1. 仇恨起源：主角遭受重大伤害/失去重要东西
   ↓
2. 隐忍积蓄：主角隐藏实力，暗中准备（长期铺垫）
   ↓
3. 第一次报复：对次要敌人出手（小爽点）
   ↓
4. 发现真相：仇人背后还有更大的阴谋（升级）
   ↓
5. 第二次报复：对主要敌人出手（中高潮）
   ↓
6. 代价支付：复仇让主角失去更重要的东西（虐点）
   ↓
7. 选择：继续复仇 vs 放下（终极考验）
   ↓
8. 终局：复仇完成，但主角已不同（代价不可逆）
```

### 关键设计

| 阶段 | 必须有 | 不能有 |
|------|--------|--------|
| 隐忍积蓄 | 主角的成长过程 | 主角突然变强 |
| 第一次报复 | 爽点 | 一次性解决所有仇人 |
| 发现真相 | 更大的阴谋 | 仇人只是"单纯的坏" |
| 代价支付 | 不可逆失去 | 复仇无代价 |
| 终局 | 主角的改变 | 主角回到原点 |

### 适用

复仇文、黑暗流、虐主流。

## 骨架自检

选定骨架后，检查：

- [ ] 骨架匹配故事核心（升级/揭秘/感情/复仇）
- [ ] 每个阶段都有"不可逆事件"（不是"差点发生"）
- [ ] 主角状态每阶段都有变化（不是"变强了"这种空话）
- [ ] 升级/转折有代价（不是白得的）
- [ ] 结局和起点形成反差（不是"更强了"）
- [ ] 骨架不 rigid——允许局部偏离主线

> **核心规则**：骨架是"地图"不是"牢笼"——写到中段如果发现更好的走向，可以调整骨架，但要记录调整原因到 `90-运行/决策记录.md`。

## 骨架选择决策树

> **用途**：当你不确定该选哪种骨架时，按这个决策树逐步判断。

```
第一步：你的故事最核心的是什么？
  ├─ 主角从弱变强，一路升级 → 英雄之旅
  ├─ 主角身份/真相层层揭秘 → 身份揭秘型
  ├─ 多个主角/群像并进 → 多线并进型
  ├─ 悬疑推理，破案/解谜 → 悬疑揭秘型
  ├─ 感情为主，关系递进 → 关系递进型
  ├─ 穿越/重生，先知优势 → 先知改命型
  ├─ 种田/基建，逐步发展 → 螺旋上升型
  └─ 复仇主线，黑化/洗白 → 复仇螺旋型

第二步：你的主角是什么类型？
  ├─ 成长性主角（从弱到强） → 英雄之旅 / 螺旋上升
  ├─ 解密型主角（寻找真相） → 身份揭秘 / 悬疑揭秘
  ├─ 情感型主角（关系驱动） → 关系递进
  ├─ 先知型主角（知道未来） → 先知改命
  └─ 复仇型主角（有明确仇人） → 复仇螺旋

第三步：你的故事规模？
  ├─ 短篇（<30 万字） → 三幕式（简化版）
  ├─ 中篇（30-80 万字） → 标准骨架
  └─ 长篇（80 万+ 字） → 复合骨架（主骨架 + 副骨架）

第四步：是否需要混合骨架？
  ├─ 主线是升级，副线是复仇 → 英雄之旅 + 复仇螺旋
  ├─ 主线是揭秘，副线是感情 → 身份揭秘 + 关系递进
  ├─ 主线是悬疑，副线是升级 → 悬疑揭秘 + 英雄之旅
  └─ 单一主线 → 一种骨架即可
```

## 混合骨架方法

> **用途**：当故事有多条主线时，用不同骨架组合。

### 混合原则

- 主骨架决定全书节奏和关键节点
- 副骨架决定局部结构和角色弧线
- 混合最多 2 种，不要 3 种以上
- 副骨架的节点必须和主骨架的关键节点对齐

### 常见组合

| 组合 | 主骨架 | 副骨架 | 效果 | 示例 |
|------|--------|--------|------|------|
| **升级+复仇** | 英雄之旅 | 复仇螺旋 | 主角变强的同时逐步复仇 | 玄幻复仇文 |
| **揭秘+感情** | 身份揭秘 | 关系递进 | 真相揭露影响关系变化 | 悬疑言情 |
| **悬疑+升级** | 悬疑揭秘 | 英雄之旅 | 破案过程中主角变强 | 侦探修真 |
| **种田+揭秘** | 螺旋上升 | 身份揭秘 | 发展势力同时揭开身世 | 基建种田 |

### 混合节点对齐

```
示例：英雄之旅（主） + 复仇螺旋（副）

英雄之旅节点        复仇螺旋节点          对齐方式
─────────────────────────────────────────────────
冒险召唤            仇恨起源              同时发生
第一道门槛          隐忍积蓄（开始）      主角开始准备
深渊                第一次报复            主角对次要敌人出手（小爽点）
变形/觉醒           发现真相              主角发现仇人背后更大阴谋
第二道门槛          第二次报复            主角对主要敌人出手
终极考验            代价支付              复仇让主角失去更重要的东西
归来                终局                  复仇完成，主角已不同
```

## 骨架适配指南

> **用途**：选定骨架后，根据你的故事类型进行微调。

### 玄幻/仙侠适配

| 骨架 | 适配要点 | 常见调整 |
|------|---------|---------|
| 英雄之旅 | 把"冒险召唤"换成"觉醒灵根" | 门槛 = 境界突破 |
| 螺旋上升 | 把"种田"换成"修炼" | 每次循环 = 一个小境界 |
| 复仇螺旋 | 把"隐忍积蓄"换成"闭关修炼" | 报复 = 击杀仇人的手下 |

### 都市/职场适配

| 骨架 | 适配要点 | 常见调整 |
|------|---------|---------|
| 三幕式 | 把"触发事件"换成"项目危机" | 高潮 = 项目交付/商业对决 |
| 身份揭秘 | 把"身份"换成"职场秘密" | 揭秘 = 发现公司阴谋 |
| 螺旋上升 | 把"种田"换成"业务拓展" | 每次循环 = 一个项目周期 |

### 悬疑/推理适配

| 骨架 | 适配要点 | 常见调整 |
|------|---------|---------|
| 悬疑揭秘 | 保持原样 | 假真相 = 误导线索 |
| 身份揭秘 | 把"身世"换成"凶手身份" | 假身份 = 嫌疑人 |
| 先知改命 | 把"先知"换成"推理预判" | 改命 = 阻止犯罪 |

### 言情/感情适配

| 骨架 | 适配要点 | 常见调整 |
|------|---------|---------|
| 关系递进 | 保持原样 | 表白 = 第一次确定关系 |
| 英雄之旅 | 把"冒险"换成"追求" | 门槛 = 误会/阻碍 |
| 身份揭秘 | 把"身世"换成"前任/秘密" | 揭秘 = 发现对方隐瞒的事 |

## 骨架快速检查（3 分钟）

回答这 3 个问题：

1. 你的故事用一句话说，最核心的是什么？（升级/揭秘/感情/复仇/悬疑）
2. 主角的起点和终点最大的反差是什么？（弱→强？无知→全知？单身→伴侣？）
3. 全书有几个"不可逆事件"？（每个阶段至少 1 个）

**评分**：
- 3 题都能清晰回答：骨架健康
- 1-2 题说不清：需要补充对应部分
- 3 题都说不清：需要回到"大纲阶梯"第 2 阶
