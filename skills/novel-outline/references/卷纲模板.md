> 文件名：`20-大纲/分卷/volume-XX.md`

## 一、卷级定位

### 1.1 在全书中扮演什么角色

用 1-2 句话说清这卷在全书中独一无二的功能：

```
这卷是 {{全书阶段}} 的 {{具体功能}}——
它负责 {{建立/升级/转折/深化/收束}} {{什么}}，
为 {{下一卷}} 的 {{什么}} 做铺垫。
```

**示例**：
```
这卷是全书"起势期"的"立人设+埋主线"——
它负责建立主角的初始状态和被追杀的核心矛盾，
为第 2 卷的"逃亡+成长"做铺垫。
```

### 1.2 读者读完这卷应该有什么感受

| 感受类型 | 具体描述 |
|----------|----------|
| 爽 | 读者觉得"主角终于……" |
| 虐 | 读者觉得"太惨了，希望主角能……" |
| 悬 | 读者觉得"到底怎么回事，下一卷必须看" |
| 甜 | 读者觉得"这对 CP 我磕了" |
| 燃 | 读者觉得"热血，主角冲啊" |
| 其他 | {{自定义}} |

**不要写"多种感受"**——每卷聚焦 1-2 种主感受。

## 二、开卷设计

### 2.1 开卷承诺（第 1 章必须给读者的）

读者读完第 1 章后，应该明确知道：

- [ ] 主角当前的状态（位置/能力/目标）
- [ ] 本卷的核心悬念/压力是什么
- [ ] 读者追下去能看到什么（爽点/虐点/悬念）

**开卷承诺的具体写法**：

```
第 1 章结束时，读者应该知道：
1. 主角 {{现在在做什么 / 面临什么}} 
2. 接下来会发生 {{什么类型的事}}（战斗/揭秘/感情推进/其他）
3. 这件事的 {{悬念/压力/期待点}} 是什么
```

### 2.2 开卷钩子（第 1 章的章尾发动机）

从 6 种中选 1 种：

- [ ] 新问题：开卷抛出新的核心问题
- [ ] 不可逆决定：主角做了无法回头的决定
- [ ] 倒计时：某件事即将发生
- [ ] 权力反转：局势突然改变
- [ ] 情绪余震：强烈的情绪冲击
- [ ] 关系破口：重要关系出现裂缝

### 3.2 中段节奏分布

本卷选择哪种节奏型（详见 `卷级节奏型模板.md`）：

- [ ] 标准型：起 → 承 → 小高潮 → 承 → 大高潮 → 收束
- [ ] 加速型：起 → 承 → 高潮 → 更高潮 → 终极大高潮
- [ ] 波浪型：起 → 小高潮 → 回落 → 中高潮 → 回落 → 大高潮
- [ ] 慢热型：长铺垫 → 一次大爆发 → 余震

> 节奏强度：1=极慢（过渡），3=正常（推进），5=极快（高潮）

## 九、自检清单

卷纲完成后，逐项检查：

- [ ] 功能定位明确（一句话能说清这卷在全书中扮演什么）
- [ ] 核心卖点聚焦（不超过 2 种主感受）
- [ ] 开卷承诺具体（读者读完第 1 章知道要追什么）
- [ ] 卷末钩子强（不是"平静收束"）
- [ ] 核心转折不可逆（不是"差点发生"）
- [ ] 铺垫分布散（不集中在一两章）
- [ ] 升级有代价（不是白得的）
- [ ] 字数预算合理（高潮卷可长，过渡卷可短）
- [ ] 跨卷衔接顺（本卷末接下一卷首）
- [ ] 角色弧线有变化（不只是主角变强）

> **核心规则**：10 项中若有 3 项以上说不清，回到对应阶梯重审。

## 十、填写示例（完整版）

> **用途**：展示一个填好的卷纲长什么样，供参考对比。

### 示例：第 3 卷《血火试炼》

```markdown
# 第 3 卷：血火试炼

- **卷号**：第 3 卷（全书共 12 卷）
- **卷名**：血火试炼
- **功能定位**：发展（深化）
- **核心卖点**：爽点 + 悬念
```

#### 一、卷级定位

**在全书中扮演什么角色**：
这卷是全书"发展期"的"能力质变+信任危机"——
它负责让主角从"有潜力的新人"升级为"能独当一面的战士"，
同时通过导师的背叛让主角失去对权威人物的信任，
为第 4 卷的"孤身对抗组织"做铺垫。

**读者读完这卷应该有什么感受**：
爽（主角终于证明自己）+ 悬（导师到底站在哪边？）

#### 二、开卷承诺（第 1-3 章）

**卷首钩子**：主角通过了入门测试，但发现测试内容和自己家族的灭门惨案有关。

**读者期待**：
1. 主角能在正式训练中获得什么独特能力？
2. 灭门惨案的真相会和训练有什么关系？
3. 那个在测试中帮助主角的神秘人是谁？

#### 三、中段升级（第 4-25 章）

**能力升级**：
- 获得"血火战气"（家族秘传，但主角以为是普通功法）
- 代价：每次使用会消耗生命力，但主角不知情
- 限制：需要情绪达到极端愤怒才能发动

**关系升级**：
- 与导师：从信任到怀疑（发现导师隐瞒了测试真相）
- 与战友：从竞争到合作（共同面对外部威胁）

**地位升级**：
- 从"新人"到"正式成员"
- 但发现"正式成员"的代价是失去部分自由

#### 四、卷末高潮（第 26-30 章）

**核心冲突**：
组织下达了任务：消灭主角家族的残余势力。
主角发现：家族的灭门案正是组织策划的。
导师的立场：导师早就知道，但一直隐瞒。

**主角决定**：
表面接受任务，暗中调查真相。
代价：必须欺骗最信任的战友。

**卷末钩子**：
主角在任务现场发现了自己"已死"的亲人还活着——但被组织控制了。

#### 五、字数与节奏

- **计划字数**：8 万字
- **预计章节**：35 章
- **节奏型**：波浪型（有两次小高潮：获得血火战气、战友合作；一次大高潮：发现真相）

#### 六、自检结果

- [x] 功能定位明确
- [x] 核心卖点聚焦（爽+悬）
- [x] 开卷承诺具体
- [x] 卷末钩子强（亲人还活着）
- [x] 核心转折不可逆（主角知道真相，无法回到"无知"状态）
- [x] 铺垫分布散（测试、功法、导师行为都有铺垫）
- [x] 升级有代价（生命力消耗）
- [x] 字数预算合理
- [x] 跨卷衔接顺（第 4 卷主角将孤身对抗组织）
- [x] 角色弧线有变化（主角失去信任）
```

## 十一、常见填写错误

| 错误 | 表现 | 修复 |
|------|------|------|
| 功能定位太泛 | "这卷是发展期"——没有具体功能 | 用"负责……为……做铺垫"句式 |
| 核心卖点太多 | "爽+虐+悬+甜+燃"——什么都想要 | 聚焦 1-2 种，其他作为辅助 |
| 开卷承诺模糊 | "主角开始了新旅程" | 具体说读者会期待什么 |
| 中段升级无代价 | "主角变强了" | 加代价：生命/信任/自由/道德 |
| 高潮可逆 | "差点发现真相" | 必须是"已经知道，无法回头" |
| 卷末钩子太弱 | "主角准备出发" | 用"追读钩子设计.md"检查强度 |
| 角色无弧线 | "配角帮助了主角" | 配角也有自己的欲望和变化 |
| 伏笔无计划 | "埋了一个伏笔" | 必须有"计划回收卷"和"回收方式" |
| 字数不现实 | 标准卷写了 50 章 | 按"字数反推→节奏分配"重新计算 |
| 跨卷断连 | 本卷末和下卷首无关 | 确保下卷首在 3 章内回应本卷末 |

## 十二、卷纲快速检查（5 分钟）

回答这 5 个问题，每个不超过 20 字：

1. 这卷在全书中独一无二的功能是什么？
2. 读者读完这卷最强烈的感受是什么？
3. 卷末什么事让读者必须看下一卷？
4. 主角这卷失去了什么/得到了什么（不可逆）？
5. 如果跳过这卷，全书哪里会断掉？

**评分**：
- 能回答 5 题：卷纲健康
- 能回答 3-4 题：需要补强对应部分
- 能回答 <3 题：需要回到"大纲阶梯"重审
