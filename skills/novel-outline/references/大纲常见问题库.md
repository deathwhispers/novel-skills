## 一、结构问题（10个）

### Q1：主线不清晰

**症状**：说不出"这本书讲什么"。
**诊断**：
1. 能用一句话说清核心冲突吗？
2. 这句话里有主角、追求、阻力吗？
**修复**：
- 回到 `大纲阶梯.md` 第 1 阶
- 填空：一个 {{状态}} 的 {{主角}}，因为 {{触发事件}}，必须在 {{限制}} 内 {{追求}}，但 {{阻力}} 让这件事不断升级。

### Q2：节奏太平

**症状**：每章都差不多，没有起伏。
**诊断**：
1. 画出紧张度曲线——是否一直在 3 左右？
2. 是否有"低谷"让读者喘息？
**修复**：
- 检查 `卷级节奏型模板.md`
- 确保每卷有：起 → 承 → 小高潮 → 承 → 大高潮
- 小高潮和大高潮的强度差距至少 2 级

### Q3：卷与卷脱节

**症状**：读完卷一，不觉得必须读卷二。
**诊断**：
1. 卷一结尾是否有"非读不可"的钩子？
2. 卷二开头是否直接回应卷一结尾？
**修复**：
- 卷一结尾必须抛出新问题或恶化旧问题
- 卷二开头必须在 3 章内回应卷一结尾的事件

### Q4：高潮太弱

**症状**：高潮只是"主角打赢了"，没有冲击力。
**诊断**：
1. 高潮时主角付出了什么代价？
2. 有什么意外发生？
3. 世界有什么不可逆的变化？
**修复**：
- 给高潮增加 3 个元素之一：代价 / 意外 / 不可逆变化
- 参考 `剧情结构骨架库.md` 的高潮设计

### Q5：升级疲劳

**症状**：主角每卷只是"变强了"，读者觉得重复。
**诊断**：
1. 除了能力，还有什么维度在升级？
2. 升级的代价是否越来越大？
**修复**：
- 引入非能力升级：心理崩溃、关系破裂、地位丧失
- 让升级的代价累积：旧伤复发、信任丧失、道德污点

## 二、角色问题（10个）

### Q6：主角没有成长

**症状**：卷一和卷末的主角完全一样。
**诊断**：
1. 主角的核心信念有变化吗？
2. 主角的欲望有变化吗？
**修复**：
- 设计"信念挑战"：每卷让主角的核心信念受到一次冲击
- 参考 `角色档案模板.md` 的成长弧线部分

### Q7：反派太弱

**症状**：反派只是"坏"，没有让读者感到威胁。
**诊断**：
1. 反派有明确的欲望和恐惧吗？
2. 反派有让主角"无法忽视"的手段吗？
**修复**：
- 给反派一个"优势"：信息优势、资源优势、道德优势
- 让反派在主角最脆弱的时候出手

### Q8：配角工具人

**症状**：配角只为主角服务，没有自己的故事线。
**诊断**：
1. 配角有自己的欲望和恐惧吗？
2. 配角有自己的升级节点吗？
**修复**：
- 给每个重要配角设计 1 个独立的小弧线
- 让配角的选择影响主线

### Q9：爱情线突兀

**症状**：两个人突然相爱，没有铺垫。
**诊断**：
1. 他们有多少次"非必要"的交集？
2. 他们有共同的秘密或创伤吗？
**修复**：
- 增加"共同经历"：一起面对危险、共享秘密
- 增加"互救"：不是单向拯救，是互相拯救

### Q10：角色太多记不住

**症状**：10 章出场了 20 个角色，读者分不清。
**诊断**：
1. 每个角色有独特的"声音标记"吗？
2. 每个角色有明确的功能定位吗？
**修复**：
- 合并功能重复的角色
- 给每个角色一个"记忆点"：口头禅、习惯动作、外貌特征

## 三、钩子与悬念问题（10个）

### Q11：章尾没有钩子

**症状**：每章结尾都是"主角休息了"或"主角准备出发"。
**诊断**：
1. 章尾是否有让读者想知道的事？
2. 读者会立刻点下一章吗？
**修复**：
- 参考 `追读钩子设计.md`
- 确保每章结尾有至少 ★★ 强度的钩子

### Q12：悬念太久不收

**症状**：读者等了 50 章还没等到答案，失去耐心。
**诊断**：
1. 核心悬念已经悬置多久？
2. 期间有"部分揭示"维持兴趣吗？
**修复**：
- 长悬念每 5-10 章给一次"部分揭示"
- 或每 5-10 章给一次"假揭示"（看似答案，实际更深）

### Q13：伏笔埋了忘收

**症状**：读者注意到某个细节，但后文再也没提。
**诊断**：
1. 有伏笔追踪表吗？
2. 每个伏笔都有回收计划吗？
**修复**：
- 使用 `伏笔设计指南.md` 的追踪表
- 每卷检查：已埋的伏笔是否有回收计划

### Q14：信息释放太急

**症状**：一章把所有信息都说了，后面没有可揭示的。
**诊断**：
1. 这章释放了多少新信息？
2. 这些信息是否引发新问题？
**修复**：
- 参考 `信息释放节奏.md`
- 每次释放信息时，确保引发 1 个新问题

### Q15：信息释放太慢

**症状**：5 章没有新信息，读者觉得"没什么发生"。
**诊断**：
1. 这 5 章每章都有新信息吗？
2. 没有新信息的章节在做什么？
**修复**：
- 没有新信息的章节必须有：关系推进 / 角色展示 / 氛围营造
- 否则删除或合并

## 四、场景与战斗问题（10个）

### Q16：战斗只是数字比拼

**症状**：战斗描述是"A 打了 B，B 反击，A 更强，赢了"。
**诊断**：
1. 战斗中有策略变化吗？
2. 战斗中有信息差利用吗？
**修复**：
- 参考 `战斗系统设计.md`
- 每场战斗至少 1 次"以弱胜强"或"信息差利用"

### Q17：场景只是背景板

**症状**：场景换了一个又一个，但对剧情没影响。
**诊断**：
1. 场景本身制造了冲突或解决冲突吗？
2. 角色对场景有情感联结吗？
**修复**：
- 让场景参与剧情：地形限制、环境危险、文化冲突
- 或让场景承载情感：故乡、创伤地点、圣地

### Q18：对话只是信息传递

**症状**：对话是"A 告诉 B 发生了什么事"。
**诊断**：
1. 对话中有潜台词吗？
2. 对话中有关系张力吗？
**修复**：
- 对话必须同时做 3 件事之一：推进剧情 / 揭示角色 / 推进关系
- 避免"汇报式"对话

## 五、世界观问题（10个）

### Q19：设定只是装饰

**症状**：世界设定很详细，但没有影响角色的选择。
**诊断**：
1. 设定让角色的什么选择变难了？
2. 设定制造了哪些冲突？
**修复**：
- 回到 `世界观构建指南.md`
- 每个设定必须回答：它会影响什么剧情或选择？

### Q20：战力崩坏

**症状**：主角突然变强，或反派突然变弱，没有合理解释。
**诊断**：
1. 升级有明确的晋升过程吗？
2. 代价系统被遵守了吗？
**修复**：
- 参考 `力量体系设计.md`
- 每次升级必须有代价，每次战斗必须有代价

## 六、快速诊断流程

### 5 分钟大纲体检

回答这 10 个问题，每个回答不超过 20 字：

1. 主角是谁？（名字 + 身份）
2. 他想要什么？（具体目标）
3. 什么在阻止他？（具体阻力）
4. 失败了会怎样？（具体代价）
5. 第一卷结束时他有什么变化？
6. 最后一卷结束时他有什么变化？
7. 全书最让读者意外的情节是什么？
8. 全书最让读者感动的情节是什么？
9. 读者为什么必须追下一卷？
10. 读者为什么必须追下一章？

**评分**：
- 能回答 8+ 题：大纲基本健康
- 能回答 5-7 题：需要针对性补强
- 能回答 <5 题：需要回到 `大纲阶梯.md` 重新构建

## 使用提示

1. **大纲卡住时**：快速浏览本文件，找到对应问题
2. **修复时**：先看诊断，确认问题类型，再按修复方案执行
3. **修复后**：用同样的诊断标准再次检查
4. **每卷结束时**：执行 5 分钟大纲体检
5. **严禁**：跳过诊断直接修改——可能改错方向
