# Causality Map Template

```md
# Scene Causality Map

| Scene / Chapter | Goal | Obstacle | Turn | Consequence | Notes |
|-----------------|------|----------|------|-------------|-------|
| Ch 1 Scene 1 | | | | | |
```

使用规则：

- `Goal` 必须是角色想达成的事
- `Obstacle` 必须是真实阻力，不是空气
- `Turn` 必须说明局面在哪里变了
- `Consequence` 必须说明这个场景如何把下一场推出来
