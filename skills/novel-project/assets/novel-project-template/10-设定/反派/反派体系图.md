# 反派体系图

> **使用说明**：百万级长篇的反派往往不是单个人，而是「体系」——BOSS-副 BOSS-打手-卧底-被利用的棋子。
> 单凭 `10-设定/角色/反派.md` 装不下这种结构。
> 每个核心反派体系单独建一份本表（命名为 `反派体系-名称.md`），本文件作为索引。

## 反派体系名称：反派体系名

> 比如：围剿龙族势力、青云宗内鬼组织、皇室暗卫、北原邪修联盟等。

## 反派层级图

> 用树状图描述这个反派的内部层级。读者通过这张图能理解"为什么主角要打这么多次"——因为有层层剥皮。

```
最终 BOSS / 大反派头目
   │
   ├── 副 BOSS 一（名字）—— 一句话定位
   │      │
   │      ├── 打手 A（名字）—— 功能
   │      ├── 打手 B（名字）—— 功能
   │      └── 卧底（名字）—— 潜伏在 X 势力
   │
   ├── 副 BOSS 二（名字）—— 一句话定位
   │      │
   │      └── ...
   │
   ├── 暗子（名字）—— 潜伏在 X 势力
   │
   └── 棋子（名字）—— 被利用，本人不知情
```

### 填写示例（围剿龙族势力）

```
赵长老（幕后主谋，围剿龙族势力后代）
   │
   ├── 江鹤（卧底）—— 潜伏在散修联盟
   │      │
   │      └── 散修弟子若干（棋子，被江鹤利用，本人不知情）
   │
   ├── 李昭（明面打手）—— 青云宗内门大师兄
   │      │
   │      └── 师弟若干（工具人）
   │
   ├── 皇室某王爷（高阶合作者）—— 大周皇室
   │
   └── 神秘势力"暗火"（终极后台）—— 始终未现身
```

## 反派角色的"明知度"

> 反派体系中不同角色对"自己是反派"这件事的认知不同。这影响他们的人性复杂度。

| 角色 | 是否知道自己是反派 | 是否知道全貌 | 心理状态 | 读者观感 |
|------|----------------|------------|---------|---------|
| BOSS | 是 | 是 | 坚定 | 可恨 |
| 副 BOSS | 部分 | 不全 | 挣扎 | 复杂 |
| 卧底 | 是 | 不全 | 矛盾 | 值得同情 |
| 棋子 | 否 | 否 | 无辜 | 工具人 |
| 被利用的弱者 | 否 | 否 | 痛苦 | 让人心疼 |

> **填表说明**：
> - **是否知道自己是反派**：角色是否知道自己在和主角对立
> - **是否知道全貌**：角色是否知道整个反派体系的全貌（很多反派只看到自己这一层）
> - **心理状态**：他做这些事时的内心状态
> - **读者观感**：读者会怎么看待他

## 反派递进出场

> 长篇里反派不能一次性全部登场。必须有节奏地"剥洋葱"。

| 卷 | 揭露的反派层级 | 揭露方式 | 读者震惊度 |
|----|--------------|---------|----------|
| 第 1 卷 | 打手（李昭） | 主角直接冲突 | 低（预期内） |
| 第 2 卷 | 副 BOSS 的影子 | 通过事件间接露出 | 中 |
| 第 3 卷 | 卧底（江鹤） | 主角识破 | 中（要巧妙） |
| 第 4 卷 | 副 BOSS（赵长老） | 撕破脸 | 高（出乎意料） |
| 第 5 卷 | 最终 BOSS（暗火） | 真相大白 | 极高（全局翻盘） |

## 反派体系自检

- [ ] 反派体系是否有 3 层以上的层级（少于 3 层太扁平）
- [ ] 反派体系中是否有至少 1 个"被利用的棋子"（增加复杂性）
- [ ] 反派角色是否有"明知度"差异（全员都知道全貌，太扁平）
- [ ] 反派递进出场的节奏是否合理（一次全出 or 一次只出 1 个）
- [ ] 反派内部矛盾是否在主线中被利用过
- [ ] 反派体系的"绝招"是否被主角正面破解过
- [ ] 反派体系的"软肋"是否被主角利用过
