# 地点名称

> 使用说明：写一个新地点时，复制这个文件，命名为「地点名.md」，填入内容。
> 核心地点建议独立建文件，次要地点可在 `地点.md` 中追加章节。
> 与 `世界层级图.md` 配对使用：层级图描述"世界怎么嵌套"，本表描述"单个地点长什么样"。
