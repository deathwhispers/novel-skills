# 文风指南

记录本书的写作风格和语言习惯，防止长篇连载中风格漂移。

## 基本设定

- 叙述视角：第三人称有限 / 第三人称全知 / 第一人称 / 多视角交替
- 时态（中文中体现为）：现在时感 / 过去时感
- 叙述距离：贴近主角内心 / 保持客观距离 / 切换

## 语言风格

- 句式偏好：长短句结合 / 短句为主 / 长句铺陈
- 对话风格：直白 / 含蓄 / 幽默 / 冷峻
- 描写密度：精简（一笔带过）/ 中等（关键场面详细）/ 浓墨重彩
- 方言/口头禅使用：有无；具体是哪些
- 特殊用语库：本书自创/特有的词汇（如功法名称、等级体系等）

### 视角轮换规则

> 百万级长篇多视角的"切换节奏"必须明确，否则会乱。

| 规则 | 默认值 | 调整方向 |
|------|-------|---------|
| 主视角戏份比例 | 60% | 高潮卷可降到 45% |
| 副视角戏份比例 | 25% | 感情卷可升到 40% |
| 反派视角戏份比例 | 10% | 反派卷可升到 20% |
| 视角切换最小间隔 | 3 章 | 不建议短于 3 章 |
| 视角切换最大连续 | 8 章 | 不建议连续 8 章同一视角 |
| 同一章切视角 | 不允许 | 一章只能有一个主视角 |
| 反派视角的使用 | 限定章节 | 不要全程穿插，会破坏悬念 |

### 视角切换的"事件接力"原则

> 每次切换视角，必须满足三件套：

1. **事件接力**：A 视角章节尾留问题，B 视角章节头接问题
2. **信息差**：B 视角不知道 A 视角已经知道的关键信息（形成悬念）
3. **主动行为**：B 视角人物在这一章至少做 1 个主动决定（不是"看"）

### 多视角下"信息盲区"的处理

> 配 `20-大纲/视角信息盲区表.md` 使用——确保每个视角人物"不知道什么"是被设计过的，不是作者偷懒。

### 视角人物失效的常见死法

- ❌ 视角人物连续 5 章没有任何主动行为（变成"摄像机"）
- ❌ 视角切换没有事件接力（A 章节尾和 B 章节头毫无关系）
- ❌ 反派视角频繁使用（破坏主角视角的代入感）
- ❌ 切换过于频繁（每 1-2 章就切，读者跟不上）
- ❌ 上帝视角（所有视角人物"知道得太多"）
- ❌ 切换后叙述距离骤变（A 视角贴近内心，B 视角全知客观）
