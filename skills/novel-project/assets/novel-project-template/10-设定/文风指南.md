# 文风指南

记录本书的写作风格和语言习惯，防止长篇连载中风格漂移。

## 基本设定

- 叙述视角：第三人称有限 / 第三人称全知 / 第一人称 / 多视角交替
- 时态（中文中体现为）：现在时感 / 过去时感
- 叙述距离：贴近主角内心 / 保持客观距离 / 切换

## 语言风格

- 句式偏好：长短句结合 / 短句为主 / 长句铺陈
- 对话风格：直白 / 含蓄 / 幽默 / 冷峻
- 描写密度：精简（一笔带过）/ 中等（关键场面详细）/ 浓墨重彩
- 方言/口头禅使用：有无；具体是哪些
- 特殊用语库：本书自创/特有的词汇（如功法名称、等级体系等）

### 视角切换策略

不预设比例、间隔或“一章一视角”。为每次切换说明它服务什么：补充信息、制造距离、改变情绪理解、形成对照或让读者重新解释前文。

| 视角切换 | 切换目的 | 读者多知道/少知道什么 | 叙述距离如何变化 | 边界标记 |
|---|---|---|---|---|
| A → B | | | | |

### 多视角下"信息盲区"的处理

> 配 `20-大纲/视角信息盲区表.md` 使用——确保每个视角人物"不知道什么"是被设计过的，不是作者偷懒。

### 视角风险

- 无意让人物知道他不可能知道的信息
- 切换后读者无法判断谁在感知或判断
- 为解释设定而临时进入方便的视角
- 叙述距离变化没有内容上的作用
- 视角只是摄像机，没有选择性注意、误解或价值判断
