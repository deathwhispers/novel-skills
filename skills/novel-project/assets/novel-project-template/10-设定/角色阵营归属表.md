# 角色阵营归属表

> **使用说明**：本表回答"每个角色属于哪个势力、立场强度、变节可能性"——这是势力博弈和角色立场的总表。
> 与 `10-设定/势力关系矩阵.md` 配对使用：关系矩阵是势力对势力，本表是势力对角色。
> 长篇网文里角色往往不会永远归属一个势力——会出现"身在曹营心在汉""明面归属甲实际听命于乙"的情况，本表要追踪这些复杂状态。

## 角色立场强度（-5 到 5）

- **5**：死忠/血誓，宁死不叛
- **4**：核心成员，决策参与
- **3**：常规成员，服从组织
- **2**：外围/客卿，关系松散
- **1**：合作方，有利益绑定
- **0**：中立/无势力
- **-1**：被动敌对（如被追杀）
- **-2**：公开敌对
- **-3**：死敌
- **-4**：被组织追杀
- **-5**：血仇

## 角色的"利益-情感"双轴

> 一个人对势力的态度，既有"利益"（利益一致或冲突），也有"情感"（喜欢或厌恶）。这二者经常错位。
> 长篇网文里最戏剧化的场景是：利益上应该站在某势力一边，情感上却和另一边更深。

| 角色 | 利益上倾向 | 情感上倾向 | 错位程度 | 这段错位驱动了什么戏 |
|------|----------|----------|---------|------------------|
| | | | | |
| | | | | |
| | | | | |

> **示例**：
>
> | 角色 | 利益上倾向 | 情感上倾向 | 错位程度 | 这段错位驱动了什么戏 |
> |------|----------|----------|---------|------------------|
> | 林尘 | 龙族传承（利益） | 青云宗同门（情感） | 高 | 第 3 卷必须在二者间选择 |

## 角色阵营自检

- [ ] 是否有角色立场长期不变（> 50 章没有立场变化）——提示角色在"工具化"
- [ ] 主角所在势力是否覆盖至少 3 种不同类型的成员（不要全是师徒或全是同龄弟子）
- [ ] 是否有角色立场与公开身份不一致（暗线足够多，故事才有内味）
- [ ] 是否有高变节概率角色已经被"无预兆"地使用过——提示回收草率
- [ ] 立场漂移轨迹是否和角色的"弧线"对齐（变节不应该和角色逻辑冲突）
