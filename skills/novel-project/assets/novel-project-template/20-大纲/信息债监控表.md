# 信息债监控表

> **核心问题**：这条信息债是什么时候埋的？已经多少章没碰了？会不会读者已经忘了？
> 信息债（伏笔/悬念/未揭示的信息）是追读的核心动力，但过期不碰的信息债会变成毒点。
> **规则**：每 10 章扫描一次本表，标出即将「过期」的信息债。
> 百万级长篇新增**债的等级（S/A/B/C）**和**紧迫性矩阵**——百万级长篇信息债数量多，统一按"过期预警"4 档管理粒度太粗。

## 债的等级（S/A/B/C）

> 不同等级的信息债保鲜期、触碰频率、回收压力完全不同。

| 等级 | 定义 | 保鲜期 | 触碰频率 | 失败代价 |
|------|------|-------|---------|---------|
| **S 贯穿全书** | 全书主谜题，主角真实身份/世界真相/终局对手 | 50 章 | 至少每 30 章显性触碰 | 读者核心期待崩塌 |
| **A 跨卷** | 卷末钩子对应的谜题，关键人物命运 | 25 章 | 至少每 15 章显性触碰 | 整卷立意崩 |
| **B 卷内** | 单卷内的伏笔，关键转折/揭露 | 12 章 | 至少每 8 章显性触碰 | 单卷节奏塌 |
| **C 章内** | 单章钩子，未解决的动作/场景 | 3 章 | 章尾处理 | 读者弃章 |

## 信息债总览

| 线头 | 类型 | 等级 | 埋下章节 | 上次触碰 | 距上次触碰 | 计划回收章节 | 紧迫性 | 过期预警 | 翻面代价 |
|------|------|------|---------|---------|-----------|------------|-------|---------|---------|
| | 线索/关系债/物件债/承诺债/情绪债/身份谜题 | S/A/B/C | | | (章) | | 高/中/低 | 安全/注意/危险/过期 | |
| | | | | | | | | | |
| | | | | | | | | | |

> **填表说明**：
> - **紧迫性**：高=3 章内必须碰 / 中=10 章内必须碰 / 低=该回收时再碰
> - **翻面代价**：回收这条债时，会对哪些已设的事实造成冲击
>   （如：身份谜题回收可能推翻主角的入门合法性，必须提前设计好退路）

## 紧迫性矩阵

> 把"债的等级"和"距上次触碰的章数"画成矩阵，自动得出下一步动作。

| 等级 \ 距上次触碰 | 0-10 章 | 11-20 章 | 21-30 章 | 31-50 章 | 50 章+ |
|---------------|--------|---------|---------|---------|--------|
| **S 贯穿全书** | 维持 | 隐性强化 | 显性提醒 | 显性提醒 | 立即触碰 |
| **A 跨卷** | 维持 | 隐性强化 | 显性提醒 | 立即触碰 | 立即触碰 |
| **B 卷内** | 维持 | 显性提醒 | 立即触碰 | 立即触碰 | 立即触碰 |
| **C 章内** | 维持 | 立即触碰 | — | — | — |

> **矩阵用法**：在表中找到当前债的等级和距上次触碰的交叉格，按格子里的指令执行。
> 这比记"过期预警"4 档更精确。

> **过期预警说明**：
> - **安全**：10 章内碰过
> - **注意**：11-20 章没碰了，需要尽快刷存在感
> - **危险**：21-30 章没碰了，很大一部分读者已经遗忘
> - **过期**：30 章以上没碰，建议重新规划这条线的回收方式

## 不同类型信息债的合理保鲜期

| 类型 | 定义 | 最长不碰间隔 | 读者遗忘风险 |
|------|------|-------------|-------------|
| 线索 | 指向某个真相的碎片信息 | 20 章 | 中 |
| 关系债 | 一段关系中的未兑现承诺或未爆发的冲突 | 15 章 | 高 |
| 物件债 | 某个关键物品的来历/用途/后果 | 25 章 | 中 |
| 承诺债 | 角色之间的约定/威胁/誓言 | 20 章 | 中 |
| 情绪债 | 未消化的情绪冲突或误解 | 12 章 | 高 |
| 身份谜题 | 角色真实身份/来历的悬疑 | 15 章 | 极高 |

## 翻面代价追踪

> 回收一条 S 级或 A 级债时，往往会推翻已经写进正文的事实。本表追踪"翻面时需要同步处理的事"。

| 债 | 翻面时会推翻什么 | 需要同步补的设定 | 风险等级 |
|----|--------------|--------------|---------|
| 主角是龙族太子 | 入门合法性 / 青云宗对主角的处置 | 主角入门时的合法性退路 / 龙族覆灭前已布的暗子 | 高 |
| 师父是反派 | 师父教导主角的所有经验 | 师父教导中"误导"与"真实"的比例 | 极高 |
| 大周皇室是幕后黑手 | 整本书的"国家"概念 | 皇室渗透到什么程度 / 哪些是真心 | 极高 |

## 需要立即处理的信息债

> 标记为「危险」或「过期」的债，优先在 3 章内安排一次触碰（不一定要回收，提醒即可）。

| 线头 | 危险等级 | 上次触碰 | 最近一次提醒的方式 | 计划行动 |
|------|---------|---------|------------------|---------|
| | | | 角色提起/间接影响/类似事件暗示 | |
| | | | | |

## 信息债触碰方式

> 不一定非要在该回收的时候才碰。可以用以下方式「刷存在感」：
> 详细策略库见 `20-大纲/回收/伏笔强化策略库.md`。

| 方式 | 效果 | 适用场景 |
|------|------|---------|
| 角色不经意提起 | 温和提醒，不增加紧迫感 | 中期线头 |
| 间接影响当前事件 | 让读者感觉到「这条线还在动」 | 长线伏笔 |
| 类似事件暗示 | 提醒读者「之前也有过类似的事」 | 对称结构 |
| 代价浮出水面 | 让读者想起「这条线如果爆了会很严重」 | 即将回收的线 |
| 误判或假回收 | 给读者一个假答案，再推翻 | 悬疑线 |
| 变形 | 债本来指向 A，最后指向 B | 中期升级 |
| 升级 | 债本来很小，最后变成主线 | 翻转结构 |
| 角色内化 | 债不再被讨论，但角色的行为持续受影响 | 长线伏笔的"消音" |

## 信息债回收节奏建议

| 全书进度 | 应该收多少 | 建议 |
|---------|-----------|------|
| 0-25% | 埋线期 | 只埋不收，但每条已埋线至少碰 1-2 次 |
| 25-50% | 小回收期 | 回收 20-30% 的短线，长线开始升级/变形 |
| 50-75% | 中回收期 | 回收 50% 左右的线，最长的那几条线开始加速 |
| 75-100% | 大回收期 | 回收 90%+ 的线，留少数做续集/余韵 |

## 信息债自检

- [ ] S 级债的保鲜期是否被严格管控（避免 50 章不碰）
- [ ] 翻面代价是否被提前规划（避免回收时硬伤）
- [ ] 不同等级的债是否被差异化处理（不要所有债都按"过期预警"4 档一刀切）
- [ ] 是否有"该升级但没升级"的债（如：本是 B 级但实际承担了 A 级的作用）
- [ ] 紧迫性矩阵中的"立即触碰"格是否都被处理了
