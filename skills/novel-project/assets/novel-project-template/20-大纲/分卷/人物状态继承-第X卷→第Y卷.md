# 人物状态继承：第 {{卷号}} 卷 → 第 {{下一卷号}} 卷

> **使用说明**：本表专门追踪两个卷之间"人物状态"的继承——避免卷末和卷首人物状态不一致。
> 配合 `20-大纲/分卷/卷间衔接检查.md` 的 checklist 使用，但比 checklist 更结构化。
> 每完成一卷，新建一份本表（命名为 `人物状态继承-第X卷→第Y卷.md`）。

## 继承自检

- [ ] 主角所有维度（位置/修为/身体/情绪/认知/目标）是否都和卷末一致
- [ ] 重要配角状态是否都和卷末一致
- [ ] 反派状态是否都和卷末一致

- [ ] 卷末伏笔清单是否全部继承给下卷
- [ ] 上一卷的追读动力是否有下卷的兑现计划
- [ ] 是否有"作者忘了"导致的不一致（这种 bug 一定要在这一步发现）
