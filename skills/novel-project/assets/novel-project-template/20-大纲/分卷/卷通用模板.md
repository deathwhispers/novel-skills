 # 第{卷号}卷卷纲

 > **使用说明**：本模板是卷纲的完整字段集。不需要一次性填完。
 > 先填 **核心字段**（角色、开卷、卷末、转折），再填扩展字段。
 > 填完核心字段后就可以拆节拍卡，扩展字段可以在写节拍卡的过程中同步细化。
 > 第一卷的节奏型参考 `20-大纲/首卷发射台.md`。

 ## 核心字段

 ### 这一卷在全书中扮演什么角色

 写本卷在全书主线中的功能：是起势段、发展段、转折段、深化段还是收束段。

 ### 这一卷主要卖什么

 写本卷最核心的读点承诺——读者为什么愿意追完这一卷。

 ### 开卷局面

 写本卷开始时主角的处境、外部压力和内部状态。

 ### 中段如何升级

 写中段如何抬高代价、引入新变量或加深关系强度。

 ### 卷末要收住什么、放大什么

 写卷末必须兑现的局部满足，以及为下一卷放大的长期承诺。

 ### 这一卷必须出现的几个关键转折

 写 3-5 个非出现不可的关键翻面点。

 - 转折一：（第 X 章附近）
 - 转折二：
 - 转折三：
 - 转折四：
 - 转折五：

 ### 主线推进量

 写本卷结束时，主线从 A 推进到 B 的具体变化。

 ### 感情线/关系线推进量

 写本卷结束时，关键关系从 X 变化到 Y 的具体变化。

 ### 悬念线推进量

 写本卷埋了什么新线、升级了什么旧线、收回了什么线。

 ### 卷末钩子

 写这一卷结尾留下的最直接影响下一卷读欲的东西。

 ### 本卷写到一半时最容易塌的地方

 写本卷写作中最容易灌水、偏移或失速的位置。

### 本卷角色戏份分配

> 百万级长篇每卷必须保证所有核心角色都有戏份，否则读者会觉得"这人去哪了"。
> 写卷纲时按角色逐个规划"本章出场多少次、推动什么关系、有什么状态变化"。

| 角色 | 戏份类型 | 出场章节 | 关系变化 | 状态变化 | 风险 |
|------|---------|---------|---------|---------|------|
| | 主线/感情/悬念/反派/工具 | | 推动/不变化 | 强/弱/无 | 戏份过少或过多 |
| | | | | | |
| | | | | | |
| | | | | | |

> **填表说明**：
> - **戏份类型**：本卷这个角色主要承担什么（主线推进/感情推进/悬念推进/反派推进/工具角色）
> - **关系变化**：本卷和谁的关系发生推进（写"林尘↔苏晚：暧昧→明确"）
> - **状态变化**：身体/认知/处境是否有变化（写"林尘：受伤→恢复"）
> - **风险**：标"戏份过少"或"戏份过多"或"功能不清"
