# 卷间衔接检查

写完一卷、准备开下一卷之前，先走完这套检查。不要跳过。

## 第一卷收束检查

- [ ] 本卷的核心冲突是否已经解决或阶段性收束
- [ ] 本卷主角是否发生了可感知的变化（能力/认知/关系至少有一项）
- [ ] 本卷埋下的伏笔是否已标记哪些在本卷回收、哪些延到后续卷
- [ ] 本卷末尾是否有足够的"余韵时刻"，而不是仓促结尾
- [ ] 本卷留下的续卷钩子是否清晰且有力
- [ ] 本卷提到的地点、人物、势力是否在设定文档里已建立档案

## 下卷启动准备

- [ ] 下卷的核心冲突是什么？主角要面对的新问题是什么？
- [ ] 下卷相比本卷，最大的"翻面"（局面变化/关系变化/认知变化）是什么？
- [ ] 下卷需要引入哪些新角色/新势力/新地点
- [ ] 下卷需要保留哪些旧角色/旧线继续推进
- [ ] 下卷的时间线起点：紧接本卷末尾 / 跳跃一段时间
- [ ] 下卷的基调变化：更轻松 / 更沉重 / 更紧张 / 更开阔
