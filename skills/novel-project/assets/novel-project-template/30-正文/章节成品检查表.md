# 章节成品检查表

写完后、标记"完成"前，逐项检查。不要求每项都做到满分，但要有意识记录弱项。

## 基本层

- [ ] 本章篇幅是否符合项目媒介约束，或偏离理由是否成立
- [ ] 本章的节拍卡是否已全部实现
- [ ] 本章核心目标（一个明确的戏剧任务）是否达成
- [ ] 本章是否有真正的推进（信息推进 / 关系推进 / 局面推进 / 角色推进至少一项）

## 对抗层

- [ ] 本章是否有明确阻力 / 冲突 / 对抗
- [ ] 阻力是否具体（不是"命运"或抽象概念，而是具体的人和事）
- [ ] 阻力是否足够强（让读者不确定主角能否跨过去）
- [ ] 主角面对阻力是否展现了主动性（不是被动承受）

## 细节层

- [ ] 本章是否有至少 2-3 个具体的细节抓手（动作/对话/环境中的一个具体意象）
- [ ] 这些细节是否服务于情绪或角色塑造，而非单纯炫技
- [ ] 对话是否带有潜台词（角色间真的有信息差或情绪差）

## 因果层

- [ ] 本章所有重要事件是否有前文铺垫（不凭空发生）
- [ ] 本章是否埋了新的因果线（为后续章节做铺垫）
- [ ] 因果关系是否自然（不是作者硬推）

## 章尾发动机

- [ ] 章尾是否有明确的追读推动力（悬念 / 期待 / 情绪余韵至少一项）
- [ ] 章尾推动力是否来自本章内容的自然延伸（不是为了悬念强扭的）
- [ ] 读者读完本章后最可能产生的情绪是什么（应该可预测）

## 连续性层

- [ ] 人物状态是否与上一章接续（受伤/情绪/位置/关系）
- [ ] 设定信息是否与 `10-设定/` 中的硬设定一致
- [ ] 时间线是否与 `10-设定/时间线.md` 一致
- [ ] 本章涉及的所有旧线索是否都查过前文状态
