# 篇幅规划指南

篇幅是生产、发布或投稿约束，不是文学质量标准。先确定写作模式，再决定需要多少规划。

## 按模式规划

- 商业连载：确认平台、更新频率、单章参考区间、存稿和阶段兑现。
- 类型长篇：确认总量级、卷/部结构和关键结构节点，章节长度允许按场景变化。
- 文学叙事：优先确认整体体量、投稿/出版要求和叙事密度，不预设日更或单章钩子。
- 短篇：确认投稿字数上限、结构比例和删减余量。
- 探索起草：只设本次试写上限或时间盒，不预测全书字数。

## 复核

- 当前数字来自真实平台/投稿要求，还是习惯假设？
- 篇幅变化是否来自内容需要，而非补水或机械压缩？
- 计划是否给试写、修订和休息留出余量？
- 如果模式改变，哪些数字应立即失效？

明显偏离预算时，先检查写作单位和场景边界。只有平台硬限制或结构失效才强制拆分、压缩或扩写。
