# 常见 AI 腔对照表

## 句式类

| AI 腔 | 自然替代 |
|-------|---------|
| 由于……所以…… | 直接写原因或结果，不用连词 |
| 他/她感到/觉得/认为 | 删掉，换成动作或对话 |
| 随着…… | 拆成两句，不要用「随着」开头 |
| 不禁/不由地/忍不住 | 换成具体动作：「他笑了一声」vs「他忍不住笑了」 |
| 说明/意味着/代表 | 留给读者判断，不要说穿 |
| 在……的带领下/帮助下 | 换成具体互动 |
| 只见/就看到 | 删掉，直接写画面 |

## 结构类

| AI 腔 | 自然替代 |
|-------|---------|
| 每个段落 3-5 句均衡 | 长短段交替，关键段落可只有一句 |
| 先景后人 | 先人后景，或人和景同时出现 |
| 对话后紧跟解释 | 删掉解释，信任读者能看懂 |
| 叙事后紧跟心理 | 把心理化成行动或对话潜台词 |
| 每章开头重述前文 | 直接推进，不要回顾 |

## 词汇类

| AI 腔 | 自然替代 |
|-------|---------|
| 点了点头 | 点了下头 / 没说话 |
| 与此同时 | 删掉，或者用「这时」 |
| 然而/但是 | 一半的情况可以删掉，用短句转折 |
| 众所周知 | 不要用，直接写世界普遍认知 |
| 毫无疑问 | 删掉，用事实说话 |
| 某种/某种程度 | 换成具体描述或直接删 |

## AI 味检测流程

**第一步：扫描标记**（2 分钟）
1. 搜索「感到/觉得/认为/意识到」→ 标记所有位置
2. 搜索「由于/随着/与此同时」→ 标记所有位置
3. 搜索「不禁/不由/忍不住」→ 标记所有位置
4. 搜索「可能/大概/似乎/某种」→ 标记所有位置

**第二步：分类处理**（3 分钟）
| 标记类型 | 处理方式 | 优先级 |
|---------|---------|--------|
| 解释腔 | 删掉，检查上下文是否可读 | P0 |
| 概括情绪 | 换成动作/对话/环境 | P0 |
| 过渡词 | 删掉或换成短句 | P1 |
| 安全表达 | 改成精确表达 | P1 |
| 均匀句式 | 改写长短交替 | P2 |

**第三步：通读验证**（2 分钟）
- 读改后的段落，感觉是否"像人在说话"
- 检查是否有"删了后反而不通"的地方→回退

## 常见 AI 味组合

| 组合 | 表现 | 修复 |
|------|------|------|
| **解释+概括** | "他感到愤怒，因为对方背叛了他" | 删掉"感到"和"因为"，写动作 |
| **过渡+过渡** | "随着天色渐暗，与此同时……" | 拆成两句，删掉过渡词 |
| **安全+安全** | "某种程度上的某种变化" | 改成精确描述 |
| **均匀+均匀** | 连续 5 句都是"他做了什么，然后做了什么" | 长短交替，加中断 |
| **先景+解释** | "夕阳很美。这说明一天结束了。" | 删解释，让夕阳自己说话 |

## 快速检查（1 分钟）

随机抽 3 段：
1. 有「感到/觉得」吗？→ 有则改
2. 有连续 3 句同样开头吗？→ 有则改
3. 有「由于/随着」吗？→ 有则考虑删

**任一题"有"→进入 AI 味检测流程**
