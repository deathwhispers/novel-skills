# 前后照应检查表

重点检查这几类“中文长篇最常丢”的东西：

1. 前章刚立的目标，这章是否还在推动
2. 前章刚受的伤、刚知道的信息、刚破的关系，这章是否还在生效
3. 提过一次的重要物件、口头承诺、秘密、误会，这章是否被碰到
4. 章尾扔出的悬念，这章开头是否接住
5. 前文明确写过的规则、称呼、地理、时间线，这章有没有打架

如果丢了，不要只记“漏了”，要记清：

- 丢的是哪一条
- 丢在什么位置
- 补在哪里最顺

如果是长篇中后段，再额外问：

- 这条线如果继续不碰，会不会伤追读
- 它是该正式回收，还是至少要提醒一次还活着

---

## 常见前后照应错误

| 错误 | 表现 | 修复 |
|------|------|------|
| **目标遗忘** | 主角立下"必须救父亲"的目标，10 章后只字不提 | 每 3-5 章至少提一次进度 |
| **伤势恢复** | 角色重伤后没有恢复过程突然痊愈 | 加恢复描写或时间过渡 |
| **承诺未兑现** | "我一定会回来"然后 20 章没回来也没交代 | 加"未能兑现"的场景 |
| **误会不解** | 两个角色误会了 50 章还在误会 | 设计误会解除的触发点 |
| **物件失踪** | 重要的信物提过一次后再也没出现 | 设计至少 1 次"被看到"的场景 |
| **规则打破** | 前文说"此阵法只能用三次"，后文用了四次 | 解释第四次或改前文 |
| **称呼打架** | 同一角色在同一场景被不同称呼 | 统一或设计"称呼变化"的剧情 |

---

## 快速检查（1 分钟）

对照前章最后 3 个关键信息：
1. 目标/伤势/关系 → 这章还在生效吗？
2. 章尾悬念 → 这章开头接住了吗？
3. 前章承诺 → 这章有推进吗？

**任一题"没有"→记录到"丢失清单"并标记补回位置**
