## 五点五、章间推进密度

> 百万级长篇最容易在卷中段出现"章间空转"——多章虽然都有事件但读者感觉"没在推进"。本节专门排查这个问题。

### 章间推进密度矩阵

> 行=章节，列=四个推进维度，每格打分（0=无，1=弱，2=中，3=强）。
> 满格 12 分，< 6 分提示"空转"。

| 章节 | 信息推进 | 关系推进 | 局面推进 | 角色成长 | 累计 |
|------|---------|---------|---------|---------|------|
| Ch X | | | | | |
| Ch X+1 | | | | | |
| Ch X+2 | | | | | |
| ... | | | | | |

> **填表说明**：
> - **信息推进**：本章是否揭示了新的关键信息
> - **关系推进**：本章是否有关系变化（不是"角色见面"，是"关系有进展或破裂"）
> - **局面推进**：本章是否改变了主角的处境（位置/状态/能力）
> - **角色成长**：本章是否让主角/重要角色发生了认知或行为变化

### 推进密度曲线

### 章间空转的常见形态

| 形态 | 表现 | 风险 |
|------|------|------|
| 场景堆叠 | 连续 3 章写同一场景的不同角度 | 高（读者会跳过） |
| 信息无承接 | 章尾抛新信息，下章不接 | 中（读者会觉得"作者忘了"） |
| 关系无变化 | 连续 5 章关系状态不变 | 中（读者会弃） |
| 角色无成长 | 主角连续 10 章无认知/能力变化 | 高（核心推进力丧失） |
| 事件堆积 | 章章都在"发生事"，但都不重要 | 高（看起来在写其实在空转） |

### 章间空转自检

- [ ] 是否有连续 3 章累计 < 6 分（提示空转）
- [ ] 是否有连续 5 章"角色成长"维度为 0（提示主角"工具化"）
- [ ] 是否有连续 5 章"关系推进"维度为 0（提示感情线停滞）
- [ ] 是否有连续 5 章"信息推进"维度为 0（提示悬念枯竭）
- [ ] 累计均分最低的 3 章是否能合并或删除
- [ ] 中段（30-70%）的密度曲线是否塌陷
