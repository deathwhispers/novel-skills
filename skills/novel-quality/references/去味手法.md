# 去味动作库

按问题类型选择操作，不要一次性全上。

## 去解释腔（最优先）

1. 通读一章，标记所有「因为/所以/由于/这意味着/这说明」
2. 删掉标记句，检查上下文是否仍然可读
3. 如果不可读：把解释内容拆进角色对话或动作
4. 如果仍然不可读：那这条信息需要换位置，而不是在这里解释

## 去均匀感

1. 找连续 3 句以上同样句式开头——改写其中至少 2 句
2. 找 3 个以上连续中等长度段落——把其中一个拆成短句段
3. 找平滑过渡——在场景中间加一个中断（打断、声响、意外发现）
4. 对话不要一来一回太多次——3 轮对话后加动作或感知

## 去概括情绪

1. 搜索「感到」「觉得」「心里想」「意识到」
2. 每个标记处问：能不能用动作/对话/环境变化替代
3. 如果不能替代：保留但不要叠加（不要「感到悲伤又愤怒又无助」）

## 去角色同声腔

1. 挑一章，去掉所有对话标签（他说/她说）
2. 读一遍——能分清谁在说话的，角色声线合格
3. 分不清的，给每个角色加一两个口语标记：
   - 急性子：说话短，爱打断
   - 慢性子：说话带语气词，爱反问
   - 地位高：命令句多
   - 地位低：犹豫词多
   - 年轻人：用流行语
   - 老年人：用老派比喻

## 去安全表达

1. 搜索「可能/大概/似乎/好像/某些/有点」
2. 每个标记处：删掉，或者改成精确表达
3. 例外：角色性格谨慎时的台词可以保留
