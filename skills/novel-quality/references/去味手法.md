# 语言风险修订

“AI味”不是词语集合，而是语言选择与人物、场景和叙事目的脱节。先说明受损效果，再决定是否修改。

## 1. 重复解释

检查动作、对话、心理和作者说明是否连续表达同一信息。删掉最弱的一层，再读上下文；如果解释承担必要的因果、背景或节奏作用，就保留。

“因为、所以、意味着、感到、觉得、似乎”本身不构成问题。只有它们让信息重复、判断空泛或人物声音失真时才修改。

## 2. 节奏失去功能

不要为了长短交替而拆句，也不要为了“自然”强行插入声响和动作。检查句子、段落和停顿是否与注意力、压力和情绪变化同步。

- 紧张场景可以长句压迫，也可以短句切断。
- 平静场景可以保持均匀，让微小偏差产生力量。
- 连续对话可以不插动作，只要关系、语气和信息在变化。

## 3. 情绪表达失真

情绪命名、身体反应、动作、环境和沉默都是工具。判断人物在这一刻会如何感知和表达，而不是一律“show, don't tell”。避免用摔杯、攥拳、咬唇等通用动作替换所有情绪。

## 4. 角色同声

区分角色时优先看：

- 他先注意什么、忽略什么
- 他会直接回答、转移、反问还是沉默
- 他如何判断风险、身份和关系
- 压力下句法和信息保留如何变化

不要靠给每个角色增加固定口癖、语气词或刻板年龄标签制造差异。

## 5. 安全与含混

“可能、大概、似乎、好像”可以表达不确定知觉、谨慎人格或叙述距离。只有无意削弱判断时才改为精确表达。

## 修订闭环

1. 引用问题原文。
2. 说明它损害的具体效果。
3. 写明必须保留的人物、信息和节奏。
4. 做最小修改。
5. 对比修改前后；若只是更符合规则但更不像人物，恢复原文。
