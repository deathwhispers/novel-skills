# 叙事质量审计

章节体检判断“是否成立”，本审计判断“选择是否有效”。根据写作模式选择 2-3 个维度，不要求每章集齐全部效果。

## 1. 模式与目的

- 这一段在当前作品中要产生什么阅读效果？
- 它的节奏、场景形态和结尾是否服务当前写作模式？
- 如果删掉或换成通用写法，作品会失去什么？

不把商业连载的强钩子用于所有小说，也不把文学化留白强加给需要清晰兑现的类型场景。

## 2. 人物真实

- 人物先注意什么、回避什么、误解什么？
- 重要反应是否来自既有欲望、经验、压力和关系？
- 人物语言与选择是否包含他自己的限制，而非作者最方便的答案？

人物可以行动、等待、观察、拒绝或改变理解。判断重点是选择是否属于这个人。

## 3. 压力与情绪运动

- 场景开始和结束时，人物承受的压力或情绪理解发生了什么变化？
- 变化是逐步累积、突然断裂，还是有意保持停滞？
- 情绪是否通过最适合人物的方式表达，而非套用通用动作？

有效变化可以是外部局面、关系距离、自我认识或读者理解，不强制不可逆事件。

## 4. 视角与信息控制

- 读者看到的细节是否经过当前视角选择？
- 什么被说出、延迟、误读或省略？为什么？
- 解释与留白是否都出现在需要的位置？

信息清楚和信息含混都可以有效，关键是是否形成预期的理解、悬念、亲密或距离。

## 5. 语言、形式与回响

- 用词、句法、段落、对话和意象是否准确承担效果？
- 哪一处语言最不可替换？哪一处最像通用占位符？
- 局部形式是否与整章、前文或主题产生回响？

不要求每章使用新技法。重复可以建立风格、仪式和主题；变化也必须有内容上的理由。

## 审计方法

每个选定维度输出：

1. 原文证据
2. 已产生的阅读效果
3. 仍受损或不足的效果
4. 必须保留的内容
5. 最小修订建议

无法引用原文时，标记“证据不足”，不做确定结论。

## 通过条件

- 硬门禁全部通过。
- 当前写作模式的核心目的成立。
- 选定软目标有原文证据；不足项已做最小修订或记录为训练目标。

未选维度不构成失败，软目标不足不自动触发整章重写。
