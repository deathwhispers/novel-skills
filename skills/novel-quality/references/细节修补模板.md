## 4. 改写动作

- 先判断当前缺的是感知、关系、空间、因果还是必要解释
- 情绪命名与可观察后果按人物和节奏选择，不机械替换
- 让角色说话和动作继续保有区分度
- 删掉只起装饰作用、不能承载信息的细节

## 常见细节错误

| 错误 | 表现 | 修复 |
|------|------|------|
| **无效总结** | 情绪标签重复了上下文已经表达的信息 | 删除重复层，或保留最准确的一层 |
| **空间失焦** | 读者无法理解人物位置或互动 | 补当前视角真正会注意的空间证据 |
| **情绪悬空** | "她哭了"但没有前因后果 | 加触发事件和情绪变化过程 |
| **对话失去关系变化** | 长对话只是交换事实 | 调整回避、争夺、误解或信息保留；不强制插动作 |
| **细节装饰** | 大量描写与剧情无关 | 删掉或换成承载信息的细节 |
| **视角漂移** | 突然知道其他角色的心理 | 限制在视角人物的感知范围内 |
| **时代错位** | 古代角色说现代话 | 检查词汇和表达方式 |
| **身份错位** | 皇帝说话像市井百姓 | 加身份对应的语气和用词 |

## 快速检查（1 分钟）

随机抽 1 段对话：
1. 读者能否理解人物此刻的注意焦点和空间关系？
2. 细节是否承担人物、信息、情绪或节奏作用？
3. 对话的直接或含蓄程度是否符合人物关系？

任一题“没有”时先判断是否真的损害当前写作模式，再决定是否补强。
