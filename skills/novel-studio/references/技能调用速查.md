## 组合工作流

### 写一章的完整链路

1. `$novel-writing` 按节拍卡写正文
2. `$novel-quality` 做章节体检
3. `$novel-quality` 按体检结果修订
4. 写章节状态增量；每 5 章再合并总账

### 卷末整理

1. `$novel-quality` 做卷体检 + 长线扫描
2. `$novel-quality` 修复发现的问题
3. 执行卷间衔接检查（`20-大纲/分卷/卷间衔接检查.md`）
4. 归档已完卷到 `50-归档/`
5. 更新 `90-运行/决策记录.md`

### 更新日常

1. 写本章状态增量
2. 新 hard canon 立即同步
3. 每 5 章合并伏笔、因果和人物状态
4. 更新唯一的阶段性写作训练目标
