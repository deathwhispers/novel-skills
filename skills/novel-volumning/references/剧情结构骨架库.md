# 剧情结构骨架库

> **使用说明**：网文有 10+ 种成熟结构型。每种结构型有它的"必备元素"和"高潮分布"。
> 写百万级长篇前必须先选对结构型——结构型错了一切都错。
> 本表是 8 种主流结构型的"模板"——每种都包含结构骨架、必备元素、节奏型、适配题材。

## 一、8 种主流剧情结构型

> 每种结构型都有"主驱动""核心爽点""典型问题"三大要素。

| 结构型 | 主驱动 | 核心爽点 | 典型问题 |
|--------|------|--------|---------|
| **升级流** | 主角不断突破更高境界 | 升级带来的实力碾压 | 后期容易疲软 |
| **无敌流** | 主角开局就是最强 | 装逼打脸/降维打击 | 缺少成长感 |
| **复仇流** | 主角为某事/某人复仇 | 一路碾压仇人 | 后期动力不足 |
| **穿越/重生流** | 主角利用前世记忆改变命运 | 知道未来的优势 | 容易写崩 |
| **系统流** | 系统发布任务/奖励 | 完成任务的即时反馈 | 容易同质化 |
| **无限流** | 主角进入一个个副本 | 闯关/解谜 | 后期副本难设计 |
| **种田/经营流** | 主角从小处发展势力 | 势力从无到有 | 节奏慢 |
| **权谋/朝堂流** | 主角在政治斗争中翻盘 | 智斗/反杀 | 门槛高 |

## 二、升级流

### 结构骨架

```
第一阶段：起点（凡/入门）
   │
   ↓ 经历生死/奇遇/传承
   │
第二阶段：初入江湖
   │
   ↓ 中期最大考验
   │
第三阶段：小有名气
   │
   ↓ 卷末小高潮
   │
第四阶段：站在舞台中央
   │
   ↓ 全书最大考验
   │
终局：巅峰对决
```

### 必备元素

- 明确的等级体系（每个等级清晰可量化）
- 等级突破的"硬门槛"（资源/感悟/生死）
- 等级对应的"新地图"（去更大的地方）
- 等级对应的"新敌人"（更高一级的反派）

### 高潮分布

- 每卷 1-2 个升级
- 升级必须伴随"新地图+新敌人+新关系"
- 关键升级点要"足够大"（元婴→化神这种跨级）

### 适配题材

- 玄幻 / 仙侠 / 修真 / 都市异能 / 游戏

### 常见翻车

- ❌ 升级太快——读者没时间代入
- ❌ 升级太慢——读者弃书
- ❌ 升级没代价——失去紧张感
- ❌ 升级后没新地图——失去新鲜感

## 三、无敌流

### 结构骨架

```
主角开局就是最强（或快速获得最强）
   │
   ↓ 用实力碾压各种反派
   │
各个配角/势力登场
   │
   ↓ 偶尔遇到能威胁主角的人
   │
每次都是主角"装作不敌"再反杀
   │
终局：遇到真正能威胁的敌人
```

### 必备元素

- 主角的"无敌感"要强（写得读者相信）
- 反派不能太弱（要让读者担心）
- "装作不敌"的反杀要多次出现
- 偶尔有"能威胁主角"的人出现

### 高潮分布

- 主角"被迫出手"——每次都是高潮
- 反派"装作有实力"被揭穿
- 真正的强敌出现——大高潮

### 适配题材

- 都市 / 兵王 / 战神 / 神豪 / 神医

### 常见翻车

- ❌ 主角太强导致没紧张感
- ❌ 反派太弱让读者觉得无聊
- ❌ 缺少"成长感"（主角不变强）
- ❌ 一开始就无敌导致没期待

## 四、复仇流

### 结构骨架

```
主角带着仇恨重生/觉醒
   │
   ↓ 第一个复仇目标
   │
短期复仇（卷级）——打脸仇人
   │
   ↓ 中期揭示更大仇恨
   │
中期复仇（卷级）——发现幕后黑手
   │
   ↓ 终局面对最终仇人
   │
终局：终极大复仇
```

### 必备元素

- 明确的"仇恨触发事件"（开场就要写清）
- 复仇目标分等级（小仇/中仇/大仇）
- 每个复仇都有"爽点+代价"
- 主角不能"无脑复仇"——要有智慧

### 高潮分布

- 每个小仇的"复仇完成"是高潮
- 揭示更大阴谋是转折
- 终局决战是卷级高潮

### 适配题材

- 都市 / 玄幻 / 历史 / 权谋

### 常见翻车

- ❌ 复仇动机不充分
- ❌ 复仇过程太顺（没阻力）
- ❌ 仇人太弱（无趣）
- ❌ 复仇后主角失去目标

## 五、穿越/重生流

### 结构骨架

```
主角带着前世记忆/现代知识重生
   │
   ↓ 利用"先知"优势
   │
短期：避免前世悲剧
   │
   ↓ 中期：发现更多前世没注意的事
   │
中期：改变命运的关键节点
   │
   ↓ 长期：弥补前世遗憾
   │
终局：这一世过完
```

### 必备元素

- 前世/原世界的清晰设定
- 主角的"先知优势"必须体现
- 改变的代价（蝴蝶效应）
- 主角的两难（原命运 vs 新命运）

### 高潮分布

- 改变前世悲剧——爽点
- 蝴蝶效应导致新问题——转折
- 弥补前世遗憾——情感高潮

### 适配题材

- 历史 / 都市 / 玄幻 / 仙侠重生

### 常见翻车

- ❌ "先知"优势太明显——失去戏剧性
- ❌ 蝴蝶效应没体现——失去真实感
- ❌ 主角知道太多——显得"开挂"
- ❌ 跟前作比没新意

## 六、系统流

### 结构骨架

```
主角获得"系统"——发布任务/奖励
   │
   ↓ 完成新手任务
   │
短期：完成系统任务升级
   │
   ↓ 中期：发现系统背后有人
   │
中期：被系统反向利用
   │
   ↓ 长期：与系统博弈
   │
终局：摆脱系统/与系统合二为一
```

### 必备元素

- 系统的明确规则（任务/奖励/惩罚）
- 任务难度递增
- 系统的"人性化"（让读者喜欢）
- 系统背后的真相（伏笔）

### 高潮分布

- 完成大任务——爽点
- 系统出问题——转折
- 真相揭晓——高潮

### 适配题材

- 都市 / 玄幻 / 游戏 / 无限

### 常见翻车

- ❌ 系统奖励太随意——失去规则感
- ❌ 系统太"工具化"——没有情感
- ❌ 任务同质化
- ❌ 系统背后的真相写崩

## 七、无限流

### 结构骨架

```
主角被拉入"主神空间"等副本系统
   │
   ↓ 第一个副本（熟悉规则）
   │
短期：闯一个个副本
   │
   ↓ 中期：发现副本之间的联系
   │
中期：副本升级 / 真相开始
   │
   ↓ 长期：终极副本
   │
终局：返回现实 / 改变主神
```

### 必备元素

- 副本的清晰规则
- 副本之间的"奖励链"
- 队友的生与死
- 主神空间的存在意义

### 高潮分布

- 每个副本的高潮是"闯关成功"
- 队友牺牲是虐点
- 终极副本是全卷高潮

### 适配题材

- 都市无限 / 灵异 / 玄幻 / 科幻

### 常见翻车

- ❌ 副本规则不清
- ❌ 副本同质化
- ❌ 队友乱死
- ❌ 主神空间的意义没写清

## 八、种田/经营流

### 结构骨架

```
主角接手"破败家业"或创建新势力
   │
   ↓ 短期：解决眼前的生存问题
   │
短期：慢慢发展
   │
   ↓ 中期：势力初步成型
   │
中期：面对外部威胁
   │
   ↓ 长期：势力壮大
   │
终局：势力成为一方霸主
```

### 必备元素

- 主角的"初始资源"（哪怕很小）
- 招募配角的"分阶段"
- 势力的"成长曲线"
- 外部威胁的"分阶段"

### 高潮分布

- 势力的每个里程碑是高潮
- 外部威胁被击退是爽点
- 势力壮大到一定程度是大高潮

### 适配题材

- 都市 / 田园 / 历史 / 经营

### 常见翻车

- ❌ 节奏太慢
- ❌ 主角"全能"（什么都会）
- ❌ 配角同质化
- ❌ 没有明确外部威胁

## 九、权谋/朝堂流

### 结构骨架

```
主角进入朝堂/政治体系
   │
   ↓ 短期：站队
   │
短期：找到靠山
   │
   ↓ 中期：开始反击
   │
中期：一步步清除对手
   │
   ↓ 长期：触及权力核心
   │
终局：登上权力巅峰
```

### 必备元素

- 清晰的派系结构
- 主角的"政治智慧"（不是武力）
- 多方势力的博弈
- 信息差的设计

### 高潮分布

- 每次"反杀"是高潮
- 派系变化是转折
- 终局对决是全卷高潮

### 适配题材

- 历史 / 都市 / 权谋 / 宫斗

### 常见翻车

- ❌ 主角"全能"（没有弱点）
- ❌ 反派太弱智
- ❌ 信息差太大（不真实）
- ❌ 缺少"政治智慧"的具体表现

## 十、混合结构型

> 实际百万级长篇往往是 2-3 种结构型的组合。

### 常见组合

| 主结构 | 辅结构 | 例子 |
|--------|------|------|
| 升级流 | 复仇流 | 主角升级是为了复仇 |
| 穿越流 | 升级流 | 穿越后用金手指升级 |
| 系统流 | 无限流 | 系统发布副本任务 |
| 升级流 | 种田流 | 主角升级 + 创建势力 |
| 复仇流 | 权谋流 | 主角复仇涉及政治斗争 |

### 组合的 4 个原则

1. **主辅分明**：有 1 个主结构 + 1-2 个辅结构
2. **节奏互补**：主结构提供大节奏，辅结构提供小节奏
3. **爽点叠加**：不同结构型提供不同爽点
4. **避免冲突**：不要让两种结构型的核心机制冲突

## 十一、结构型选择决策树

> 不知道选哪种结构型？用这张图判断。

```
你的主角开局是什么？
│
├── 普通人 → 升级流
├── 已经很强 → 无敌流
│
你希望主角的"动机"是什么？
│
├── 复仇 → 复仇流
├── 改变命运 → 穿越/重生流
├── 变强 → 升级流
├── 经营 → 种田流
│
你希望读者获得的核心爽点是什么？
│
├── 实力碾压 → 升级流 / 无敌流
├── 智斗反杀 → 权谋流
├── 闯关惊喜 → 无限流
├── 改变命运 → 穿越/重生流
├── 慢养成 → 种田流
│
你的题材是什么？
│
├── 玄幻/仙侠 → 升级流
├── 都市 → 复仇流 / 无敌流 / 种田流
├── 历史 → 权谋流 / 穿越流
├── 灵异/无限 → 无限流
├── 经营 → 种田流
```

## 十二、结构型自检

> 选定结构型后，逐项检查。

- [ ] 主角的开局状态是否符合该结构型
- [ ] 必备元素是否都设计了
- [ ] 高潮分布是否合理
- [ ] 适配题材是否匹配
- [ ] 是否避开了常见翻车
- [ ] 是否有混合结构（如有，组合是否合理）
- [ ] 全书结尾是否符合该结构型的"标配"
