# 卷纲写作规范

## 基本原则

### 1. 每一卷都要有「不可替代性」

如果这一卷删掉后全书主线不受影响，说明这一卷没有存在的理由。
每一卷必须：
- 在冲突层级上推进了全书主线
- 让主角发生了不可逆的变化（能力/认知/关系至少一项）
- 为下一卷制造了非解决不可的问题

### 2. 卷与卷之间不是「下一章」的关系

用户常犯的错误：把卷当作「长一点的章节」来写。
纠正方法：
- 每一卷都有自己的开卷→中段→卷末弧线
- 每一卷都有自己的卖点——不是「同一个故事继续讲」，而是「同一个故事进入新阶段」
- 每一卷结束时，读者应该觉得「这一卷没白看」，同时「下一卷非追不可」

### 3. 卷纲粒度控制

卷纲不是越细越好。合适的粒度是：

| 粒度 | 应该包含 | 不应该包含 |
|------|---------|-----------|
| 卷纲 | 开卷/中段/卷末、3-5个转折、高潮分布、追读分段 | 章节级细节、场景级对话、具体描写 |
| 节拍卡 | 每章任务、阻力、变化、钩子 | 正文内容 |
| 正文 | 具体叙述、描写、对话 | 大纲级别的规划 |

**判断标准**：从卷纲能直接拆出章节节拍卡，但卷纲不需要代替节拍卡。

### 4. 每卷只做一段升级

不要在一卷里试图做所有事。一卷只做一件事：
- 第一卷：立人+立钩子（让读者知道为什么要追）
- 第二卷：加压+升级（让读者相信能持续好看）
- 第三卷：翻面+深化（让读者感到局面真的变了）
- 第四卷：加压+逼近真相
- 第五卷：收束+兑现

如果一本书有 10 卷，不是「把事情做 10 次」，而是「把一件事的 10 个阶段分别写好」。

## 常见错误

### 错误一：卷纲等于「本章发生了什么」

```
❌ 第一卷：主角加入宗门，参加入门考试，被师兄欺负，遇到神秘老人，获得金手指。
```

这只是事件列表，不是卷纲。改正：

```
✅ 第一卷：起势段——主角从街头孤儿到踏入修仙世界。
核心卖点：看一个被所有人看不起的废柴如何在入门测试中逆袭。
开卷：主角在街头被追杀，走投无路。
中段：入门测试中遭遇针对，被迫暴露隐藏实力。
卷末：成功入门，但被更强的内门弟子盯上。
留下的钩子：那个盯上他的人，为什么对他有莫名的敌意？
```

### 错误二：所有卷的冲突在同一层级

```
❌ 第一卷：和同门争资源
第二卷：和另一宗门争资源
第三卷：和散修争资源
```

如果冲突只在「争资源」这一层反复，读者在第 2 卷就会腻。改正：

```
✅ 第一卷：争资源——能不能活下来
第二卷：争地位——能不能被看见
第三卷：争立场——选择哪个阵营
第四卷：争真相——谁在操纵一切
第五卷：争命运——要不要牺牲自己
```

### 错误三：卷纲只有骨架没有血肉

```
❌ 第一卷
开卷：主角很弱
中段：主角变强了一点
卷末：主角更强了
```

这等于什么都没写。改正需要添加具体约束：

```
✅ 第一卷
开卷：主角是青云宗外门弟子，灵根最差，被人当杂役使唤。但他身体里有一颗上古龙族的残魂。
中段：意外获得秘境试炼资格，被内门弟子设局陷害，龙魂被迫暴露。
卷末：成功通过试炼晋升内门，但龙魂的秘密被宗门某个长老察觉。
留下的钩子：这位长老，似乎正是当年封印龙族的人。
```

### 错误四：卷与卷之间没有升级感

每一卷的卷纲写完后，对照升级阶梯检查：
- 这一卷的主角比上一卷更强了，还是面对了更大的麻烦
- 这一卷的敌人比上一卷更难对付，还是用了新的压迫方式
- 这一卷的关系比上一卷更近了，还是更危险了
- 这一卷揭示的信息比上一卷更多了，还是留下了更大的谜题

如果以上四个问题的答案都是「没变」，这一卷的卷纲需要重做。

## 卷纲质量门

写一卷卷纲时，逐项自检：

- [ ] 这一卷在全书中是不可替代的（删掉会影响主线）
- [ ] 读完卷纲能说清楚「这一卷最值钱的读点是什么」
- [ ] 读完卷纲能说清楚「这一卷结束时主角哪里不一样了」
- [ ] 读完卷纲能说清楚「下一卷为什么非写不可」
- [ ] 这一卷的冲突和上一卷不在同一层级
- [ ] 这一卷的主要对手比上一卷更难对付
- [ ] 这一卷至少有一段关系发生了不可逆变化
- [ ] 这一卷的高潮分布合理（不是一平到底）
- [ ] 这一卷的追读承诺是分段兑现的（不是全部压在卷末）
- [ ] 如果读者只读这一卷（不追全书），他也能获得局部满足

**如果 10 项中超过 3 项不能回答「是」，说明卷纲还需要细化。**
 
 ## 首卷的特殊节奏
 
 第一卷是全书唯一没有前文的卷。这意味着它的节奏型和其他所有卷都不同。
 
 ### 首卷的三个核心差异
 
 **1. 前 3 章决定一切。** 其他卷的第 1-3 章可以做承接、过渡、铺垫。首卷的第 1-3 章必须完成：立主角、抛异常、让读者产生追读决定。没有缓冲余地。
 
 **2. 首卷必须同时做两件事：建立认知 + 推进剧情。** 其他卷可以假设读者已经了解世界观。首卷必须在推进剧情的同时，把世界规则、势力格局、力量体系等信息自然融入剧情——不能单独交代。
 
 **3. 首卷的卷末承诺不是"下卷做什么"而是"全书做什么"。** 其他卷的卷末钩子只针对下一卷。首卷的卷末钩子要让读者对整本书有预期——不只是期待下一卷，而是期待整本书的未来。
 
 ### 首卷的节奏建议
 
 标准首卷节奏（以 12 章为例）：
 
 | 章节 | 功能 | 核心任务 | 节奏提示 |
 |------|------|---------|---------|
 | 1-2 章 | 异常钩子 | 立主角、抛异常、让读者产生"这人有意思"的感觉 | 快！第一章结尾必须有钩子 |
 | 3-4 章 | 压力显现 | 异常转化为具体问题，主角被迫做出第一次选择 | 主角至少主动做一次决定 |
 | 5-7 章 | 世界展开 | 自然带出世界观、引入第一个关键配角/势力 | 世界观融入剧情，不单独说明 |
 | 8-10 章 | 第一次翻面 | 局面不可逆变化、主角面对进入故事以来最大的代价 | 翻面不是"知道了"，而是"失去了" |
 | 11-12 章 | 卷末收束+全书承诺 | 局部满足兑现、留下对全书的长期期待 | 卷末钩子指向全书，不只看下一卷 |
 
 ### 首卷最容易犯的错误
 
 1. **前 3 章铺设定**：用前三章介绍世界观——读者在第 3 章就会弃书。应该在剧情中自然带出。
 2. **主角太被动**：首卷主角如果是"被欺负→被拯救→被收留→被发现"，全程被动接受，读者无法代入。至少让他在前三章做一次主动选择。
 3. **信息量过大**：首卷不需要让读者了解整个世界。只需要够理解当前冲突的信息就够了。
 4. **卷末只有"钩子"没有"满足"**：首卷结尾如果只是"接下来会有大事发生"而没有局部满足，读者会觉得自己被吊了一卷没有任何回报。
 
 ## 跨卷钩子质量标准
 
 每一卷的卷末钩子是维系读者跨卷追读的核心动力。但钩子有质量差异——有些钩子有效，有些钩子是毒钩。
 
 ### 钩子类型与适用位置
 
 | 类型 | 定义 | 最强效果的位置 | 力量来源 |
 |------|------|---------------|---------|
 | **信息钩** | "这件事的真相到底是什么" | 悬疑卷、转折卷、中段卷 | 读者对答案的渴求 |
 | **关系钩** | "他们之间的关系会怎样发展" | 感情线为主的卷、发展卷 | 读者对关系的牵挂 |
 | **压力钩** | "这个危机主角怎么解决" | 开篇卷、升级卷、倒数第二卷 | 读者对主角命运的紧张 |
 | **选择钩** | "主角会做出什么选择" | 转折卷、深化卷、收束卷 | 读者对角色价值观的关注 |
 | **情绪钩** | "这件事之后角色会怎样" | 任何一卷（特别是刚经历重大事件后） | 读者对角色状态的情感连接 |
 
 ### 钩子质量评估表
 
 | 维度 | 强钩子 | 弱钩子 | 毒钩子 |
 |------|-------|-------|-------|
 | **不可回避性** | 下一卷非解决不可，不解决故事会断 | 下一卷可以不碰，转而写别的事 | 读者知道作者下一卷也不会碰 |
 | **因果必然性** | 是这一卷所有事件自然导致的后果 | 是作者强行留下的悬念 | 和本卷事件无关，硬加悬念 |
 | **情感重量** | 读者真正在意这个答案/选择/危机 | 读者觉得"知道了也行" | 读者根本不想知道答案 |
 | **清晰度** | 读者知道下一卷要面对什么 | 读者模糊觉得"可能还有事" | 读者完全不知道钩子指向什么 |
 | **时效性** | 下一卷不马上处理就失去力量 | 放三卷再处理也行 | 放三卷后读者已经忘了 |
 
 ### 钩子自检清单
 
 写卷末钩子后，逐一回答：
 
 - [ ] 这个钩子是本卷事件的自然后果，还是强行加上的？
 - [ ] 如果下一卷不直接从这里开始，读者会感到明显断线吗？
 - [ ] 这个钩子是"读者真正想知道的"，还是"作者想让读者知道的"？
 - [ ] 这个钩子的力量能持续到下一卷出版/更新吗？
 - [ ] 这个钩子是否承诺了下一卷的核心卖点？（"下一卷最值钱的就是这个钩子要回答的问题"）
 - [ ] 如果删掉这个钩子，本卷的完整性会不会受影响？
 
 **底线**：如果一个钩子同时满足以下条件，请直接删除或重写——不是本卷事件的必然后果、读者没有情感关注、下一卷不从这里开始也没关系。
 
 ## 卷序合理性验证
 
 多卷规划完成后，按以下标准检查卷序是否合理。如果超过 3 项不能回答"是"，说明卷序可能需要调整。
 
 ### 必要性验证
 
 对每一卷，问：
 
 - [ ] 删除这一卷，全书主线会受影响吗？
 - [ ] 这一卷在冲突层级上比前一卷更高了吗？
 - [ ] 这一卷结束时，主角的状态和开卷时不可逆地不同了吗？
 - [ ] 这一卷的卷末钩子直接导致下一卷非写不可吗？
 - [ ] 如果把这一卷和下一卷交换位置，会不会出现因果错乱？
 
 ### 连贯性验证
 
 对相邻两卷，问：
 
 - [ ] 卷 N 的卷末钩子是否在卷 N+1 的开头被明确接住？
 - [ ] 卷 N+1 的主角状态是否和卷 N 结束时的状态一致？
 - [ ] 卷 N 引入的新设定/新角色在卷 N+1 是否还在发挥影响？
 - [ ] 卷 N+1 是否不能从卷 N 直接跳过独立展开？（如果能独立展开，说明卷与卷之间的联系太弱）
 - [ ] 两卷之间是否没有"又回到起点"的感觉？（主角在卷 N 已经解决的问题，卷 N+1 不应该重新出现）
 
 ### 升级感验证
 
 对全书的卷序，问：
 
 - [ ] 冲突形态是否逐卷升高（生存→地位→权力→信念→命运）？
 - [ ] 敌人强度是否逐卷质变（同门→宗门→势力→世界→超维）？
 - [ ] 关系深度是否逐卷加深（初识→合作→信任→依赖→牺牲→升华）？
 - [ ] 主角能力/地位是否逐卷提升？
 - [ ] 读者是否能够感受到"每卷都比上一卷更重、更难、更深入"？
 - [ ] 是否有任何一卷读完让读者觉得"和上一卷差不多"？
 
 ### 卷序调整预警信号
 
 如果出现以下迹象，说明卷序可能需要调整：
 
 1. **前置了不该前置的高潮**：第一卷就写了"宗门生死存亡"的大场面，后续卷只能降级写"弟子间小摩擦"。——需要把第一卷冲突调低，或者把全书定位调整为更高强度。
 2. **连续两卷同节奏型**：两卷都是发展型（升级、展开、加入新变量），第二卷让读者感到"这不和上一卷一样吗"。——需要把其中一卷改为转折或深化节奏型。
 3. **反转卷来得太晚**：前三卷全是发展升级，到第四卷才做转折——读者已经在第三卷中段感到腻了。——建议把转折卷提前一卷。
 4. **收束卷工作量过大**：最后一卷既要回收所有伏笔、又要完成主角成长、又要解决最终冲突、又要给每个配角归宿——说明前面的卷在"甩锅"，需要提前分流一部分回收工作到倒数第二卷。
