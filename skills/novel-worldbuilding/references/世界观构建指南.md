> **使用时机**：立项后、写第 1 章前。

## 一、核心原则

**先冲突，后设定。** 设定只有让角色的选择变难时才有价值。

```
冲突需求 → 设定支撑 → 剧情验证
```

| 原则 | 说明 | 反例 |
|------|------|------|
| 先冲突后设定 | 先确定核心矛盾，再设计支撑它的世界规则 | 先写 5000 字世界历史，再发现用不上 |
| 先 hard canon 后 flavor | 不可违反的规则优先，细节可以后补 | 花三天设计货币体系，主角根本不用钱 |
| 人物优先于世界 | 读者通过角色感知世界，不是百科全书 | 开篇 3 章纯世界说明，主角还没出场 |
| 渐进式丰富 | 开书只建骨架，写到哪补到哪 | 开书第一天就把所有设定填满 |

## 三、从零构建五步流程

### Step 1：确定世界类型（30 分钟）

回答三个问题：
1. **规模**：个人冒险 / 国家斗争 / 世界存亡 / 宇宙战争？
2. **基调**：光明 / 灰暗 / 残酷 / 荒诞 / 史诗？
3. **核心差异**：这个世界与现实最大的不同是什么？

### Step 2：画权力地图（1 小时）

1. 列出所有势力（至少 3 个）：
   - 势力 A：欲望 __ / 恐惧 __ / 资源 __ / 弱点 __
   - 势力 B：欲望 __ / 恐惧 __ / 资源 __ / 弱点 __
   - 势力 C：欲望 __ / 恐惧 __ / 资源 __ / 弱点 __

2. 画出势力关系网：
   ```
   A ←[盟友]→ B ←[敌对]→ C
   ```

3. 找出**结构性矛盾**（不可调和的利益冲突）：____

### Step 3：锁死 Hard Canon（30 分钟）

写出 5-10 条绝对不可违反的规则：

| 类型 | 规则 | 约束对象 |
|------|------|---------|
| 物理上限 | | |
| 社会禁忌 | | |
| 资源限制 | | |
| 绝对代价 | | |

> 这些规则将约束后续所有创作，防止战力崩坏。

### Step 4：设计主角的初始位置（30 分钟）

- **社会层级**：底层 / 中层 / 顶层？
- **资源状况**：有 __ / 缺 __
- **信息状况**：知道 __ / 被 __ 蒙蔽
- **行动空间**：能做 __ / 不能做 __

### Step 5：建立最小可运转集（1 小时）

创建以下文件（仅填核心字段，其余留白）：

1. `硬设定.md` —— 5-10 条规则
2. `主角.md` —— 欲望、恐惧、矛盾、秘密
3. `核心冲突.md` —— 对立双方 + 争夺对象 + 不可调和性
4. `开场地点.md` —— 感官细节 + 叙事功能
5. `初始势力.md` —— 欲望、恐惧、与主角关系

> **红线**：这 5 个文件不完成，不开写第 1 章。

## 四、常见陷阱与修复

| 陷阱 | 症状 | 修复 |
|------|------|------|
| 百科全书陷阱 | 设定写 5000 字，第 1 章还没开始 | 问：这个设定会让谁的选择变难？ |
| 铁板一块陷阱 | 势力内部没有矛盾 | 每个势力至少设计 1 个内部裂痕 |
| 免费超能力陷阱 | 力量没有代价，战力无限膨胀 | 每次突破必须有身体/精神/关系/道德代价 |
| 时间混乱陷阱 | 不同地点事件时间对不上 | 建立统一时间轴 |
| 文化真空陷阱 | 世界只有政治和战斗，没有日常 | 设计 3 个日常细节（饮食/节日/禁忌） |
| 设定孤岛陷阱 | 各设定间无关联 | 每个新设定必须链接到已有 2 个设定 |

## 五、世界的动态演化

世界不是静态的——它应该随剧情**被改变**。

**变化触发器**：

| 类型 | 示例 | 追踪方式 |
|------|------|---------|
| 主角造成的 | 主角推翻帝国 | `世界状态变更日志.md` |
| 独立发生的 | 天灾、政变 | `势力兴衰曲线.md` |
| 揭示的 | 原本存在的真相 | `地点变化追踪.md` |

## 六、六维度交叉影响矩阵

> **核心洞察**：六个维度不是独立的——它们相互塑造。改变一个维度，其他维度必然受影响。

### 物理世界 → 社会世界

| 物理特征 | 社会后果 | 剧情机会 |
|---------|---------|---------|
| 资源极度稀缺 | 垄断、黑市、阶级固化 | 底层革命 |
| 频繁天灾 | 宗教兴盛、宿命论 | 科学 vs 信仰 |
| 地理隔绝 | 文化差异大、信息闭塞 | 文化冲突 |
| 魔力/灵气充沛 | 力量阶层、宗门制度 | 凡人逆袭 |

### 社会世界 → 物理世界

| 社会特征 | 物理后果 | 剧情机会 |
|---------|---------|---------|
| 高度集权 | 大兴土木、环境破坏 | 自然反噬 |
| 商业兴盛 | 交通发达、城市化 | 乡村凋零 |
| 战争不断 | 边境焦土、难民潮 | 流民起义 |

### 历史世界 → 规则世界

| 历史事件 | 规则后果 | 剧情机会 |
|---------|---------|---------|
| 曾经的大灾变 | 某些技术/力量被列为禁忌 | 禁忌诱惑 |
| 古代战争 | 某些区域永久封锁 | 探险解禁 |
| 前人成功 | 某条道路被视为唯一正途 | 质疑传统 |

### 规则世界 → 价值世界

| 规则特征 | 价值后果 | 剧情机会 |
|---------|---------|---------|
| 力量可以遗传 | 血统至上、阶层固化 | 平民觉醒 |
| 力量需要献祭 | 人命有价、功利主义 | 道德困境 |
| 力量来源是秘密 | 知识垄断、控制信息 | 信息解放 |

### 价值世界 → 神秘世界

| 价值立场 | 神秘后果 | 剧情机会 |
|---------|---------|---------|
| 极度理性 | 无法解释的现象 = 威胁 | 信仰危机 |
| 极度信仰 | 未知 = 神的领域，不可探究 | 科学觉醒 |
| 实用主义 | 只关心"有没有用" | 伦理失控 |

### 神秘世界 → 物理世界

| 神秘元素 | 物理后果 | 剧情机会 |
|---------|---------|---------|
| 神真实存在 | 神迹改变地理、气候 | 神怒/神恩 |
| 异界可通往 | 地理边界不稳定 | 入侵/殖民 |
| 远古力量残留 | 某些区域异常危险 | 探险/封印 |

## 七、世界类型速查表

> 不同类型世界的构建重点不同——不要试图在一个世界里塞入所有元素。

### 修仙世界

**核心差异**：个体可以通过修炼突破生命极限

| 构建重点 | 设计要点 | 常见陷阱 |
|---------|---------|---------|
| 力量体系 | 等级分明、寿命延长、越级挑战可能 | 战力膨胀、寿命失衡 |
| 社会结构 | 宗门制度、实力为尊、弱肉强食 | 过于黑暗导致读者不适 |
| 资源争夺 | 灵石、灵药、洞天福地 | 资源无限导致无冲突 |
| 历史传承 | 上古遗迹、失落功法、前人布局 | 前人过于强大导致无法超越 |

**关键设定**：寿命与力量的关系、天劫机制、飞升机制

### 奇幻世界

**核心差异**：世界有魔法/超自然力量，但非人人都能使用

| 构建重点 | 设计要点 | 常见陷阱 |
|---------|---------|---------|
| 魔法来源 | 天赋、学习、神赐、契约 | 魔法万能导致无悬念 |
| 社会结构 | 魔法师 vs 凡人、种族差异 | 种族设定流于表面 |
| 政治格局 | 王国、城邦、公会 | 政治过于复杂读者记不住 |
| 神话体系 | 神系、创世神话、预言 | 神过于干预导致凡人无价值 |

**关键设定**：魔法代价、魔法限制、魔法与科技的平衡

### 科幻世界

**核心差异**：技术驱动，物理法则可被理解和利用

| 构建重点 | 设计要点 | 常见陷阱 |
|---------|---------|---------|
| 技术水平 | FTL？AI？基因编辑？寿命延长？ | 技术万能导致无悬念 |
| 社会形态 | 星际帝国、企业国家、AI统治 | 社会过于理想化/妖魔化 |
| 资源约束 | 能源、宜居星球、稀有元素 | 资源无限导致无冲突 |
| 伦理困境 | 克隆、意识上传、基因改造 | 只提出问题不探索答案 |

**关键设定**：技术上限、技术副作用、技术鸿沟

### 武侠世界

**核心差异**：人体极限可通过武学突破，但仍在物理范围内

| 构建重点 | 设计要点 | 常见陷阱 |
|---------|---------|---------|
| 武学体系 | 门派、内功、招式、兵器 | 武功万能导致无悬念 |
| 江湖规则 | 道义、恩怨、面子、规矩 | 规则过于抽象无法执行 |
| 朝堂江湖 | 朝廷与武林的关系 | 两者脱节无互动 |
| 历史背景 | 架空 or 真实历史改编 | 历史考据过多影响叙事 |

**关键设定**：内功上限、武学失传机制、江湖规矩的强制执行

## 八、设定压测方法

> **目的**：在开写前验证设定是否足够支撑剧情。
> **方法**：用"压力测试"模拟剧情场景，检查设定是否崩坏。

### 压测 1：极端场景测试

选择一个极端场景，检查设定是否还能自洽：

| 测试场景 | 检查问题 | 通过标准 |
|---------|---------|---------|
| 主角以弱胜强 | 低级战胜高级是否可能？ | 有明确路径（信息差/环境/策略） |
| 力量极限使用 | 顶级战力全力以赴会怎样？ | 不超过 hard canon 上限 |
| 长期消耗战 | 持续战斗 100 章，战力还合理？ | 升级速度可控 |
| 大规模战争 | 万人战场，个体战力还重要？ | 有个体力挽狂澜的合理条件 |

### 压测 2：多角色测试

让不同角色在同一个场景中行动，检查设定是否公平：

| 角色 A | 角色 B | 同一场景 | 冲突结果 | 是否合理 |
|--------|--------|---------|---------|---------|
| 顶级战力 | 顶级战力 | 正面对决 | | |
| 顶级战力 | 情报专家 | 遭遇战 | | |
| 新手 | 老江湖 | 智斗 | | |

> **规则**：如果某个角色在任何场景都无解强/弱，说明设定失衡。

### 压测 3：时间跨度测试

假设故事写 1000 章，检查设定能否支撑：

- [ ] 主角从 1 级升到顶级，升级曲线合理？
- [ ] 世界在这段时间内有足够的变化？
- [ ] 势力兴衰有迹可循？
- [ ] 硬 canon 没有被突破？

### 压测 4：读者质疑测试

想象读者问这些问题，你能否用设定回答：

- "为什么主角不直接 XXX？" → 设定中是否有明确的限制？
- "为什么反派不早点 XXX？" → 设定中是否有信息差或代价？
- "这个世界为什么没有 XXX？" → 设定中是否有历史原因或物理限制？

## 九、自检清单

开写前检查：
- [ ] 核心冲突能让主角至少做 10 次重大选择
- [ ] 硬设定不超过 10 条（太多记不住）
- [ ] 主角有明确的欲望和恐惧
- [ ] 至少 1 个势力与主角立场对立
- [ ] 世界有 3 个可感知的日常细节
- [ ] 代价系统已经定义（获得力量的必须代价）

---

## 使用提示

1. **不要一次性填满**：Step 5 的 5 个文件完成后即可开写
2. **写到哪补到哪**：新角色出场 → 建角色卡；新势力出场 → 建势力卡
3. **定期回溯**：每卷结束时检查 hard canon 是否被违反
4. **记录变更**：任何 hard canon 的修改必须记录原因
5. **保持神秘**：底层设定（作者掌握的）不要完全揭示给读者
