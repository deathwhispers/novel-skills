> **用途**：设计自洽、有代价、可升级的力量体系。让力量成为冲突的引擎，不是无敌的通行证。
> **核心原则**：没有免费的力量。每次使用都必须有代价。

---

## 一、力量三要素

```
来源 → 表现 → 代价
```

| 要素 | 问题 | 示例 |
|------|------|------|
| **来源** | 力量从哪里来？ | 内力、血统、科技、信仰、契约 |
| **表现** | 力量如何显现？ | 火焰、念力、变形、预知 |
| **代价** | 使用要付出什么？ | 寿命、理智、关系、道德 |

> **规则**：缺少任何一个要素，体系都会崩坏。

## 二、来源设计

### 来源类型

| 类型 | 特点 | 风险 | 典型作品 |
|------|------|------|---------|
| 修炼型 | 后天努力，等级分明 | 升级疲劳 | 修仙、武侠 |
| 血统型 | 先天注定 | 宿命论 | 奇幻、超英 |
| 科技型 | 外部装备，可复制 | 装备依赖 | 科幻 |
| 信仰型 | 与神/灵契约 | 信仰崩溃 | 宗教、暗黑 |
| 情感型 | 情绪驱动，不稳定 | 逻辑混乱 | 奇幻、言情 |
| 知识型 | 理解规则即力量 | 门槛过高 | 克苏鲁 |

### 来源限制

- 可以被剥夺？____
- 可以被转移？____
- 有副作用？____
- 不同来源间冲突？____

### 等级设计原则

1. **金字塔结构**：等级越高，人数越少
2. **能力跃迁**：每个等级要有质变，不是量变
3. **晋升仪式**：需要标志性事件（考试/战斗/觉醒）
4. **晋升代价**：每次晋升失去什么（Innocence/关系/身体）

| 等级 | 剧情阶段 | 主角目标 | 对手等级 |

| 3-4 | 发展 | 建立势力、复仇 | 4-5 |
| 5 | 高潮 | 改变世界 | 5 |

## 四、代价系统

### 代价类型

| 类型 | 描述 | 示例 |
|------|------|------|
| 身体代价 | 伤害身体 | 减寿、残疾、变异 |
| 精神代价 | 侵蚀心智 | 疯狂、失忆、人格分裂 |
| 关系代价 | 伤害他人 | 亲友受伤、被孤立 |
| 道德代价 | 需要作恶 | 杀人、背叛、牺牲无辜 |
| 资源代价 | 消耗稀缺资源 | 灵石、生命力、时间 |

### 代价层次

- **轻微**：偶尔使用无大碍（日常）
- **中度**：频繁使用会累积伤害（战斗）
- **重度**：极限使用会永久损伤或死亡（绝境）

### 代价与选择绑定

> 最好的体系制造"没有完美选择"的困境：
- 用力量救人 → 代价是伤害自己
- 不用力量 → 代价是看着人死
- 找第三种方法 → 代价是冒险和时间

## 五、能力设计

### 能力模板

| 能力名称 | 等级 | 效果 | 限制 | 代价 | 标志表现 |
|---------|------|------|------|------|---------|
| | | | | | |

### 能力设计原则

1. 每个能力都有**反制方法**（没有无敌能力）
2. 能力有**使用场景限制**（雨天不能火攻）
3. 能力间有**克制关系**（水克火，火可蒸水）
4. 能力**强度与代价成正比**

### 组合能力

> 多个能力组合产生新效果或倍增代价：
- A + B = 新效果
- A + B = 代价倍增

## 六、力量与社会的绑定

> **核心原则**：力量体系不是孤立的战斗系统——它塑造社会结构、经济模式、文化价值观。

### 力量如何塑造社会

| 力量特征 | 社会后果 | 示例 |
|---------|---------|------|
| 力量可遗传 | 血统贵族、阶层固化 | 龙族血统、世家制度 |
| 力量需长期修炼 | 师徒制度、宗门体系 | 修仙宗门、武术门派 |
| 力量来源稀缺 | 垄断、黑市、战争 | 灵石矿争夺 |
| 力量可交易 | 商品化、阶级分化 | 魔法道具市场 |
| 力量有副作用 | 歧视、隔离、隐藏 | 异能者被排斥 |

### 力量与经济的绑定

| 经济要素 | 力量的影响 | 剧情机会 |
|---------|-----------|---------|
| 货币 | 力量可否替代货币？ | 以力易物 |
| 劳动力 | 力量是否提高生产力？ | 力工、灵农 |
| 贸易 | 力量差异是否影响贸易？ | 强国垄断 |
| 税收 | 强者是否免税？ | 反抗征税 |

### 力量与文化的绑定

| 文化要素 | 力量的影响 | 剧情机会 |
|---------|-----------|---------|
| 审美 | 强者为美 / 力量为丑 | 外貌歧视 |
| 教育 | 力量训练是否为核心课程？ | 文武分科 |
| 婚姻 | 是否追求力量联姻？ | 政治婚姻 |
| 葬礼 | 强者是否有特殊葬礼？ | 陵墓探险 |

## 七、力量的异常与边缘

> **核心原则**：任何规则都有边缘——边缘地带是故事的富矿。

### 异常类型

| 异常 | 描述 | 对体系的冲击 | 剧情机会 |
|------|------|------------|---------|
| **突变** | 未修炼却觉醒力量 | 打破血统/修炼神话 | 天才觉醒 |
| **退化** | 力量突然丧失 | 强者变凡人 | 重新理解力量 |
| **混合** | 两种力量融合产生新效果 | 打破力量分类 | 双血统主角 |
| **免疫** | 对某种力量完全免疫 | 打破强弱逻辑 | 天生克星 |
| **共鸣** | 与特定环境/人共鸣增强 | 打破个体上限 | 羁绊之力 |

### 边缘群体

| 群体 | 处境 | 诉求 | 剧情机会 |
|------|------|------|---------|
| 无能力者 | 被歧视、被保护 | 尊严 / 独立 | 凡人的智慧 |
| 失控者 | 力量不受控 | 控制 / 解脱 | 悲剧英雄 |
| 异端 | 使用禁忌力量 | 力量 / 接纳 | 禁忌的诱惑 |
| 残废强者 | 失去力量 | 复仇 / 重生 | 浴火重生 |

## 八、力量的叙事功能

> **核心原则**：力量不仅是战斗工具——它是角色内心的外化。

### 力量 = 内心外化

| 角色内心 | 力量表现 | 剧情隐喻 |
|---------|---------|---------|
| 压抑的愤怒 | 爆发型力量（失控） | 情绪需要出口 |
| 渴望控制 | 精密操控型力量 | 对失控的恐惧 |
| 自我否定 | 力量不稳定 | 自我认同危机 |
| 渴望联结 | 共鸣型力量（需他人） | 独立的反面 |

### 力量的"觉醒"设计

| 觉醒触发 | 内心状态 | 力量表现 | 剧情意义 |
|---------|---------|---------|---------|
| 保护他人 | 爱 / 责任 | 超越极限 | 角色成长 |
| 极度愤怒 | 失控 | 毁灭性力量 | 代价显现 |
| 放下执念 | 释然 | 力量质变 | 主题揭示 |
| 接纳自我 | 认同 | 力量稳定 | 角色成熟 |

## 九、平衡检查

### 战力膨胀检查

- [ ] 顶级战力有明确上限（不能毁灭世界）
- [ ] 低级在特定条件可战胜高级（情报/准备/环境）
- [ ] 战力差距不是绝对的
- [ ] 升级速度合理（不是一卷从 1 跳到 10）

### 代价一致性

- [ ] 每次使用力量的代价都被记录
- [ ] 累积代价有显现（身体/精神恶化）
- [ ] 代价不能被轻易消除（无万能解药）
- [ ] 代价驱动剧情（角色因代价做选择）

---

## 使用提示

1. **开书前**：确定来源类型、3-5 个等级、核心代价
2. **新增能力**：检查是否与现有体系冲突
3. **角色升级**：记录代价累积
4. **战斗设计**：利用限制和克制制造智斗
5. **严禁**：战力无限膨胀——每次升级必须有代价
