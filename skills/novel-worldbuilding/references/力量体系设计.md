# 力量体系设计

> **用途**：设计自洽、有代价、可升级的力量体系。让力量成为冲突的引擎，不是无敌的通行证。
> **核心原则**：没有免费的力量。每次使用都必须有代价。

---

## 一、力量三要素

```
来源 → 表现 → 代价
```

| 要素 | 问题 | 示例 |
|------|------|------|
| **来源** | 力量从哪里来？ | 内力、血统、科技、信仰、契约 |
| **表现** | 力量如何显现？ | 火焰、念力、变形、预知 |
| **代价** | 使用要付出什么？ | 寿命、理智、关系、道德 |

> **规则**：缺少任何一个要素，体系都会崩坏。

---

## 二、来源设计

### 来源类型

| 类型 | 特点 | 风险 | 典型作品 |
|------|------|------|---------|
| 修炼型 | 后天努力，等级分明 | 升级疲劳 | 修仙、武侠 |
| 血统型 | 先天注定 | 宿命论 | 奇幻、超英 |
| 科技型 | 外部装备，可复制 | 装备依赖 | 科幻 |
| 信仰型 | 与神/灵契约 | 信仰崩溃 | 宗教、暗黑 |
| 情感型 | 情绪驱动，不稳定 | 逻辑混乱 | 奇幻、言情 |
| 知识型 | 理解规则即力量 | 门槛过高 | 克苏鲁 |

### 来源限制

- 可以被转移？____
- 有副作用？____
- 不同来源间冲突？____

### 等级设计原则

1. **金字塔结构**：等级越高，人数越少
2. **能力跃迁**：每个等级要有质变，不是量变
3. **晋升仪式**：需要标志性事件（考试/战斗/觉醒）
4. **晋升代价**：每次晋升失去什么（Innocence/关系/身体）

| 3-4 | 发展 | 建立势力、复仇 | 4-5 |

## 四、代价系统

### 代价类型

| 类型 | 描述 | 示例 |
|------|------|------|
| 身体代价 | 伤害身体 | 减寿、残疾、变异 |
| 精神代价 | 侵蚀心智 | 疯狂、失忆、人格分裂 |
| 关系代价 | 伤害他人 | 亲友受伤、被孤立 |
| 道德代价 | 需要作恶 | 杀人、背叛、牺牲无辜 |
| 资源代价 | 消耗稀缺资源 | 灵石、生命力、时间 |

### 代价层次

- **轻微**：偶尔使用无大碍（日常）
- **中度**：频繁使用会累积伤害（战斗）
- **重度**：极限使用会永久损伤或死亡（绝境）

### 代价与选择绑定

> 最好的体系制造"没有完美选择"的困境：
- 用力量救人 → 代价是伤害自己
- 不用力量 → 代价是看着人死
- 找第三种方法 → 代价是冒险和时间

| | | | | | |

### 能力设计原则

1. 每个能力都有**反制方法**（没有无敌能力）
2. 能力有**使用场景限制**（雨天不能火攻）
3. 能力间有**克制关系**（水克火，火可蒸水）
4. 能力**强度与代价成正比**

### 组合能力

> 多个能力组合产生新效果或倍增代价：
- A + B = 新效果
- A + B = 代价倍增

## 六、平衡检查

### 战力膨胀检查

- [ ] 顶级战力有明确上限（不能毁灭世界）
- [ ] 低级在特定条件可战胜高级（情报/准备/环境）
- [ ] 战力差距不是绝对的
- [ ] 升级速度合理（不是一卷从 1 跳到 10）

### 代价一致性

- [ ] 每次使用力量的代价都被记录
- [ ] 累积代价有显现（身体/精神恶化）
- [ ] 代价不能被轻易消除（无万能解药）
- [ ] 代价驱动剧情（角色因代价做选择）

---

## 使用提示

1. **开书前**：确定来源类型、3-5 个等级、核心代价
2. **新增能力**：检查是否与现有体系冲突
3. **角色升级**：记录代价累积
4. **战斗设计**：利用限制和克制制造智斗
5. **严禁**：战力无限膨胀——每次升级必须有代价
