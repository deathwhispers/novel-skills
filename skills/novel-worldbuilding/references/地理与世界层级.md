## 一、世界层级结构

### 层级金字塔

```
        [世界/位面/宇宙]
              ↓
        [大陆/海域/区域]
              ↓
        [国家/宗门/城邦]
              ↓
        [城市/据点/聚落]
              ↓
        [核心场景/建筑/秘境]
```

### 各层级定义

| 层级 | 规模 | 功能 | 设计要点 |
|------|------|------|---------|
| 世界 | 最大边界 | 确定故事的上限 | 有哪些位面/大陆？是否可穿越？ |
| 大陆 | 地理单元 | 气候、资源、文化的差异 | 每个大陆的核心特征是什么？ |
| 国家 | 政治单元 | 权力、法律、军队的载体 | 国家之间为什么没统一？ |
| 城市 | 生活单元 | 角色的日常活动场所 | 城市的"气味"是什么？ |
| 场景 | 叙事单元 | 具体剧情发生地 | 这个场景适合发生什么戏？ |

## 四、地点的动态变化

地点应该随剧情**变化**，不是静态背景。

### 变化追踪表

| 章节 | 地点 | 变化事件 | 变化前 | 变化后 | 对剧情影响 |
|------|------|---------|--------|--------|-----------|
| | | | | | |

### 变化的三种类型

1. **物理变化**：被毁、重建、改造、易主
2. **社会变化**：权力更迭、人口流动、功能转变
3. **情感变化**：角色对地点的情感转变（从安全到恐惧）

## 六、地理与剧情的绑定

### 地理驱动剧情

| 地理特征 | 制造的冲突 | 剧情机会 |
|---------|----------|---------|
| 高山阻隔 | 信息不通、救援困难 | 信使冒险、翻越天险 |
| 资源稀缺 | 争夺、垄断、黑市 | 走私、谈判、战争 |
| 季节变化 | 通行受阻、作物歉收 | 饥荒、迁徙、囤货 |

---

## 使用提示

1. **开书前**：设计 3-5 个核心场景，每个有明确叙事功能
2. **新地点出场**：复制模板，填基本信息和感官细节
3. **地点反复使用**：每次重新阅读感官细节，保持一致性
4. **地点被破坏/改变**：更新变化追踪表
5. **严禁**：地点只有名字没有氛围——读者通过感官感受地点
