# 时间线与历史

> **用途**：追踪世界历史和故事时间线，确保因果链完整、时间逻辑自洽。
> **使用时机**：开书前建立框架，每写一章更新。

---

## 一、时间线结构

### 四层时间轴

```
远古史（模糊）→ 近代史（详细）→ 当代史（精确）→ 当前（精确到时刻）
```

| 层级 | 时间精度 | 内容 | 功能 |
|------|---------|------|------|
| 远古史 | 百年/千年 | 起源、神话、第一次大事件 | 营造神秘感 |
| 近代史 | 年 | 50-100 年前的关键事件 | 解释当前格局 |
| 当代史 | 月/日 | 主角出生到现在的大事 | 影响主角人生 |
| 故事时间 | 时刻 | 每章的时间标记 | 精确追踪 |

---

## 二、各层级设计

### 远古史模板

> **原则**：模糊处理，只记录对当前剧情有影响的事件。

| 时间 | 事件 | 遗产（至今影响） | 当前状态 |
|------|------|----------------|---------|
| | | | |

**远古史使用规则**：
- 不直接揭示——通过遗迹、传说、禁忌暗示
- 每个远古事件必须链接到当前的某个冲突或规则
- 不影响当前剧情的远古事件——删除

### 近代史模板

> **原则**：详细记录。这是当前格局的根源。

| 时间 | 事件 | 直接后果 | 遗留问题 |
|------|------|---------|---------|
| | | | |

**近代史关键问题**：
- 这些事件创造了什么伤疤？
- 哪些势力因此崛起？哪些因此衰落？
- 主角的父辈/前辈在这些事件中扮演什么角色？
- 有哪些未解决的仇恨或承诺？

### 当代史模板

> **原则**：精确记录。直接影响主角的人生。

| 年份 | 事件 | 涉及角色 | 对主角的影响 |
|------|------|---------|------------|
| | | | |

**当代史的个人时间线**：
- 主角出生前：父母的关键事件
- 主角 0-10 岁：童年形成事件
- 主角 10-18 岁：性格定型事件
- 主角 18-现在：触发故事的事件

## 三、时间规则

### Hard 时间规则（不可违反）

| 规则 | 内容 | 违反后果 |
|------|------|---------|
| 通行时间 | 地点 A 到 B 最短 ____ 天 | 角色不可能瞬移 |
| 仪式周期 | ____ 事件每 ____ 发生一次 | 无法随意提前/延后 |
| 成长时间 | 修炼/养伤/怀孕最低 ____ 时间 | 无法速成 |
| 信息传递 | 消息从 A 到 B 需 ____ 天 | 无法即时通讯 |

## 四、时间矛盾检查

写完一卷后检查：

- [ ] 同一角色不可能在同一时间出现在两个地点
- [ ] 事件的因果顺序正确（先因后果）
- [ ] 通行时间与地理距离匹配
- [ ] 多线叙事的时间同步无误
- [ ] 季节/天气描述与时间一致
- [ ] 角色的年龄/成长时间与时间线匹配
- [ ] 信息的传递时间合理（没有未卜先知）

## 使用提示

1. **开书前**：建立第二、三层（近代史、当代史）的基础框架
2. **每写一章**：标注时间标记，更新章节级时间线
3. **发现时间矛盾**：立即修正，更新所有关联记录
4. **引入远古史**：保持模糊，只记录与当前相关的部分
5. **严禁**：时间线只是背景——每个时间点都必须是因果链的一环
