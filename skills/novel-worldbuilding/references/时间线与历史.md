## 一、时间线结构

### 四层时间轴

```
远古史（模糊）→ 近代史（详细）→ 当代史（精确）→ 当前（精确到时刻）
```

| 层级 | 时间精度 | 内容 | 功能 |
|------|---------|------|------|
| 远古史 | 百年/千年 | 起源、神话、第一次大事件 | 营造神秘感 |
| 近代史 | 年 | 50-100 年前的关键事件 | 解释当前格局 |
| 当代史 | 月/日 | 主角出生到现在的大事 | 影响主角人生 |
| 故事时间 | 时刻 | 每章的时间标记 | 精确追踪 |

## 二、各层级设计

### 远古史模板

> **原则**：模糊处理，只记录对当前剧情有影响的事件。

| 时间 | 事件 | 遗产（至今影响） | 当前状态 |
|------|------|----------------|---------|
| | | | |

**远古史使用规则**：
- 不直接揭示——通过遗迹、传说、禁忌暗示
- 每个远古事件必须链接到当前的某个冲突或规则
- 不影响当前剧情的远古事件——删除

### 近代史模板

> **原则**：详细记录。这是当前格局的根源。

| 时间 | 事件 | 直接后果 | 遗留问题 |
|------|------|---------|---------|
| | | | |

**近代史关键问题**：
- 这些事件创造了什么伤疤？
- 哪些势力因此崛起？哪些因此衰落？
- 主角的父辈/前辈在这些事件中扮演什么角色？
- 有哪些未解决的仇恨或承诺？

### 当代史模板

> **原则**：精确记录。直接影响主角的人生。

| 年份 | 事件 | 涉及角色 | 对主角的影响 |
|------|------|---------|------------|
| | | | |

**当代史的个人时间线**：
- 主角出生前：父母的关键事件
- 主角 0-10 岁：童年形成事件
- 主角 10-18 岁：性格定型事件
- 主角 18-现在：触发故事的事件

## 三、预言、征兆与循环

> **用途**：预言不是剧透——是制造紧张感的工具。征兆不是装饰——是暗示命运的方式。

### 预言系统设计

| 要素 | 设计要点 | 示例 |
|------|---------|------|
| **预言来源** | 神谕、先知、古籍、梦境 | 神庙预言、家族诅咒 |
| **预言模糊度** | 越模糊越安全，越精确越危险 | "有人将带来毁灭" vs "主角将在 30 天后死亡" |
| **预言实现方式** | 自证预言 / 反讽实现 / 延迟实现 | 俄狄浦斯式、麦克白式 |
| **预言可信度** | 绝对可信 / 部分可信 / 完全虚假 | 读者知道真/假？ |

### 征兆与预兆

| 类型 | 表现 | 含义 | 使用频率 |
|------|------|------|---------|
| 自然征兆 | 血月、地震、异兽 | 大事件将至 | 稀用 |
| 个人征兆 | 噩梦、幻觉、身体不适 | 角色命运转折 | 偶尔 |
| 社会征兆 | 流言、恐慌、异常行为 | 社会即将动荡 | 根据需要 |

**征兆使用规则**：
- 征兆不应直接揭示结果——只暗示方向
- 同一个征兆可以被不同角色不同解读
- 虚假的征兆（误读）可以制造戏剧性

### 时间循环与重复

| 循环类型 | 机制 | 叙事功能 | 示例 |
|---------|------|---------|------|
| 历史循环 | 同样的事件周期性发生 | 宿命感 | 王朝更替 |
| 个人循环 | 角色重复前人的命运 | 家族诅咒 | 世代恩怨 |
| 季节循环 | 自然周期影响剧情 | 时间压力 | 必须在冬天前完成 |

## 四、历史事件的回响设计

> **核心原则**：历史事件不是只发生一次——它们的回响会在不同时间点以不同形式出现。

### 回响模型

```
原始事件（近代史）
  → 第一代回响（当代史）：直接参与者的人生被改变
    → 第二代回响（故事时间）：后代/继承者面对遗产
      → 第三代回响（高潮）：原始事件的最终清算
```

### 回响追踪表

| 原始事件 | 第一代回响 | 第二代回响 | 第三代回响（高潮） |
|---------|-----------|-----------|------------------|
| 帝国建立 | 开国功臣崛起 | 功臣后代成为贵族 | 贵族腐败导致革命 |
| 大屠杀 | 幸存者 PTSD | 幸存者的孩子复仇 | 复仇引发新战争 |
| 技术突破 | 技术垄断 | 垄断导致阶层固化 | 技术泄露引发混乱 |

### 回响的检查

- [ ] 每个重大历史事件都有至少 2 个回响
- [ ] 回响在不同代际以不同形式出现
- [ ] 主角是某个回响的参与者（而非旁观者）
- [ ] 高潮时刻是多个回响的交汇点

## 五、时间规则

### Hard 时间规则（不可违反）

| 规则 | 内容 | 违反后果 |
|------|------|---------|
| 通行时间 | 地点 A 到 B 最短 ____ 天 | 角色不可能瞬移 |
| 仪式周期 | ____ 事件每 ____ 发生一次 | 无法随意提前/延后 |
| 成长时间 | 修炼/养伤/怀孕最低 ____ 时间 | 无法速成 |
| 信息传递 | 消息从 A 到 B 需 ____ 天 | 无法即时通讯 |

### Soft 时间规则（弹性处理）

- 不同地区的时间感知差异：____
- 特殊地点的时间流速差异（如秘境）：____
- 节日、庆典等文化时间：____

## 四、时间矛盾检查

写完一卷后检查：

- [ ] 同一角色不可能在同一时间出现在两个地点
- [ ] 事件的因果顺序正确（先因后果）
- [ ] 通行时间与地理距离匹配
- [ ] 多线叙事的时间同步无误
- [ ] 季节/天气描述与时间一致
- [ ] 角色的年龄/成长时间与时间线匹配
- [ ] 信息的传递时间合理（没有未卜先知）

## 使用提示

1. **开书前**：建立第二、三层（近代史、当代史）的基础框架
2. **每写一章**：标注时间标记，更新章节级时间线
3. **发现时间矛盾**：立即修正，更新所有关联记录
4. **引入远古史**：保持模糊，只记录与当前相关的部分
5. **严禁**：时间线只是背景——每个时间点都必须是因果链的一环
