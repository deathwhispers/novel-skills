# 时间线管理模板

> **使用说明**：追踪世界历史和当前时间线。不是编年史——是**因果链的记录**。
> 每个时间点都要回答：它导致了什么？它被什么导致？

---

## 时间线总览

### 时间尺度
- **故事总跨度**：____ 年 / ____ 月 / ____ 日
- **当前时间点**：故事开始后的第 ____ 天
- **时间推进速度**：1 章 = ____ 天（不固定，标注每章的时间跨度）

### 时间层级
```
远古史（模糊）→ 近代史（详细）→ 当代史（精确到天）→ 当前（精确到时刻）
```

---

## 第一层：远古史（世界的起源与神话）

> **原则**：模糊处理，保留神秘感。只记录对当前剧情有影响的事件。

| 时间 | 事件 | 影响至今的遗产 | 当前状态 |
|------|------|---------------|---------|
| | | | |

**远古史的使用规则**：
- 不直接揭示——通过遗迹、传说、禁忌暗示
- 每个远古事件都要链接到当前的某个冲突或规则
- 如果某个远古事件不影响当前剧情，删除它

---

## 第二层：近代史（50-100 年前的关键事件）

> **原则**：详细记录。这是当前格局的根源。

| 时间 | 事件 | 直接后果 | 当前遗留问题 |
|------|------|---------|------------|
| | | | |

**近代史的关键问题**：
- 这些事件创造了什么伤疤？
- 哪些势力因此崛起？哪些因此衰落？
- 主角的父辈/前辈在这些事件中扮演什么角色？
- 有哪些未解决的仇恨或承诺？

---

## 第三层：当代史（主角出生到现在）

> **原则**：精确记录。直接影响主角的人生。

| 年份 | 事件 | 涉及角色 | 对主角的影响 |
|------|------|---------|------------|
| | | | |

**当代史的个人时间线**：
- 主角出生前：父母的关键事件
- 主角 0-10 岁：童年形成事件
- 主角 10-18 岁：性格定型事件
- 主角 18-现在：触发故事的事件

---

## 时间线规则

### 硬时间规则（不可违反）
- 地点 A 到地点 B 的最短通行时间：____
- 某个仪式/事件的固定周期：____
- 角色成长的最低时间（如修炼、养伤）：____

### 软时间规则（弹性处理）
- 不同地区的时间感知差异
- 特殊地点的时间流速差异（如秘境）
- 节日、庆典等文化时间

---

## 时间线矛盾检查表

写完一卷后检查：
- [ ] 同一角色不可能在同一时间出现在两个地点
- [ ] 事件的因果顺序正确（先因后果）
- [ ] 通行时间与地理距离匹配
- [ ] 多线叙事的时间同步无误
- [ ] 季节/天气描述与时间一致
- [ ] 角色的年龄/成长时间与时间线匹配

## 使用提示

1. **开书前**：建立第二、三层的基础框架
2. **每写一章**：标注时间标记，更新章节级时间线
3. **发现时间矛盾时**：立即修正，更新所有关联记录
4. **引入远古史时**：保持模糊，只记录与当前相关的部分
5. **严禁**：时间线只是背景——每个时间点都必须是因果链的一环
