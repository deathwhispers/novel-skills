# 母题系统指南

> **使用说明**：母题（Motif）是反复出现的主题、象征、意象，是故事的"DNA"。
> 母题不是装饰——是**结构工具**，用来强化情感、暗示命运、制造回响。

---

## 一、母题的三个层级

```
大母题（主题） → 中母题（象征） → 小母题（意象）
```

| 层级 | 功能 | 示例 |
|------|------|------|
| **大母题** | 故事的核心追问 | "自由与秩序的冲突" |
| **中母题** | 反复出现的象征物 | 一只折翼的鸟、断裂的锁链 |
| **小母题** | 场景中的意象细节 | 雨天、红色、镜子、火焰 |

> **规则**：每部小说应该有 1-3 个大母题，3-5 个中母题，无数个小母题。

---

## 二、大母题：故事的核心主题

### 设计方法
回答三个问题：
1. **这个问题在故事开始时的答案是什么？**（初始立场）
2. **这个问题在故事结束时的答案是什么？**（最终立场）
3. **主角在这个问题上的转变是什么？**（弧线核心）

### 常见大母题类型

| 类型 | 追问 | 适合的故事 |
|------|------|-----------|
| **自由 vs 秩序** | 个人自由重要还是集体稳定重要？ | 反乌托邦、武侠 |
| **传承 vs 革新** | 传统应该守护还是打破？ | 家族、修仙 |
| **复仇 vs 宽恕** | 伤害必须报复还是可以放下？ | 悬疑、复仇 |
| **真相 vs 和平** | 知道真相更重要还是维持和平更重要？ | 政治、谍战 |
| **个人 vs 集体** | 一个人应该为群体牺牲到什么程度？ | 战争、灾难 |
| **命运 vs 选择** | 命运可以被改变吗？ | 奇幻、科幻 |
| **永生 vs 死亡** | 永生是祝福还是诅咒？ | 玄幻、科幻 |

- **象征意义**：代表什么主题/情感/命运？
- **出场规律**：何时出现？（高潮前？决策时？悲剧后？）
- **变化轨迹**：象征物的状态是否随剧情变化？

### 象征物的使用规则
1. **不要解释**——让读者自己感受到象征意义
2. **不要过度使用**——每次出场要有新意
3. **让象征物参与剧情**——不只是背景，而是推动事件
4. **最终揭示**——在高潮时刻，象征物的真正意义才被完全理解

## 五、母题的编排策略

### 卷级母题分布
| 卷数 | 大母题呈现 | 中母题出场 | 小母题密度 |
|------|-----------|-----------|-----------|
| 卷一（建立） | 提出问题 | 引入象征物 | 低密度 |
| 卷二（深化） | 主角选择立场 | 象征物首次变化 | 中密度 |
| 卷三（转折） | 立场受到挑战 | 象征物被毁/重生 | 高密度 |
| 卷四（高潮） | 最终答案 | 象征物最终揭示 | 最高密度 |

### 母题的回响设计
> 在高潮时刻，让早期出现的母题再次出现，制造"回响"效果。

**回响公式**：
- 第 1 次出现（建立）：让读者注意到
- 第 2 次出现（提醒）：让读者想起第 1 次
- 第 3 次出现（回响）：在高潮时刻出现，情感爆炸

## 七、母题检查清单

每卷完成后检查：
- [ ] 大母题在本卷是否有明确呈现？
- [ ] 中母题是否至少出场一次？
- [ ] 小母题是否自然融入场景（不突兀）？
- [ ] 母题是否参与了剧情（不只是装饰）？
- [ ] 母题的状态是否有变化（如象征物被毁）？
- [ ] 读者能否感受到母题的存在（不需要解释）？

## 使用提示

1. **开书前**：确定 1-3 个大母题和 3-5 个中母题
2. **每卷规划时**：安排母题的出场时机和状态变化
3. **写场景时**：检查是否可自然融入一个小母题
4. **高潮设计时**：让母题回响，制造情感冲击
5. **严禁**：母题只是装饰——每个母题都必须参与剧情或情感
