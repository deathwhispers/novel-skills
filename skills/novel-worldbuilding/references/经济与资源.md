## 七、资源替代链

> **用途**：当主资源枯竭时，什么可以替代？替代过程如何制造冲突？

### 替代链设计

```
主资源（枯竭）→ 第一替代（昂贵）→ 第二替代（副作用）→ 终极替代（变革）
```

| 阶段 | 资源 | 状态 | 社会反应 | 剧情机会 |
|------|------|------|---------|---------|
| 充裕期 | | 廉价丰富 | 浪费、依赖 | 建立依赖 |
| 紧缺期 | | 价格上涨 | 节约、寻找替代 | 恐慌、投机 |
| 枯竭期 | | 几乎绝迹 | 争夺、战争 | 资源战争 |
| 替代期 | | 替代品出现 | 适应、变革 | 技术革命 |

### 替代冲突

| 替代资源 | 被替代资源 | 利益受损方 | 受益方 | 冲突 |
|---------|-----------|-----------|--------|------|
| 新能源 | 旧能源 | 旧能源势力 | 新势力 | 新旧战争 |
| 新货币 | 旧货币 | 旧货币持有者 | 发行者 | 货币战争 |

## 使用提示

1. **开书前**：确定 1-2 个战略资源和货币系统
2. **势力冲突时**：考虑经济动机（不只是意识形态）
3. **角色行动时**：考虑经济约束（没有钱寸步难行）
4. **制造危机时**：经济崩溃是最真实的危机
5. **严禁**：势力行动没有经济逻辑——战争很花钱
