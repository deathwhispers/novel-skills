# 写中 Prompt

> 每次只加载当前 Prompt。先给写作模式、人物状态、当前场景和可靠 voice；所有篇幅、段数、强度与结尾方式服从实际写作效果。

## 目录

对话；潜台词与群像；行动；亲密关系；心理；环境；类型兑现；结尾；节奏；续写；视角；形式实验；卡文续段。

## Prompt 1：对话

```text
写一段发生在具体空间里的对话。让双方各自注意不同事物，并通过措辞、回避、动作、沉默和信息策略显出差异。

【人物与关系】{{}}
【表面话题与真实需要】{{}}
【当前压力】{{}}
【voice 证据】{{}}

不要为每个角色制造口头禅，也不强制最后一句成为钩子。对话结束在关系或理解发生有效变化的位置。
```

## Prompt 2：潜台词或群像

```text
【适用】双人潜台词或多人场景。

让每个人只在其立场、知识和风险允许的范围内说话。用注意力、插话、结盟、沉默和空间位置管理群像，不要求人人平均发言。

输出可直接进入正文的场景，不附人物功能说明。
```

## Prompt 3：行动、战斗或追逐

```text
从当前视角写行动场景。空间、能力限制、判断和身体反馈必须可追踪；每个动作来自人物的即时目标，而不是招式展示。

结果可以是胜负、逃脱、误判、关系改变或新认识。只有故事已建立相应 stakes 时才加入重大代价，不强制造成不可逆伤亡。
```

## Prompt 4：亲密、告白或诀别

```text
从双方既有关系与此刻风险出发写场景。让情感通过具体选择、迟疑、身体距离和没有说出口的东西出现。

不要套固定阶段，也不要自动用拥抱、眼泪、牺牲或误会证明强度。结尾服从人物当下能承受的真实程度。
```

## Prompt 5：心理变化

```text
写人物在行动或感知中的心理运动。让外部刺激、旧经验、身体反应和选择互相作用；保留矛盾与未命名感受。

避免连续解释“他意识到/这意味着”。若人物无法得出结论，就停在其能感知的边界。
```

## Prompt 6：环境与意象

```text
通过当前视角选择环境细节。只写人物此刻会注意且能改变氛围、行动、关系或主题回声的内容。

不要把环境写成独立说明书，也不要让所有景物机械对应情绪。允许环境与人物感受抵牾。
```

## Prompt 7：类型兑现（按需）

```text
【适用】商业连载、类型长篇，或作者明确要求某种类型体验。

写出前因、人物投入、压力积累、兑现与后果。爽感、恐惧、悬疑或痛感必须从人物和因果中产生；不按固定强度、段数或字数生成。
```

## Prompt 8：章尾或场景尾

```text
根据写作模式生成 3 个不同落点：推动型、余震/回响型、留白/暂时停顿型。每个版本说明它把读者注意力留在哪里。

商业连载可优先考虑继续阅读动力；文学叙事、短篇和探索起草允许封闭、安静或歧义。不要凭空添加陌生人、秘密、倒计时或反转。
```

## Prompt 9：节奏与场景切换

```text
检查当前片段的注意力和压力如何运动。只在重复、跳跃、必要过程缺失或读者无法定位时调整。

场景长短、切换数量和快慢没有固定比例。给出最小改动，并说明它改善的是清晰度、张力、沉浸、呼吸还是形式效果。
```

## Prompt 10：续写防漂

```text
基于最近有效正文、章节增量、人物状态、hard canon 和当前任务续写。

只接住现在仍活跃且与当前人物有关的后果；不要求一章处理所有悬念，也不强制新埋一条债。延续 voice 的注意力、回避方式和句法倾向，不复制标志句。
```

## Prompt 11：视角设计

```text
判断维持或切换视角哪一种更能控制体验和信息。若切换，明确新视角独有的欲望、盲区和行动，并给读者足够定位信号。

不使用固定章节间隔、戏份比例或“同章绝不切换”的通用禁令；只有非预期跳转和无法定位才是硬伤。
```

```text

```

## Prompt 13：卡文最小续段

```text
不要补新设定。给出 3 个不超过一个动作或一次感知变化的续段方向：人物选择、关系/信息变化、叙述距离变化。说明各自会打开什么可能，再按作者选择继续正文。
```
