# 写中阶段 AI Prompt 模板库

> **使用说明**：写正文时的具体技法——对话、战斗、感情、章尾、描写等。
> 每个 prompt 都对应 novel-writing 的具体技法模板。

---

## 一、对话写作

### Prompt 1：写一段对话

```
请按以下要求，写一段对话。

【对话信息】
- 参与角色：{{角色A}} 和 {{角色B}}
- 场景：{{}}
- 对话目的：{{信息传递/关系推进/冲突对抗/情感表达/氛围营造}}
- 字数要求：{{}}
- 必须承担的功能：{{推进剧情/改变关系/揭示设定/表达情感}}

【对话写作技法的 5 段式】
- 段 1 铺垫（1-2 轮）：建立语境
- 段 2 主体交锋（3-N 轮）：信息差驱动
- 段 3 收尾落点（1 轮）：留下钩子

【对话写作技法】
（请 AI 查阅 novel-writing/references/写作技法库/对话写作技法.md）

【角色设定】
- 角色 A：{{粘贴角色卡}}
- 角色 B：{{粘贴角色卡}}

【角色辨识度】
- 角色 A 的口头禅：{{}}
- 角色 B 的口头禅：{{}}

请输出：
1. 完整对话（用 "角色名：'对话'" 的格式）
2. 对话中的动作/环境描写
3. 对话的"刺点"（最后一句）
4. 对话后角色的内心变化（简短）
```

### Prompt 2：潜台词对话

```
请写一段有潜台词的对话。

【对话信息】
- 参与角色：{{角色A}} 和 {{角色B}}
- 场景：{{}}
- 表面话题：{{}}
- 真正要表达：{{}}
- 关系当前状态：{{}}

【潜台词的 5 种实现技法】
1. 反话正说（嘴上不在意，行为在意）
2. 答非所问（用无关回答暗示态度）
3. 动作代替语言（用动作表达说不出口的话）
4. 沉默与停顿（用空白承载情绪）
5. 重复对方的话（用回声暗示内心）

请输出：完整对话（重点是潜台词，不是直说）
```

### Prompt 3：对话刺点设计

```
请为一个对话场景设计"刺点"——最后 1-2 句让读者立刻想翻下一章的钩子。

【场景信息】
- 场景：{{}}
- 已发生：{{}}
- 当前关系：{{}}
- 刺点类型：{{新问题/不可逆决定/倒计时/权力反转/情绪余震/关系破口}}

【刺点的 4 种类型】
1. 悬念刺：抛出新问题
2. 反转刺：推翻读者预期
3. 情感刺：让读者情感强烈反应
4. 行动刺：用动作代替语言

请输出：3 个不同版本的刺点，让我挑选。
```

## 二、战斗场景

### Prompt 4：写一场战斗

```
请按以下要求，写一场战斗。

【战斗信息】
- 战斗类型：{{单挑/以弱胜强/群战/追杀/破阵/赌斗/内战/生离死别战}}
- 参与角色：{{}}
- 实力对比：{{}}
- 战斗地点：{{}}
- 战斗的戏剧任务：{{关系推进/身份揭示/信息揭示/能力突破/代价承担}}

【战斗的 5 段节奏】
- 段 1 铺垫（10-20%）：建立战斗语境
- 段 2 试探（15-20%）：评估实力
- 段 3 升级（30-40%）：主体交锋，戏剧任务出现
- 段 4 转折（10-20%）：5 种转折之一

- 写清"谁强谁弱"（境界/反应/环境破坏/过程对照）
- 战斗中的对话要有功能

【战斗的不可逆代价】
- 至少 1 个不可逆代价

请输出：完整战斗（2000-3000 字）
```

### Prompt 5：战斗中的对话

```

【战斗信息】

请输出：插入到 {{位置}} 的对话片段。

## 四、章尾设计

### Prompt 9：章尾发动机

```
请为这一章设计章尾。

【章节信息】
- 这一章发生的事：{{}}
- 角色当前状态：{{}}
- 下一章计划：{{}}

【章尾的 6 种发动机】
1. 新问题：章尾抛出新问题
2. 不可逆决定：主角做了无法回头的决定
3. 倒计时：开始倒计时
4. 权力反转：局势反转
5. 情绪余震：强烈的情绪震撼
6. 关系破口：关系出现裂缝

【关键】
- 避免平静收束（除非是过渡章）
- 章尾是"刺"——读者立刻想翻下一章
- 不要有大段描写或解释
- 至少 1 句"刺点"

请输出：3 个不同版本的章尾（每种 100-200 字），让我挑选。
```

### Prompt 10：章尾的不同强度

```
请为这一章设计 3 种不同强度的章尾。

【章节信息】
{{}}

【3 种强度】
- 弱（强度 2-3）：温和的悬念
- 中（强度 3-4）：明显的新问题或情绪冲击
- 强（强度 5）：不可逆决定/重大揭示/关系破裂

【心理描写的 5 种技法】
1. 内心独白（第一人称）
2. 行为表现（动作代替心理）
3. 身体反应（心跳/手抖/出汗）

## 六、环境描写

### Prompt 13：环境描写

```
请按以下要求，写一段环境描写。

【环境信息】
- 地点：{{}}
- 时间：{{}}
- 天气：{{}}
- 角色当前情绪：{{}}

【环境描写的 4 个功能】
1. 建立场景（让读者知道在哪）
2. 渲染气氛（用环境反映剧情）
3. 推动剧情（环境本身是情节的一部分）
4. 映射心理（环境反映角色内心）

【环境描写的 5 维】
- 视觉（光线、颜色、视野）
- 听觉（声音、寂静、回音）
- 嗅觉（气味、清新、腐臭、烟火）
- 触觉（温度、湿度、质地）
- 心理（环境对角色的影响）

【避免】
- 抽象的"美丽"或"荒凉"
- 形容词堆砌
- 与剧情无关的环境

请输出：200-400 字的环境描写。
```

### Prompt 14：环境与心理映射

```
请写一段环境描写，让环境反映角色的心理。

【角色心理】

【环境要求】
- 用环境的具体细节反映心理
- 不要直接说"他觉得压抑"
- 让读者通过环境感受心理

请输出：300 字的环境与心理映射描写。
```

## 十、续写/防漂

### Prompt 19：长篇续写

```
我正在续写 chapter-{{章节号}}（长篇连载）。

【必须先回顾】
- 上一章的最后段落：{{}}
- 90-运行/连载驾驶舱.md
- 90-运行/当前进度.md
- 角色最近的决策和状态变化

【续写要求】
- 必须接住上一章的所有悬念
- 必须延续所有角色状态
- 章尾必须给出下一章的钩子
- 不能"硬重启"——故事要连贯

【续写自检】
1. 是否接住了上一章的问题？
2. 角色状态是否延续？
3. 是否触发了至少一条旧线？
4. 是否新埋一条新债？
5. 章尾是否有钩子？

请输出：续写章节。
```
