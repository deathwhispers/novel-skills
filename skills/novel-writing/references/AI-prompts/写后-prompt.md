# 写后 Prompt

> 所有判断必须引用正文证据并说明受损或实现的阅读效果。P0 只用于连续性、人物动机、因果、hard canon 和非预期视角漂移等硬伤；不输出综合分。

## Prompt 1：完整章节体检

```text
【写作模式与作者意图】{{}}
【当前正文】{{}}
【必要上下文】{{最近正文、人物状态、hard canon、voice}}

先概括这段正文真正完成了什么，再检查：人物选择与代价、因果、视角和信息、关系或压力运动、语言与形式。

输出：最有效的 2 处原文；硬伤；最多 3 个最影响阅读效果的软问题；最小修改顺序。没有问题就不要凑数。
```

## Prompt 2：单点深读

```text
只检查 {{人物 / 因果 / 视角 / 信息 / 节奏 / voice / 形式}}。

每个结论按“原文证据 → 读者实际经历 → 为什么有效或失效 → 最小建议”输出。不要扩展到其他维度，不用词频代替阅读判断。
```

## Prompt 3：结尾效果

```text
判断结尾把注意力留在何处，以及它是否完成当前模式需要的推动、余震、回响、封闭、歧义或暂时停顿。

商业连载可额外检查承诺兑现和继续阅读动力；其他模式不因缺少钩子判失败。若建议改写，给出不添加新秘密或反转的最小版本。
```

## Prompt 4：跨章连续性

```text
对照最近有效正文、章节增量和当前章，检查人物位置、知识、关系、物件、时间和未完成行动。

只报告有双方证据的冲突；区分真实矛盾、视角差异、时间变化和资料空白。输出原文位置与最小修复，不要求当前章回收所有旧线。
```

## Prompt 5：逻辑、细节或回收修补

```text
先写必须保留的戏剧功能、人物选择、信息和 voice，再做最小修补。

修订顺序：人物与因果 → 场景和信息 → 句子与细节。不得为了“更刺激”自动添加误会、反派、秘密、伤亡或反转。输出修改片段及每处修改保护的效果。
```

## Prompt 6：语言与形式审读

```text
结合写作模式、人物和可靠 voice 样本审读正文。重点检查抽象解释是否替代体验、句法是否无意重复、意象是否失去具体性、叙述距离是否意外漂移。

引用证据并提出局部校准。不要统一句长、删除所有修饰语、制造口头禅或把偏离样本本身判为错误。
```

## Prompt 7：人物 voice 对比

```text
比较当前正文与可靠人物样本中的注意力对象、判断方式、回避策略、信息权限和句法节奏。

列出保持、合理变化和无依据漂移。只有最后一类需要修复；角色在压力、关系和成长后的声音变化可以成立。
```

## Prompt 8：卷级与长线复核

```text
【适用】完成一个阶段、卷或足够长的样本后。

检查作品承诺、人物弧线、关系、因果、信息与形式如何累积。指出真正停滞、重复或失联的线，并引用跨章证据。

商业连载可增加追读与承诺兑现维度；文学长篇不自动套章尾强度、升级速度或固定回收周期。
```

## Prompt 9：读者体验假设

```text
根据正文证据提出 2-3 个可证伪的读者体验假设，例如注意力落点、情感距离或理解负担。区分文本证据、推断和需要真实读者验证的部分。

不要虚构读者画像、平台偏好或弃读原因。
```

## Prompt 10：完成度判断

```text
判断当前单位是否已经实现作者意图并通过硬门禁。

硬门禁：已知连续性、人物动机、因果、hard canon、预期视角。软目标按写作模式与当前单位选择，不要求固定字数、模板完成率、推进、钩子、伏笔或统一强度。

输出“可以保留 / 需要最小修复 / 交由作者审美判断”，并给出证据。
```

## Prompt 11：生成续写状态增量

```text
从正文中提取下一次写作真正需要继承的变化：人物选择与状态、关系、位置与时间、物件、已公开信息、尚未完成的直接后果。

只记录正文已发生的事实；推测单列为待确认。不要把审美评价、摘要或新设定写进状态增量。
```
