## 一、角色扮演

### Prompt 1：让 AI 扮演某角色

```
你现在是 {{角色名}}。

【你的角色卡】
{{粘贴角色卡}}

【你的当前状态】
- 位置：{{}}
- 情绪：{{}}
- 目标：{{}}
- 与 {{另一个角色}} 的关系：{{}}

【你的 voice 设定】
- 优先注意：{{}}
- 回避或误解：{{}}
- 信息保留策略：{{}}
- 压力下语言变化：{{}}

【你的"语言指纹"】
- 绝不说的话：{{}}
- 判断世界的习惯：{{}}
- 可选句式与节奏证据：{{}}

【互动规则】
- 你的回复要符合角色性格
- 你的回复要符合你当前的状态
- 你的回复要根据对话推进关系或信息
- 不要"作者附身"——不要跳出角色说话
- 不要打破第四面墙

【当前对话】
{{对方说的话}}

请以 {{角色名}} 的身份回复。
```

### Prompt 2：让 AI 扮演多角色对戏

```
你现在要同时扮演以下角色：

【角色 A】
{{}}

【角色 B】
{{}}

【角色 C（如有）】
{{}}

【场景】
{{}}

【对戏规则】
- 每个角色必须有独立的语言指纹
- 每个角色的反应必须符合人设
- 禁止"五个人一起说同一件事"
- 主角的"听到"和"反应"必须不同

【触发事件】
{{}}

请按场景推进对话（每个角色至少 1 句）。
```

### Prompt 3：让 AI 扮演反派

```
你现在是反派 {{角色名}}。

【你的角色卡】
{{}}

【你的真实动机】
{{}}（不是表面上的）

【你的说话方式】
- 反派有反派的逻辑（他觉得自己是对的）
- 不要脸谱化（不是纯粹的"恶"）
- 你的话要"有说服力"

【互动规则】
- 你和主角是对立的，但你的理由要"立得住"
- 你说话要符合反派的"压迫感"
- 你可以有"灰色地带"（让读者思考你是对是错）

【当前对话】
{{}}

请以反派 {{角色名}} 的身份回复。
```

## 二、批量生成

### Prompt 4：批量生成节拍卡

```
请按以下信息，批量生成 chapter-{{X}} 到 chapter-{{Y}} 的节拍卡。

【本卷卷纲】
{{}}

【每章的理论任务（来自追读承诺分段兑现表）】
{{}}

【每章生成要求】
- 10 项必须填写
- 因果链必须连贯（上一章章尾接下一章章首）
- 整体要有"小高潮"和"中高潮"的分布
- 章尾必须明确

【输出格式】
| 章节 | 角色 | 任务 | 阻力 | 章尾 | 备注 |
|------|------|------|------|------|------|

请按顺序生成。
```

### Prompt 5：批量生成章节大纲

```
请按以下信息，批量生成 chapter-{{X}} 到 chapter-{{Y}} 的章节大纲。

【每章节拍卡】
{{}}

【每章大纲要求】
- 300-500 字的概要
- 必须包含场景数
- 必须包含章尾设计

请按顺序生成。
```

### Prompt 6：批量生成初稿

```
请按以下信息，批量生成 chapter-{{X}} 到 chapter-{{Y}} 的完整初稿。

【每章大纲】
{{}}

【整体要求】
- 每章 2000-2500 字
- 章内 4 段式节奏
- 章尾必须有钩子
- 角色 voice 一致
- 设定一致
- 因果链连贯

【注意】
- 这是"初稿"——会有需要修改的地方
- 请尽量写完整但不要过分雕琢
- 重要的对话和心理描写要写细
- 重复的内容可以略写

【输出】
- 每章的完整初稿
- 标注每章字数
- 标注"需要重点修的地方"
```

## 三、风格学习

### Prompt 7：提取项目 voice 证据

```
请从以下项目原文提取可验证的 voice 证据，不用“冷峻、细腻、自然”等空泛标签。

【章节样本】（最好 5-10 章）
{{}}

【输出维度】
1. 视角如何选择细节
2. 叙述者对人物和事件的态度
3. 句法与段落如何随压力变化
4. 对话如何回避、隐瞒或争夺关系位置
5. 意象和感官如何承担信息
6. 哪些差异是内容需要，不能被“统一文风”抹平

每个结论引用原文。输出 5-8 条可执行约束和 3 条不要表层模仿的特征。
```

### Prompt 8：按项目 voice 校准

```
请根据项目原文证据校准以下段落。保留原段的信息、人物选择和节奏功能，不追求表层仿写。

【作者文风画像】
{{}}

【原文】
{{}}

【输出】
1. 失真的具体位置及原文证据
2. 最小校准版本
3. 保留不改的差异及理由
```

### Prompt 9：检查文风一致性

```
请对比两段章节的文风是否一致。

【前段】
{{}}

【后段】
{{}}

【文风维度】
{{列出来自 Prompt 7 的 10 维}}

【输出】每维的对比 + 是否一致 + 不一致时的修正建议。
```

```

【5 种走向的差异】
```

## 五、卡文急救

### Prompt 15：卡文急救

```
我卡在 chapter-{{章节号}} 写不下去了。

【卡在哪】
{{具体描述：开篇/中段/高潮/章尾/对话/战斗/心理}}

【已有的内容】
{{前面写过的部分}}

【必须完成的任务】
{{节拍卡}}

【请帮我想 3 种"卡文急救"方案】

方案 1：跳到下一章
方案 2：换场景推进
方案 3：加新角色/新事件

【每种方案】
- 具体怎么写
- 风险（与已有剧情的兼容性）
- 优点
```

### Prompt 15.1：开篇卡文急救（5 分钟）

```
我卡在第 1 章的开篇。

【项目信息】
- 题材：{{}}
- 主线：{{}}
- 主角：{{}}

【5 分钟急救步骤】
1. 翻上一章最后一段（如有）
2. 翻节拍卡，看"被前文什么结果逼出来的"
3. 选 1 种开场钩子（动作/悬念/场景）
4. 写 200 字试试
5. 接着写 1000 字

【请按这 5 步给具体方案】
```

### Prompt 15.2：中段卡文急救（5 分钟）

```
我卡在中段。

【5 分钟急救】
1. 翻节拍卡，看"核心戏剧任务"
2. 写"冲突最激烈"的一个瞬间
3. 写 1000 字

【请给具体的"接下来怎么写"方案】
```

### Prompt 15.3：章尾卡文急救（5 分钟）

```
我卡在章尾。

【6 种发动机选一个】
1. 新问题
2. 不可逆决定
3. 倒计时
4. 权力反转
5. 情绪余震
6. 关系破口

【请给 3 种不同类型的章尾方案】
```

### Prompt 15.4：整体卡文恢复（5 天）

```
我连续卡文 3+ 天。

【请给 5 天恢复计划】
- 第 1 天：休息 + 整理
- 第 2 天：修改旧稿
- 第 3 天：写 1000 字过渡章
- 第 4 天：写 2000 字
- 第 5 天：恢复正常

【详细说明每天做什么】
```

### Prompt 16：灵感碎片转大纲

```
我有一些零散的灵感碎片，请帮我把它们转成大纲。

【灵感碎片】（任何形式：词/句/场景/情绪/画面）
{{}}

【本书信息】
- 题材：{{}}
- 主线：{{}}
- 当前进度：{{}}

【要求】

【输出】整理后的大纲。
```

## 八、失败模式扫描

### Prompt 18.1：失败模式扫描

```
请按以下信息，扫描这一章的失败模式。

【章节正文】
{{}}

【50 个失败模式】
1. 人物类（7 个）：工具人/扁平反派/突然变蠢/突然圣父/突然黑化/花瓶女主/同质化
2. 剧情类（8 个）：硬开金手指/突然变强/反派突然变弱/剧情无后果/主角无代价/强行推进/中段空转/剧情过密

5. 技法类（10 个）：抽象情绪/作者附身/套路化/多线齐开/作者说教/AI 味/对话过密/对话过疏/动作过水/动作过密

7. 风格类（5 个）：风格漂移/节奏不稳/人物前后矛盾/信息矛盾/地理矛盾
8. 对话类（5 个）：直说情绪/作者腔/同质化/信息大爆炸/嘴炮对话

【扫描输出】
1. 踩中的失败模式（按严重度排序）
2. 每条的修复建议
3. 重写方案
```
