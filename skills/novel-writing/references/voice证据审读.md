# Voice 证据审读

## 适用模式 / 何时使用

适用于作者已有至少两份认可的原文，需要判断候选正文是否延续叙述 voice。五种写作模式均可使用；模式与本次意图用于解释有依据的变化，不作为统一风格模板。

缺少可靠样本、只想模仿名家、或只需检查错字时不要使用。样本应来自作者有权使用且认可其 voice 的原文，尽量覆盖平静与受压状态；不要把待审候选混入基准。

## 生成审读包

```bash
python3 scripts/prepare_voice_review.py \
  --sample /path/to/approved-01.md \
  --sample /path/to/approved-02.md \
  --candidate /path/to/candidate.md \
  --mode 文学叙事 \
  --intent "人物刻意保持礼貌，但已开始怀疑来客" \
  --output /path/to/voice-review.md
```

样本默认每份最多收入 4000 字符，可用 `--sample-max-chars` 在脚本允许范围内调整。候选正文完整收入，样本截断状态与来源写入审读包。

## 六个比较维度

- **注意力对象**：叙述优先看人、风险、秩序、物质、关系还是概念；遗漏是否具有倾向。
- **判断方式**：人物或叙述者如何归因、比较、怀疑和修正判断。
- **信息与回避策略**：哪些信息直说、暗示、延迟、转移或拒绝命名。
- **叙述距离**：概括与场景、内外感知、叙述者与人物语言如何切换。
- **句法节奏**：句群如何展开、停顿、并置和收束；不要把平均句长当 voice。
- **压力下变化**：压力是否让既有特征压缩、断裂、过度运转或反向防御。

## 分类契约

- **保持**：候选延续了样本中可重复观察的选择，并有双方文本证据。
- **合理变化**：候选发生变化，但可由模式、场景意图、人物压力或叙述距离解释。
- **无依据漂移**：候选改变了既有选择，样本、意图与场景均不能支持该变化。

每个判断同时引用样本与候选的原文证据并标明来源。只有词频、长度或直觉相似不能构成结论；样本不支持时写“证据缺口”。不输出综合分、相似度，不自动生成改写稿。若需要修订，先由作者确认要保留的 voice 特征，再进入最小修订流程。
