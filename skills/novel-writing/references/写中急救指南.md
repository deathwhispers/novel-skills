# 写中急救指南

> 急救的目标是恢复人物与文字的运动，不是给正文添加规定动作。一次只试一个最小动作；如果问题来自方向错误，退回写作意图，而不是继续堆技巧。

## 一、先判断卡在哪里

| 表面症状 | 先检查 | 不要默认做什么 |
|---|---|---|
| 篇幅偏短 | 是否漏写必要的选择、过程或后果 | 加对话、景物或新事件凑字 |
| 篇幅偏长 | 是否重复信息、解释已显出的情绪、承担多个任务 | 为满足统一字数硬砍有效段落 |
| 场景不动 | 人物此刻想维持什么，什么迫使他回应 | 强塞第三方、冲突或反转 |
| voice 漂移 | 人物注意什么、回避什么、如何组织语言 | 加口头禅或标志动作冒充声音 |
| 不知道下一句 | 当前视角刚感知到什么，最不愿面对什么 | 随机制造意外 |
| 结尾不成立 | 当前场景造成了什么后果或余韵 | 每章强造悬念 |

## 二、五分钟动作

### 1. 篇幅失衡

- 用一句话写出本段必须完成的作用。
- 标出每一段提供的新信息、体验或变化。
- 偏短：只展开被跳过、且读者必须经历的过程。
- 偏长：优先删同义解释与重复反应；保留人物特征、节奏和潜台词。
- 若自然完成得更短或更长，记录理由后放行。

### 2. 场景没有运动

依次回答：

1. 人物进入场景时想得到、避免或维持什么？
2. 对方、环境或人物自身的矛盾怎样施压？
3. 人物必须做出什么选择、拒绝或重新理解？

缺哪一项就只补哪一项。安静场景的“运动”也可以是关系、认知或注意力变化，不要求外部冲突。

### 3. voice 漂移

- 回读同一人物 200-500 字可靠样本。
- 对比注意力对象、回避方式、信息策略、句法节奏和叙述距离。
- 只改最明显的一类偏差，再通读是否自然。
- 不用固定口头禅、身体动作或句长比例代替人物声音。

### 4. 卡在下一句

任选一个动作：

- 跳写你最清楚的瞬间，之后补桥。
- 把叙述距离拉近或推远，重新看同一时刻。
- 写下人物最不愿说出口的一句话，再决定它是否进入正文。
- 暂写一个可撤销版本，不在句子层面追求定稿。

### 5. 结尾失效

先判断当前模式需要什么：

- 商业连载：后果、未完成行动或新的压力。
- 类型长篇：阶段变化、选择代价或下一步动力。
- 文学叙事：回响、意象变化、认知裂缝或有意义的停顿。
- 短篇：封闭、翻面、余韵或精确留白。
- 探索起草：停在人物或文本刚显出新可能的位置。

## 三、回退路径

- 撞 hard canon、因果断裂、人物重大选择无依据：停笔，回上下文或写作支持。
- 连续两个场景 voice 漂移：回声音样本与文风指南。
- 所有补救都像“加料”：暂停，重写当前场景的一句话意图。
- 只是软目标不理想：先完成草稿，在写后阶段集中处理。
