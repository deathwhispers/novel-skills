## 一、心理描写的 5 大目的

| 目的 | 说明 | 例子 |
|------|------|------|
| **反映情感** | 让读者知道角色在想什么 | 喜悦/愤怒/恐惧 |
| **推动决定** | 让读者知道角色为什么做某事 | "他必须做这个决定" |
| **埋下伏笔** | 让读者知道角色藏了什么 | "他没告诉师父的是……" |
| **制造张力** | 让读者感到"他知道真相但没说" | "他看着师父离开——" |
| **展示成长** | 让读者看到角色的内心变化 | "他突然意识到……" |

> **核心规则**：心理描写不是"想什么就写什么"——必须有"功能"。

## 二、心理描写的 5 大技法

### 技法 1：内心独白

- **用法**：直接用第一人称写内心
- **例子**：
  > "我不能再犹豫了。"
  > 林尘在心里想。
  > "这一次，我必须做点什么。"
- **优点**：直接
- **缺点**：容易"直说"——失去文学性
- **适合**：内心冲突激烈的场景

### 技法 2：行为表现

- **用法**：用行为表现心理
- **例子**：
  > 林尘握紧了拳头。
  > "我不会让你死。"他低声说。
- **优点**：避免直说
- **缺点**：需要精准的行为设计
- **适合**：角色弧线推进

### 技法 3：身体反应

- **用法**：用身体反应表现心理
- **例子**：
  > 林尘的手在抖。
  > 他深吸一口气，试图让自己平静下来。
- **优点**：代入感强
- **缺点**：不能滥用（容易"全身都在抖"）
- **适合**：紧张/恐惧/激动的场景

### 技法 4：环境映射

- **用法**：用环境反映心理
- **例子**：
  > 林尘看着窗外的雨。
  > 雨水顺着玻璃流下——像是他的眼泪。
- **优点**：文学性强
- **缺点**：需要精心设计
- **适合**：心理细腻的场景

### 技法 5：矛盾冲突

- **用法**：用"想要又不敢想"的矛盾表现心理
- **例子**：
  > 他想冲过去——但他不能。
  > 他想告诉苏晚真相——但他不敢。
  > 他想相信师父——但他的眼睛告诉他那是谎言。
- **优点**：戏剧性强
- **缺点**：写得不好容易"作"
- **适合**：重大决策/关键时刻

## 三、心理描写的 5 个反模式

| 反模式 | 表现 | 危害 |
|--------|------|------|
| **直接说"我感到"** | "我感到愤怒" | 抽象 |
| **大段心理分析** | 500 字"我觉得……" | 读者弃读 |
| **多个角色同一心理** | 所有角色都"内心复杂" | 同质化 |
| **心理与行为脱节** | 内心愤怒但行为平静 | 失真 |
| **无功能心理** | 写心理但没推动剧情 | 浪费 |

## 四、心理描写的"层级"

> 心理描写有"深浅"——根据场景选择。

| 层级 | 字数 | 适合 |
|------|------|------|
| **浅层** | 50-100 字 | 普通场景 |
| **中层** | 100-300 字 | 重要时刻 |
| **深层** | 300-500 字 | 关键决策/重大事件 |
| **极深** | 500+ 字 | 极少用（角色内心独白章节） |

> **核心规则**：大多数场景用"浅层"，关键场景用"中层"或"深层"。

## 五、心理描写的"声音"

> 心理描写要符合角色 voice——不能所有人都是一个心理。

| 角色 | 心理描写的特点 |
|------|--------------|
| **沉稳角色** | 少而精的内心独白 |
| **热血角色** | 大量的身体反应 |
| **聪明角色** | 复杂的内心分析 |
| **冲动角色** | 矛盾冲突的心理 |
| **神秘角色** | 环境映射 |

> **关键**：心理描写要让读者"听出这是谁在想"。

## 六、心理描写的 5 种情绪对应技法

| 情绪 | 推荐技法 | 例子 |
|------|---------|------|
| **喜悦** | 行为表现 + 环境映射 | "林尘笑了——窗外也突然放晴" |
| **愤怒** | 身体反应 + 矛盾冲突 | "林尘的拳头握得咯咯响" |
| **恐惧** | 身体反应 + 环境映射 | "林尘的脚在抖——林子好像在盯着他" |
| **悲伤** | 内心独白 + 行为表现 | "他没说话——但他的眼睛红了一圈" |
| **爱意** | 矛盾冲突 + 行为表现 | "他想拥抱苏晚——但他只是远远看着她" |

## 七、心理描写的"5 个不写"

| 不写 | 原因 |
|------|------|
| **不写"我觉得……"开头** | 太直接 |
| **不写"他心里想……"开头** | 太直白 |
| **不写 200+ 字的纯心理** | 读者会跳 |
| **不写"他不知道的是……"开篇** | 太多"作者剧透" |
| **不写"他后来才明白……"开篇** | 同上 |

## 八、心理描写的"高级技法"——不写心理

> 心理描写的最高境界是"不写心理"——用行为/对话/环境暗示。

| 不写心理 | 怎么暗示 |
|---------|---------|
| **通过对话暗示** | "你变了。" 苏晚说。 |
| **通过沉默暗示** | 林尘没回答。 |
| **通过动作暗示** | 林尘的脚步加快了。 |
| **通过环境暗示** | 雨停了——但他没注意到。 |
| **通过反常行为暗示** | 林尘突然转身——他通常不会这样。 |

> **关键**：让读者自己"读出"心理——比直接说更深刻。

## 九、心理描写的"自检"

- [ ] 心理描写是否有"功能"？
- [ ] 是否避开了 5 个反模式？
- [ ] 心理层级是否合适（不超载）？
- [ ] 心理是否与角色 voice 一致？
- [ ] 是否有"不写心理"的尝试？
- [ ] 心理与行为是否一致？
- [ ] 是否推动剧情/弧线？

## 十、心理描写的"AI prompt"

### Prompt：写一段心理

```
请按以下要求，写一段角色心理描写。

【角色信息】
- 角色：{{}}
- 当前事件：{{}}
- 触发情绪：{{}}
- 心理变化：从 {{A}} 到 {{B}}

【心理描写的 5 种技法】
1. 内心独白
2. 行为表现
3. 身体反应
4. 环境映射
5. 矛盾冲突

【要求】
- 至少有 2 种技法
- 不要直接说"我感到"开头
- 字数：{{200-500}}
- 符合角色 voice

【输出】完整心理描写。
```
