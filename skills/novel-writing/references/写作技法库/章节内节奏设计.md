> **使用说明**：网文单章 2000-2500 字，要让读者在 3-5 分钟内持续追读，必须有章内节奏设计。
> 节拍卡是"章级别"的任务清单，卷内节奏曲线是"卷级别"的曲线。本表是"章内"——单章内部的节奏设计。
> 写章节前必读，写完后用本表自检。

## 一、章内的 4 段式结构

> 一章 2000-2500 字，按 4 段式分配最稳。

| 段 | 时长 | 占比 | 功能 |
|----|------|------|------|
| **铺垫** | 200-375 字 | 10-15% | 接上章 + 建立本章语境 |
| **推进** | 800-1250 字 | 40-50% | 主体交锋/对话/事件 |
| **高潮** | 500-750 字 | 25-30% | 本章的戏剧任务完成 |
| **余震** | 200-375 字 | 10-15% | 章尾钩子 + 收尾 |

### 段 1：铺垫（章首）

- **目的**：接上章 + 建立本章语境
- **必须包含**：
  - 时间/地点/视角人物
  - 上章悬念的延续
  - 本章的"小目标"
- **写法**：
  - 不要"昨天/今天/明天"的废话
  - 直接用动作/环境/对话开场
- **错误**：
  - ❌ "话说林尘离开青云宗后……"（流水账式开场）
  - ✅ 林尘踏入山谷的第一步踩到了一具尸骨。

### 段 2：推进（章中主体）

- **目的**：通过对话/事件/动作推进本章
- **必须包含**：
  - 至少 1 个冲突/障碍
  - 至少 1 个角色主动行为
  - 信息释放的节奏
- **写法**：
  - 不能平铺直叙——要有起伏
  - 至少 1 个"小高潮"或"小转折"
  - 不要全章都是对话或全章都是打斗

### 段 3：高潮（章末前）

- **目的**：完成本章的戏剧任务
- **必须包含**：
  - 本章核心事件的"结果"
  - 局面的不可逆变化
  - 读者的"读后效应"
- **写法**：
  - 高潮必须"足够大"——不能草草收场
  - 高潮的写法参考"反转/爽点/战斗收尾"技法

### 段 4：余震（章尾）

- **目的**：留下钩子 + 收尾
- **必须包含**：
  - 章尾发动机（6 种之一）
  - 下一章的"小目标"
- **写法**：
  - 章尾是"刺"——读者立刻想翻下一章
  - 至少 1 句"刺点"
  - 不要再有大段描写或解释

## 二、章内节奏曲线

> 单章内部的"紧张度"曲线——和卷内曲线类似，但更短。

### 标准章节奏

```
紧张度
  5 ┤                              ╭─╮
  4 ┤        ╭─╮                  │ │
  3 ┤      ╭─╯ ╰─╮    ╭╮         │ │
  2 ┤──────╯       ╰──╯ ╰──╮    ╭─╯
  1 ┤                          ╰──╯
  0 └────────────────────────────────
    铺垫    推进       高潮    余震
```

### 各种章类型

#### 类型 1：标准推进章

> 4 段式，递进到高潮。

#### 类型 2：情感章（慢节奏）

```
紧张度
  3 ┤        ╭───╮
  2 ┤      ╭─╯   ╰─╮
  1 ┤──────╯        ╰──╮
  0 ┤                   ╰──
    铺垫  情感展开  情感高潮  余震
```

#### 类型 3：战斗章（高强度）

```
紧张度
  5 ┤      ╭───────╮
  4 ┤    ╭─╯       ╰╮
  3 ┤──╭╯            ╰╮
  2 ┤  ╰╮             ╰╮
  1 ┤    ╰─╮           ╰─
  0 ┤       ╰─────────
    铺垫  试探  升级  转折  收尾
```

#### 类型 4：过渡章

```
紧张度
  2 ┤      ╭╮
  1 ┤──────╯╰───╮
  0 ┤             ╰──
    铺垫  小事件  余震
```

> 过渡章 800-1500 字即可，**不要硬撑 2000+**。

## 三、章内的"信息释放"节奏

> 一章内揭示多少信息、按什么节奏释放。

### 信息释放的 5 种位置

| 位置 | 适合 | 强度 |
|------|------|------|
| **章首** | 上章悬念的兑现 | 中 |
| **章中冲突中** | 让信息改变局势 | 中-高 |
| **章末前** | 章尾钩子的关键信息 | 高 |
| **场景转换时** | 视角切换中的信息 | 中 |
| **环境/动作中** | 暗线信息 | 弱 |

### 一章信息密度的 4 档

| 密度 | 适合 | 章节类型 |
|------|------|---------|
| **0 个新信息** | 过渡章 | 短小章 |
| **1 个新信息** | 标准章 | 大多数章节 |
| **2 个新信息** | 推进章 | 重要章节 |
| **3+ 个新信息** | 极少用 | 揭露章/转折章 |

> **核心规则**：一章 1-2 个新信息。**超过 3 个会让读者消化不了**。

## 四、章内的"情绪曲线"

> 章内读者情绪的起伏——比"紧张度"更细腻。

### 情绪代码

| 代码 | 情绪 | 场景 |
|------|------|------|
| **S** | 爽（Satisfying） | 装逼打脸、身份揭晓 |
| **T** | 紧张（Tense） | 战斗、追逐、危险 |
| **E** | 虐/痛（Emotional pain） | 失去、背叛、生离死别 |
| **R** | 浪漫（Romantic） | 暧昧、告白、亲昵 |
| **A** | 惊讶（Astonished） | 反转、意外 |
| **C** | 平静（Calm） | 过渡、日常、铺陈 |
| **B** | 悲伤（Bitter） | 牺牲、失去、遗憾 |
| **N** | 怀旧（Nostalgic） | 回忆、过去 |
| **M** | 神秘（Mystery） | 悬念、伏笔、信息不足 |

### 一章的情绪代码示例

> 第 5 章：主角救人失败，苏晚受伤 → 战斗失败但关系推进
> 情绪代码：C → T → E → R → A

- 章首（C）：铺垫，平静
- 章中前（T）：战斗紧张
- 章中后（E）：战斗失败，主角伤心
- 章末前（R）：苏晚被救下，两人关系推进
- 章末（A）：揭示某件重要信息

### 情绪的"反单调"原则

> 一章内不要全是同一种情绪。**情绪要有起伏**。

| 单调章（避免） | 起伏章（推荐） |
|--------------|--------------|
| 全章 T（战斗） | C→T→E→R（铺垫-战斗-失败-浪漫） |
| 全章 S（爽） | T→S→A→E（紧张-爽-惊讶-痛） |
| 全章 E（虐） | C→E→R（铺垫-虐-浪漫余震） |

## 五、章内的"场景切换"设计

> 一章 2000+ 字如果只有一个场景，读者会闷。
> 最佳章节有 2-3 个场景切换。

### 场景切换的 3 种方式

1. **硬切**：直接换场景
   > "林尘离开山谷时，身后的青云宗已经消失在云雾中。"
   > （下一段就是新场景）

2. **软切**：用过渡承接
   > "他走出青云宗的那一刻，山脚下正下着小雨。"

3. **视角切**：用视角人物做桥梁
   > A 视角人物离开 → B 视角人物在另一边

### 场景切换的"3 段式"

> 一个完整场景至少 3 段：进入场景 → 场景展开 → 离开场景。

```
进入场景：
  "林尘推开酒楼的门。"（用动作进入）

场景展开：
  （场景内的对话/事件/动作）

离开场景：
  "林尘走出酒楼时，天已经黑了。"
  （用动作/时间变化离开）
```

## 六、章内的"视角移动"

> 同章内可以切视角吗？——可以，但要遵守规则。

### 同章切视角的 3 个规则

1. **必须有时空间隔**：
   - ❌ A 视角在第 1 段，B 视角在第 2 段（同时间同地点）——读者会搞混
   - ✅ A 视角在青云宗，B 视角在皇都

2. **必须在场景边界切**：
   - 在场景结束后再切，不要在场景中途切

3. **必须明确标记**：
   - 用章节内的空行或场景提示
   - 视角人物的名字要在新场景首段出现

## 七、章尾的 6 种发动机

> 章尾是追读的关键——读者读完一章后必须想翻下一章。

| 发动机类型 | 写法 | 适合 |
|----------|------|------|
| **新问题** | "但赵长老的信里还提到了什么？" | 大多数章节 |
| **不可逆决定** | "苏晚握紧了剑——她做了一个决定。" | 关系转折章 |
| **倒计时** | "三天后的月圆之夜，封印就要破了。" | 紧张章 |
| **权力反转** | "李昭本想伏击主角——但他不知道，伏击他的人已经在身后。" | 反转章 |
| **情绪余震** | "主角看着师父的墓，许久没有说话。" | 情感章 |
| **关系破口** | "苏晚的眼睛里，第一次有了恨意。" | 虐章 |

> **核心规则**：章尾必须有发动机。**没有发动机的章 = 弃书点**。

## 八、章内节奏的"翻车"形态

| 翻车形态 | 表现 | 危害 |
|---------|------|------|
| **铺垫太长** | 章首 1000+ 字还没进入正题 | 读者弃章 |
| **平铺直叙** | 一章没有任何起伏 | 读者走神 |
| **高潮太快** | 章首 500 字就高潮 | 后面乏力 |
| **高潮太弱** | 章末草草收场 | 读者失望 |
| **情绪单调** | 一章全是同一种情绪 | 读者疲劳 |
| **信息过密** | 一章 5+ 个新信息 | 读者消化不了 |
| **场景单一** | 一章只一个场景 | 读者闷 |
| **章尾平** | 章尾没有钩子 | 读者不再翻下一章 |

## 九、章内节奏自检

写完一章后，逐项检查：

- [ ] 是否有 4 段式结构（铺垫-推进-高潮-余震）
- [ ] 紧张度是否有起伏（不是一平到底）
- [ ] 情绪是否有变化（不是单一情绪）
- [ ] 信息密度是否合适（1-2 个新信息）
- [ ] 是否有场景切换（建议 2-3 个）
- [ ] 是否有视角移动（如有，规则是否正确）
- [ ] 章尾是否有 6 种发动机之一
- [ ] 是否避开了 8 种翻车形态
- [ ] 本章的戏剧任务是否完成
- [ ] 读完后，读者最可能想知道什么（章尾要回答）

## 十、不同章节长度的章内节奏

### 短章（800-1500 字）

- 适合：过渡章、对话章、小战斗
- 节奏：铺垫 + 推进 + 章尾（3 段式）
- 重点：章尾必须强

### 标准章（2000-2500 字）

- 适合：大多数章节
- 节奏：4 段式
- 重点：节奏要稳

### 长章（3000-5000 字）

- 适合：大战斗、卷高潮
- 节奏：4 段式 + 多个小高潮
- 重点：必须有"喘息段"

### 超长章（5000+ 字）

- 适合：决战级
- 节奏：4 段式 + 多场景 + 多视角
- 重点：避免读者疲劳，每 1500-2000 字要有节奏变化
