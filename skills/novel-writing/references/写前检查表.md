# 写前最小确认

目标是尽快开始正文，不是把所有问题提前填满。已有上下文能回答时直接判断，不重复询问用户。

## 所有模式都确认

- 写作单位：片段 / 场景 / 章节 / 短篇
- 写作模式：商业连载 / 类型长篇 / 文学叙事 / 短篇 / 探索起草
- 当前视角人物及其此刻的注意焦点
- 人物想靠近、逃避、理解或维持什么
- 当前最相关的压力、关系或未知
- 不可违反的 canon 与不可擅自补全的空白
- 要延续的 voice 样本或语言选择

## 按模式追加

### 商业连载

- 本章承诺什么、兑现什么？
- 哪个变化或问题推动继续阅读？
- 本章触碰哪条旧线？

### 类型长篇

- 本场景提供什么类型体验？
- 因果、悬念或人物弧线推进哪一项？
- 哪条规则或信息差在施压？

### 文学叙事

- 这段通过谁的观察建立意义？
- 哪种情绪或关系不能被直接简化？
- 语言、意象或叙述距离承担什么作用？

### 短篇

- 哪些信息必须压缩或延迟？
- 当前场景如何与题眼、意象或结尾形成回声？
- 哪一处绝不能解释过满？

### 探索起草

- 当前最想发现的是人物声音、关系、场景还是情节可能性？
- 哪些内容允许暂时矛盾或未定？
- 这次起草结束后用什么判断“值得继续”？

## 何时才停下来询问

仅当缺失信息会改变人物重大选择、hard canon、核心视角或整章方向时暂停。其他小空白使用最保守写法并标记“待作者确认”。
