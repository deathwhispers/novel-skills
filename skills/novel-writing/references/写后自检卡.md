# 起稿后自检卡

正文初稿写完后，先用这张卡快速过一遍，再决定是继续下一章、先体检，还是转修订。

## 先看这一章有没有成立

- 本章到底完成了什么：
- 本章局面到底改变了什么：
- 如果删掉这一章，会不会直接断线：

## 再看最容易出问题的地方

- 有没有哪一段只是解释，没有真正发生戏
- 有没有哪一段本来该冲突，却写成了平铺说明
- 有没有哪一个转折其实是作者硬推
- 有没有哪一个人物反应和前文状态接不上

## 再看读者为什么要追

- 章尾真正挂住读者的东西是什么：
- 下一章最自然的起点是什么：
- 本章发出去后，读者最可能讨论什么：

## 最后决定转向

- 主要是漏线、打架、状态没续上 -> 转 `novel-quality`
- 主要是逻辑、细节、回收不稳 -> 转 `novel-quality`
- 主要是整体方向错了 -> 转 `novel-outline`

## 再看 voice 是不是漂了

按 `novel-quality/references/文风快速自检卡.md` 的 5 条逐项过：

- [ ] 叙述距离和前 3-5 章一致（没突然变紧贴内心或突然变客观）
- [ ] 主角的标志动作/口头禅这章至少出现 1-2 次
- [ ] 核心角色对话长度和留白匹配他/她的卡
- [ ] 描写密度没明显偏离文风基线
- [ ] 没出现禁用语清单里的词

> 任何一条不匹配，先改这一段（不要整章重写）。
> 连续 2 条不匹配 = 本章 voice 漂了，要回到章节层修订。

---

## 常见自检遗漏

| 遗漏 | 表现 | 修复 |
|------|------|------|
| **戏没立住** | "这一章写了什么？"答不上来 | 回到节拍卡，确认核心戏剧任务 |
| **解释代替戏** | 大段"由于……所以……" | 删解释，加动作/对话 |
| **转折硬推** | "然后他就赢了"——怎么赢的？ | 加阻力/代价/信息差 |
| **章尾收束** | "主角休息了一晚" | 加钩子/问题/压力 |
| **状态不接** | 人物情绪/伤势和前文不一致 | 检查连续性 |
| **voice 漂移** | 叙述风格突然变了 | 加标志性动作，调叙述距离 |
| **没有旧线** | 连续 3 章不碰旧线 | 在下一章设计旧线触碰点 |

## 快速决定转向

自检后 1 分钟决定：

| 问题类型 | 主要症状 | 转向 |
|---------|---------|------|
| 漏线/打架/状态断 | 前后接不上 | `novel-quality`（体检） |
| 逻辑/细节/回收不稳 | 因果不清楚 | `novel-quality`（修订） |
| 整体方向错了 | 这章不该存在 | `novel-outline`（重审节拍卡） |
| 只是句子不好 | 戏都对，但读起来不顺 | `novel-quality`（去味） |
| 都通过 | 戏立住、逻辑顺、voice 稳 | 继续写下一章 |
