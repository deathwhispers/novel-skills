# 写后轻量自检

先完整读一遍正文，再检查。不要边读边按词表修改。

## 1. 硬门禁

- 视角、时间、地点、伤势、物件和人物已知信息是否连续？
- 重要选择能否从欲望、压力、认知或关系中理解？
- 是否无意违反 hard canon？
- 是否擅自把未知设定写成事实？

硬门禁失败时做最小修复；骨架失效才回到节拍卡或重写。

## 2. 模式目标

- 商业连载：承诺、兑现与继续阅读动力是否清楚？
- 类型长篇：类型体验、因果、悬念或弧线是否推进？
- 文学叙事：观察、情绪真实、语言选择或主题回响是否成立？
- 短篇：压缩、结构回声、意象和结尾是否互相支持？
- 探索起草：是否发现了值得保留的人物声音、关系或故事方向？

## 3. 人物与语言

- 人物注意什么、回避什么、如何组织语言，是否仍属于这个人？
- 叙述距离和句法节奏的变化是否有作用？
- 解释、概括、动作和细节有没有重复表达同一信息？
- 最准确的一处语言是什么？最空泛的一处是什么？

不要求每章出现口头禅、标志动作、固定数量细节、冲突、反转或强钩子。

## 4. 决策

- 有硬伤：局部修复并复查。
- 模式目标未完成：修最影响阅读效果的一处。
- 只有软风险：正文后标记，不自动改写。
- 已成立：交付正文，记录状态增量。
