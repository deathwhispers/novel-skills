# 起稿后自检卡

正文初稿写完后，先用这张卡快速过一遍，再决定是继续下一章、先体检，还是转修订。

## 先看这一章有没有成立

- 本章到底完成了什么：
- 本章局面到底改变了什么：
- 如果删掉这一章，会不会直接断线：

## 再看最容易出问题的地方

- 有没有哪一段只是解释，没有真正发生戏
- 有没有哪一段本来该冲突，却写成了平铺说明
- 有没有哪一个转折其实是作者硬推
- 有没有哪一个人物反应和前文状态接不上

## 再看读者为什么要追

- 章尾真正挂住读者的东西是什么：
- 下一章最自然的起点是什么：
- 本章发出去后，读者最可能讨论什么：

## 最后决定转向

- 主要是漏线、打架、状态没续上 -> 转 `novel-quality`
- 主要是逻辑、细节、回收不稳 -> 转 `novel-quality`
- 主要是整体方向错了 -> 转 `novel-outline`
