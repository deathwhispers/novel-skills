# 写作流程深度选择

检查深度服从任务，不要求每次写作完整走完全部流程。

## 快速起草

适用：试写、探索、续写片段、寻找人物声音。

1. 判断写作模式和视角。
2. 读取最近正文与必要 canon。
3. 直接输出正文。
4. 只标记硬伤和待确认项。

## 标准写章

适用：类型长篇、常规连载章节。

1. 读取节拍卡、最近正文、相关人物与 voice。
2. 连续完成场景，在场景边界检查硬门禁。
3. 先交付正文，再做轻量自检。
4. 写章节状态增量；不逐章重写全部总账。

## 发布前精修

适用：用户明确要求精修、投稿、发布或定稿。

1. 检查硬门禁。
2. 根据写作模式选择 2-3 个质量目标。
3. 先修结构和人物，再修信息与语言。
4. 每个问题引用原文并说明受损效果。
5. 对比修改前后，确认没有损害 voice、潜台词和节奏。

## 周期复盘

适用：每 5 章或卷末。

1. 合并章节状态增量。
2. 扫描连续性、伏笔和人物状态。
3. 汇总重复写作问题。
4. 只选一个下一周期训练目标。

## 停笔条件

仅在以下情况暂停并请求作者决定：

- 两种选择会改变核心人物立场或主线方向。
- 必要 hard canon 缺失或互相冲突。
- 用户要求与已有正文存在无法保守处理的冲突。

状态文件缺失、软目标不足、字数偏离或表层风险不阻止正文交付。
