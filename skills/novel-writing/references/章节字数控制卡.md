# 篇幅预算卡

> 篇幅是生产与媒介约束，不是文学质量分数。只有平台、投稿或用户明确给出限制时才构成硬约束。

## 一、先确认约束来源

| 来源 | 处理方式 |
|---|---|
| 平台/投稿硬限制 | 在不破坏必要内容的前提下满足，超限时调整场景边界 |
| 商业连载更新预算 | 作为可持续区间，允许章节功能导致的合理波动 |
| 类型或作者习惯 | 作为节奏观察，不单独触发扩写或删改 |
| 文学、短篇、探索起草 | 以形式、压缩和阅读效果决定自然篇幅 |

## 二、写前预算

- 当前写作单位：场景 / 章节 / 片段 / 全篇
- 目标区间及来源：
- 这个单位必须让读者经历的过程：
- 可以省略、跳切或留白的部分：
- 若自然完成后偏离区间，允许怎样拆分或合并：

不按固定比例预分铺垫、推进、高潮和余震；只有明确选择商业连载四段式时才参考相应技法库。

## 三、写中观察

不要按固定字数中断连续起草。到场景边界再判断：

- 偏短是否因为跳过了必要选择、过程或后果？
- 偏长是否因为重复信息、解释已经显出的情绪，或同时承担多个写作单位？
- 当前长度是否服务叙述距离、人物体验和模式？

偏短不默认加对话、环境或新事件；偏长不默认砍人物特征、潜台词和有效节奏。

## 四、写后决定

1. 自然完成且无硬限制：放行并记录实际长度。
2. 有硬限制但内容成立：优先移动场景边界或压缩重复信息。
3. 内容未成立：修复人物、因果或场景功能，不把问题命名成“字数不足”。
4. 连续偏离预算：调整项目预算，不能要求正文长期迎合错误估算。
