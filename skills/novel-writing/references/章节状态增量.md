# 章节状态增量

正文完成后只记录本章真实发生的变化。没有变化的总账不更新。

```markdown
# chapter-XXX 状态增量

- 新增 hard canon：无 / …
- 人物状态变化：无 / …
- 关系变化：无 / …
- 新增或推进的伏笔：无 / …
- 已回收事项：无 / …
- 新增未决问题：无 / …
- 下一章必须继承的直接后果：…
- 待作者确认：无 / …
```

保存到 `90-运行/章节增量/chapter-XXX.md`。续写上下文在合并前必须读取最近相关增量；每 5 章或卷末再把增量合并进人物状态、因果图、回收总账和前文索引。只有新 hard canon 需要立即写入 `硬设定.md`；不确定内容放入“待作者确认”，不得擅自固化。
