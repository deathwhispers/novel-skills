# 正文质量标准

质量不是命中特定技巧，而是形式、人物、语言与当前写作目的相互支持。

## 共同底线

1. **连续性可信**：视角、人物状态、规则和已知信息没有无意断裂。
2. **人物可信**：重要反应来自人物已有欲望、经验、压力和关系，而非作者强推。
3. **信息清楚到恰当程度**：读者知道当前需要知道的，也保留当前不该说透的。
4. **语言有效**：总结、解释、动作、对话、心理和环境各自在需要时出现，不重复证明同一件事。
5. **形式服务内容**：节奏、段落、句法、视角距离和结尾方式匹配当前场景。

## 模式目标

| 模式 | 判断重点 |
|---|---|
| 商业连载 | 承诺与兑现清楚，读者知道为何继续读 |
| 类型长篇 | 类型体验成立，因果、悬念或人物弧线有积累 |
| 文学叙事 | 观察有选择，情绪可信，语言与主题形成压力或回响 |
| 短篇 | 信息经济，局部与整体互相照亮，结尾改变或深化前文 |
| 探索起草 | 产生值得继续的人物声音、关系、场景或问题 |

## 不使用的机械判断

- 不用对话比例、句长、细节数量或字数直接判质量。
- 不把“感到、觉得、因为、似乎”等词单独判为问题。
- 不要求所有人物主动外部行动；观察、等待、拒绝和认知变化也可以构成有效场景。
- 不要求每章不可逆变化、强钩子、爽点、反转或旧线触碰。
- 不用“摔杯、攥拳、咬唇”等动作机械替代情绪命名。

## 快速判断

引用正文回答：

1. 这段最重要的阅读效果是什么？
2. 哪处原文真正产生了这个效果？
3. 哪处内容削弱、重复或解释了这个效果？
4. 最小的有效修改是什么？

无法引用原文时，不做确定性质量结论。
