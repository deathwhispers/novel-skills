# 长篇续写防漂卡

适用于你已经写到中段，准备继续下一章时先看一眼，避免长篇越写越飘。

## 续写前先对齐

- 当前写到第几卷第几章：
- 最近 3 章最主要在推什么：
- 当前最不能丢的主线是什么：
- 当前最不能丢的关系张力是什么：
- 当前最不能丢的悬念或信息债是什么：

## 这章为什么必须写

- 这章是被上一章什么结果逼出来的：
- 如果这章不写，哪条线会断：
- 这章必须接住哪一个章尾发动机：

## 这章最容易漂在哪里

- 容易回头重复解释哪里：
- 容易写成无效过渡的地方：
- 容易忘掉的伤势 / 物件 / 关系状态：

## 续写时优先保什么

- 优先保局面变化，不优先保字数
- 优先保人物状态延续，不优先保华丽描述
- 优先保追读压力，不优先保段落工整

---

## 常见续写错误

| 错误 | 表现 | 修复 |
|------|------|------|
| **回顾过多** | 开头花 500 字回顾前文 | 压缩到 50 字，或直接动作开场 |
| **主线遗忘** | 核心目标长期没有产生压力或变化 | 在合适章节恢复目标、代价或人物对它的新理解 |
| **关系冻住** | 男女主 20 章没变化 | 加关系推进事件 |
| **voice 漂移** | 主角的注意力和信息策略无依据变化 | 回读可靠样本，校准观察顺序、回避方式和句法节奏 |
| **伤势消失** | 前章骨折，这章用手持剑 | 加"忍着痛"或改变动作 |
| **章尾弱** | "主角准备出发" | 用追读钩子设计强化 |
| **重复升级** | 连续 3 章都是"主角变强" | 引入非能力升级 |

## 快速检查（1 分钟）

续写前回答：
1. 上一章的最后一句是什么？（必须直接衔接）
2. 这章的核心任务是什么？（用动词开头，10 字以内）
3. 如果删掉这章，哪条线会断？

**任一题说不清→停下来，先读 `上下文包.md` 对齐**
